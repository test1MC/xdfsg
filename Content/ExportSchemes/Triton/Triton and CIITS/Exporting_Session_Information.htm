Exporting Session Information

Exports are downloaded as a CSV file which can be opened with Microsoft Excel.

The following information is exported from Triton DCS when using the CIITS Export:

  • Student ID
  • A single column for each question with the participant’s response

Essay responses are not included in the CIITS Export and are exported as a blank column.

  1. Click Schedules from the navigation bar and select the Exports tab.
  2. Select CIITS Export from the drop-down menu.
  3. Select the Schedules, Participant Lists, Answer Documents and Versions from the columns.
  4. Click Download File.

    The downloaded CSV file can be opened and edited in Microsoft Excel.

  5. Save the CSV file.
