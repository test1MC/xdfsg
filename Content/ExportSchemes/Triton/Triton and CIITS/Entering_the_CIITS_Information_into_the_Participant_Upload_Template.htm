Entering the CIITS Information into the Participant Upload Template

The following tasks must be completed by an Organization Manager or a Site Manager with the appropriate rights in Triton Web.

  1. Click Organization from the navigation bar and select the Import tab on the right side of the screen.
  2. Click the Template tab on the left side of the window and click Participants to download the template file.

    The template file opens in CSV format in Microsoft Excel.

  3. Copy the Student IDs column from the CIITS file and paste it into the PARTICIPANT_ID column of the Participant template.
  4. Enter the first name and last name for each student into the FIRST_NAME and LAST_NAME columns of the Participant template.
  5. Save the file in CSV format.

    When saving the file, a dialogue box like the one pictured below may appear.

    Click Yes to save the file as a CSV.

    Triton Web only accepts Upload Templates in the CSV format. The template file should not be saved as a Microsoft Excel (XLS or XLSX) file.

  6. Click the Upload tab on the left side of the window and select Participants from the Type drop-down menu.
  7. Click Browse.
  8. Select the upload file and click Open.
  9. Click Import.

 

Participant Lists will need to be set up within Triton Web. You will also need to have a set of Administration Rules and Answer Document(s) that are assigned to a Blueprint and then scheduled for administration. The Schedule must be administered via the Triton Proctor Application and the results must be finalized within Triton Web before the session information can be exported. Please see the Triton Web user guide for more information.
