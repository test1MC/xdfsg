Uploading Session Information into CIITS

You must have a test set up in CIITS that matches the Answer Document created in Triton Web.

  1. Log into CIITS.
  2. Mouse over the Assessment Admin tab and select Find a Test.

  3. Select the test from Test Central.
  4. Click Score Test from the Actions panel on the left.

  5. Click Upload Scores from the Search Results panel on the left.
  6. Select SchoolNet from the Choose File Format drop-down menu.
  7. Click Choose File or Browse, select the CSV file from the previous section and click Open.
  8. Click Upload.

    The scores are now uploaded into CIITS.
