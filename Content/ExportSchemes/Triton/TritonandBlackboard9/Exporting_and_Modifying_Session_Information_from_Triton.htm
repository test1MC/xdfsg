Exporting and Modifying Session Information from Triton

Exports are downloaded as a CSV file which can be opened with Microsoft Excel.

  1. Click Schedules from the navigation bar and select the Exports tab.
  2. Select Turning Export from the Export Type drop-down menu.
  3. Select the Schedules, Participant Lists, Answer Documents and Versions from the columns.
  4. Click Download File.
  5. Open the downloaded export file Microsoft Excel.
  6. Delete all columns EXCEPT Participant_ID and Points Earned.
  7. Rename the Participant_ID column to username.
  8. Rename the Points Earned column. This will be the name you want the gradebook to display in Blackboard.
  9. Save the file as a CSV.
