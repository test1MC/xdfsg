Turning Technologies® and
C3 SoftWorks - Bravo!™

Technical Requirements

  • Turning Technologies Hardware
    • ResponseCard RF, XR or NXT
    • RRRF 02, 03 or 04 receiver
  • Bravo! Software 1.68 or higher

Using Turning Technologies ResponseCards with Bravo!

Turning Technologies ResponseCards work with the Quiz Show and Spin Off games.

When playing in Audience Response mode, the following features will be disabled:

  • Wagering
  • Enable take turns mode
  • “Try again”
  • Ring in timer
  • The question timer will work, however when time is up, participants can still respond. Click Next to stop polling.

How to Use Turning Technologies ResponseCards with Bravo!

  1. Plug in the receiver.
  2. Open Bravo! and select either Quiz Show or Spin Off.
  3. Click Response Settings from the left menu.
  4. Select Turning Technologies ResponseCards from the Response Type drop-down menu.
  5. Select additional options within Audience Response Settings, if necessary.

    If you choose to assign ResponseCards, there can only be one captain per team. This allows for teams to “ring-in” to answer. Please note that by using a ResponseCard as a ring-in device, the polling option will be disabled.

  6. To verify that the receiver is functioning properly, uncheck Assign pads during game. Click the Connect to Hardware button. You should see the message Press a button on a keypad to add it to the list.
    Optionally, if the ResponseCards are on a channel different than the number listed next to the Change Channel button, click the Change Channel button and select the number that corresponds with the ResponseCard channel.
  7. Click OK.

    For more information on Bravo!, please visit www.c3softworks.com/manuals/bravo/help.html.
