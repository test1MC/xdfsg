Turning Technologies® and Elements

This document outlines the steps for using Turning Technologies NXT ResponseCards with Elements.

How does the integration work?

Elements provides an option for participants to take tests using Turning Technologies NXT ResponseCards (clickers). The Turning Technologies ResponseCards score assessments and, when finished, results are uploaded to Elements. In order for the ResponseCards to work with Elements, the following items are required:

  • Turning Technologies hardware - ResponseCards and RRRF-02 receiver (dark gray)
  • Turning Technologies software - TurningKey for PC or Mac
  • Elements Responder Devices software

    Both software applications should be installed on your computer prior to using Turning Technologies ResponseCards and Elements. If either software application is not installed, contact your IT Administrator.

Using Turning Technologies ResponseCards with Elements

How to Configure TurningKey and Elements

  1. Plug in the RRRF-02 receiver.
  2. Open TurningKey.
  3. Verify all ResponseCards are on the same channel as the RRRF-02 receiver.
    1. The RRRF-02 receiver channel is listed in the right panel above the Participant List. To change the channel on a PC, click Tools > Settings > Response Devices. To change the channel on a Mac, click Turning Technologies on the menu bar > Preferences > Response Devices.
    2. The device channel is displayed in the upper left of the clicker screen when the receiver is active.
  4. Login to Elements to find the test to be administered. Make a note of the Test ID located on the Dashboard.
  5. Open Elements Responder (from the Desktop).

    The following message appears if another clicker type was previously used:

  6. Click Cancel.
  7. Click File and select Setup.

  8. Select Turning Technologies from the Responder Type drop-down menu.

  9. Click OK.
  10. Enter Server Settings, Client ID and Server Name in the appropriate drop-down menus. If this information is unknown, contact your IT Administrator. Optionally, if you connect via proxy server, enter those settings in the Proxy Server Settings drop-down menu.
  11. Click OK.

How to Administer a Test

  1. Enter your Elements User ID and Password in the fields provided.
  2. Click Login.
  3. Enter a classroom name and click OK.

  4. Enter the Test ID in the Test ID drop-down menu.
  5. Click the Load Testicon.

    The following window appears:

  6. Click OK.
  7. Click the Start Test icon.

  8. Click OK.

How to Use Turning Technologies NXT ResponseCards

  1. Participants must enter their Student IDs or Login IDs on their ResponseCards.
    1. Press the center button on the ResponseCard.
    2. Enter the Student ID or Login ID on the Your ID screen.
    3. Press the center button.
    4. The ResponseCard displays the class name and number of test questions.
    5. Press OK.

      The status will display as Active on the test administrator screen.

  2. Participants should enter an answer for each question.
    1. Select an answer and press the center button.
    2. Use the up and down arrows to move the flashing cursor to a specific question number section at the top of the display screen.
    3. Use the right and left arrows to navigate to a specific question.
    4. Use the down arrow to navigate to the answer section of the display screen.
    5. Once all questions have been answered, a prompt to save the test appears on the ResponseCard. Press OK.
    6. A second prompt appears. Press OK.
    7. The ResponseCard saves the test, the responses are sent and the clicker shuts off.
  3. A participant response shows on the administrator screen as a green check (correct answer) or red check (incorrect answer).
    • A blank cell indicates that the question has not yet been answered.
    • The current score is the percentage correct for the questions that have been answered.

  4. Only if necessary, click Reset to clear a participant’s answers in Elements Responder. The following window appears.

  5. Click Yes to end the test and upload the results to Elements. Click No to clear all results.
  6. Click OK.

 

How to Review Results

Open Elements to view the results data.

If the test Dashboard is open, click the Refresh icon on the left navigation pane. If the test Dashboard is not open, go to Assessments and then select the Dashboard icon.
