Turning Technologies® and SchoolNet®

This document outlines the steps for using Turning Technologies ResponseCards with the SchoolNet software.

Technical Requirements

  • SchoolNet® 14.2 or higher
  • Microsoft® .NET Framework 2.0 or higher
  • Java™ Runtime 1.6 or 1.7 (also known as JRE6 or JRE7)

    To verify Java version in Windows 7:

    • Start > Control Panel > Programs > Java
    • Select Java
    • Click the General tab
    • Click About
  • Turning Technologies Hardware
    • ResponseCard NXT - for standard response tests
    • ResponseCard RF - for polling type tests
    • RRRF-02 receiver (dark gray) - plugs into USB Hub

Using Turning Technologies ResponseCards with SchoolNet

During class, results are gathered from participants.

How to Use Turning Technologies ResponseCards with SchoolNet

  1. Plug in the RRRF-02 receiver.
  2. Login to SchoolNet.
  3. Open the Proctor Dashboard for the test.

    Teachers (Select One):

    1. Home > Upcoming Tests > Online Test Proctor Icon
    2. Assessment Admin > Find a Test > Test Details > Actions > Proctor Dashboard
    3. Assessment Admin > Track Completion Rates > Locate Test > Locate Test > Data Collection Report Icon > Select a Section > Open Proctor Dashboard

    Non-Teachers:

    1. Assessment Admin > Track Completion Rates > Locate Test > Locate Test > Data Collection Report Icon > Select a Section > Open Proctor Dashboard
  4. Click Use Turning Tech Clickers from the menu on the right.

  5. Click Connect.
  6. Verify all ResponseCards (clickers) are on the same channel as the RRRF-02 receiver.
    1. The RRRF-02 receiver channel is listed above the Start Test button. To change the channel, click Change Channel.
    2. The Device Channel is displayed in the upper right of the screen when the ResponseCard (clicker) is active.

  7. Click the ResponseCard (clicker) icon next to each participant to assign a Device ID.

    The Device ID is a six-digit hexadecimal number found on the back of the ResponseCard (clicker) below the barcode. However, some sets of ResponseCards (clickers) may be pre-programmed 1 - 32. Pre-programmed sets have stickers on the back of the ResponseCards (clickers). In this instance, use the number on the sticker in a six-digit format. For example, a ResponseCard (clicker) pre-programmed to 2 would be entered as 000002.

  9. When finished, click Submit to send results for scoring.

    Click the Edit icon next to a participant to score any Open Response questions.

    If a Connecting screen appears, verify that the RRRF-02 receiver is connected. If it is connected, try plugging the RRRF-02 receiver into a different USB Hub.
