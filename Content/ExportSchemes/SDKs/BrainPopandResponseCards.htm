Turning Technologies® and BrainPop®

Turning Technologies’ ResponseCards and receivers can be used with BrainPop.

ResponseCards and BrainPop

BrainPop supports the use of RF, XR and NXT ResponseCards.

Using ResponseCards with BrainPop

A valid BrainPop account is required.

 

Turning Technologies Polling Support must be installed before using BrainPop. To download Polling Support, go to www.turningtechnologies.com/assetData/PollingSupport/PollingSupport_Mac.zip (Mac) or www.turningtechnologies.com/assetData/PollingSupport/PollingSupport_PC.zip (PC).

Installation Instructions for Mac:

  • Save the file to a specified location such as the Documents folder.
  • Double-click PollingSupport.dmg.
  • Move the icon into the Applications folder.
  • Double-click the icon to load the Turning Technologies Polling Support. The icon will appear in the dock.

Installation Instructions for PC:

  • Save the file to a specified location such as the Documents folder.
  • Right-click the folder and choose Extract All.
  • Double-click the extracted folder to locate PollingSupport.exe.
  • Double-click PollingSupport.exe to load the Turning Technologies Polling Support. An icon will appear in the system tray.

How to use ResponseCards with BrainPop...

  1. Plug in the receiver and open the Turning Technologies Polling Support.
  2. Login to BrainPop.
  3. Select a subject and then select a topic within that subject.
  4. Click Quiz.
  5. Select ResponseCard Quiz.

    If ResponseCard Quiz is not available, verify that the Turning Technologies Polling Support is loaded and the receiver is active.

  6. Click Yes to load a participant list, or No to run the quiz without a participant list.
  7. Optionally, enter a new receiver channel number. If the receiver channel is changed, all ResponseCards must be changed to the new channel number.
  8. Click Start Quiz.
  9. Set the Quiz Options and click Continue.
  10. Click Start Vote to open polling.

    To view participant responses, click the arrow next to Show Voters.

  11. Click Stop Vote to close polling.

    Click Correct Answer to indicate the correct answer.

    Click Answer Popularity to view a breakdown of count/percentage per answer choice.

    Click Class Charts, Individual Results or Cumulative Results to view group or individual results.

  12. Click Next Question. Repeat steps 10 - 12 for the remaining questions.
  13. Click Final Results when the quiz is complete. The results can be directly printed or downloaded to a CSV file and viewed in Microsoft Excel.
  14. Close the Turning Technologies Polling Support.
    • Mac - Click PollingSupport on the menu bar and select Exit Polling Support.
    • PC - Right-click on the Turning Technologies Polling Support icon in the system tray and select Exit.
