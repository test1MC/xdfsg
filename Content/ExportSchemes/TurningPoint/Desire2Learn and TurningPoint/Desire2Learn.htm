TurningPoint 5® and Desire2Learn

This document outlines the steps for using TurningPoint with the Desire2Learn Learning Management System (LMS).

How does the integration work?

Participants are instructed to register their response device through the Web Registration Tool hosted by Turning Technologies. The participant information from Desire2Learn is downloaded and converted into a TurningPoint participant list that is loaded before you begin your polling session. Once the session is run and the results are saved, it can be opened in Results Manager. In Results Manager, the participant list is updated with the latest registrations from the Web Registration Tool. Once the participant list is updated, the session scores are exported from TurningPoint in a specific format and then imported into Desire2Learn.

Downloading the Roster from Desire2Learn

  1. Login to Desire2Learn.
  2. Select the appropriate course below My Courses.
  3. Click Grades from the toolbar.
  4. Click Export Grades.
  5. Select Username for Key Field and Last Name and First Name for User Details on the Export Grades screen.
  6. Select an assignment to export under the Choose Grades to Export section.
  7. Click Export to CSV.

    A new window opens with a link to the download.

  8. Click the link, and when prompted, choose Save File.

    Now that the CSV file has been saved, it is time to import the information as a TurningPoint participant list.

Uploading Grades to Desire2Learn

  1. Login to Desire2Learn.
  2. Select the appropriate course below My Courses.
  3. Click Grades from the toolbar.
  4. Click Import Grades.
  5. Click Browse... to locate the saved file.
  6. Select Create new grade item when an unrecognized item is referenced and click Next.

  7. Select Numeric from the Type drop-down menu and click Next.

  8. Enter the maximum points for the assignment in the Max. Points field (default is 10) and click Next.
  9. Preview the import and click Import.

    The assignment now appears in the LMS.
