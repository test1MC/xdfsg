TurningPoint 5® TurningPoint® Cloud And PowerTeacher

This document outlines the steps for using TurningPoint Cloud with PowerTeacher.

How does the integration work?

The participant information from PowerTeacher is downloaded and converted into a TurningPoint participant list that is loaded before you begin your polling session. Once the session is run and the results are saved, it can be opened in Results Manager. Session scores are then exported from TurningPoint in a specific format and imported into PowerTeacher.

Downloading the Gradebook From PowerTeacher

  1. Open the PowerTeacher gradebook.
  2. Click the Scoresheet tab.
  3. Right-click the column header that corresponds to the assignment to which the TurningPoint Cloud results data will be uploaded.
  4. Select Export Scores Template.

  5. Save the file (.csv) to a specified location.

Uploading Grades to PowerTeacher

  1. Open the PowerTeacher gradebook.
  2. Select the class to which the results data will be added from the Classes section.

  3. Click the Scoresheet tab.
  4. From the Mode menu, click the Assignments button.
  5. Click the appropriate assignment column header to select the assignment column.
  6. Click Tools and select Import Scores. Optionally, right-click the assignment header to access the Import Scores option.
  7. Locate the saved CSV file.

    The Import Scores window opens.

  8. Click the Advanced tab.
    • Select the CSV radio button.
    • Make sure the Include First Row checkbox is checked.
    • Uncheck the Validate Student Name checkbox.
    • Choose Points from the File Score Type drop-down menu.
  9. Click Import.
  10. Verify that the scores have accurately imported to the Scoresheet.
