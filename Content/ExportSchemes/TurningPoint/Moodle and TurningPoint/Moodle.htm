TurningPoint 5® and Moodle™

This document outlines the steps for using TurningPoint with the Moodle Learning Management System (LMS).

How does the integration work?

Participants are instructed to register their response device through the Web Registration Tool hosted by Turning Technologies. The participant information from Moodle is downloaded and converted into a TurningPoint participant list that is loaded before you begin your polling session. Once the session is run and the results are saved, it can be opened in Results Manager. In Results Manager, the participant list is updated with the latest registrations from the Web Registration Tool. Once the participant list is updated, the session scores are exported from TurningPoint in a specific format and then imported into Moodle.

Downloading the Roster Center from Moodle

  1. Login to Moodle.
  2. Select the appropriate course below My Courses.
  3. Click Grades from the left Administration panel.

  4. Click the drop-down menu and select Export to a plain text file.
  5. Below Options, choose Comma as the Separator and click Submit.
  6. After previewing the rows, click Download.
  7. When prompted, choose Save File.

    Now that the TXT file has been saved, it is time to import the information as a TurningPoint participant list.

Uploading Grades to Moodle

  1. Login to Moodle.
  2. Select the appropriate course below My Courses.
  3. Click Grades from the left Administration panel.
  4. Click the drop-down menu and select Import-CSV file.
  5. Click Browse... to locate the saved file. Select Comma as the Delimiter Type and click Upload Grades.

  6. On the Import Preview screen, select Email from the Map drop-down menu and user email from the Map to drop-down menu. Below the Session drop-down menu, select new grade item.

  7. Click Upload Grades.
  8. Click Continue.

    The assignment now appears in the LMS.
