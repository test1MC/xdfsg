TurningPoint 5® and Sakai®

This document outlines the steps for using TurningPoint with the Sakai Learning Management System (LMS).

How does the integration work?

Participants are instructed to register their response device through the Web Registration Tool hosted by Turning Technologies. The participant information from Sakai is downloaded and converted into a TurningPoint participant list that is loaded before you begin your polling session. Once the session is run and the results are saved, it can be opened in Results Manager. In Results Manager, the participant list is updated with the latest registrations from the Web Registration Tool. Once the participant list is updated, the session scores are exported from TurningPoint in a specific format and then imported into Sakai.

Downloading the Gradebook from Sakai

  1. Login to Sakai.
  2. Click the appropriate course tab.
  3. Click Gradebook and select All Grades.
  4. Click Export as CSV.
  5. When prompted, choose Save File.

    Now that the CSV file has been saved, it is time to import the information as a TurningPoint participant list.

Uploading Grades to Sakai

  1. Login to Sakai and select the appropriate course tab.
  2. Click Gradebook and select Import gradebook item from spreadsheet.

  3. Click Upload Spreadsheet to Loading Dock.
  4. Name the new assignment.
  5. Click Choose File.
  6. Click Browse... to locate the saved file.
  7. Click Continue.
  8. Click Save.
  9. Verify the import and click OK.
  10. Click the Import link associated with the newly added assignment.

  11. Select the Points Received radio button, preview the import and click Import Selected.

  12. Enter the assignment name in the Title field and the point value in the Gradebook Item Point Value field.
  13. Click Submit.
  14. Click All Grades to view the assignment.
