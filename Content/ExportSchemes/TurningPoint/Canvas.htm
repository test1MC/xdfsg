TurningPoint 5® TurningPoint® Cloud and Canvas

This document outlines the steps for using TurningPoint Cloud with Canvas.

How does the integration work?

Participants are instructed to register their response device through the Web Registration Tool hosted by Turning Technologies. The participant information from Canvas is downloaded and converted into a TurningPoint participant list that is loaded before you begin your polling session. Once the session is run and the results are saved, it can be opened in Results Manager. In Results Manager, the participant list is updated with the latest registrations from the Web Registration Tool. Once the participant list is updated, the session scores are exported from TurningPoint in a specific format and then imported into Canvas.

Downloading Scores from Canvas

  1. Login to Canvas.

  2. Mouse over Courses and select the appropriate course.
  3. Click Grades from the left panel.
  4. Click the Options drop-down menu and select Download Scores (.csv).
  5. Open the CSV file in Microsoft Excel.
  6. Click File and select Save As.
  7. Name the file, browse to the desired save location and click Save.

    Now that the CSV file has been saved, it is time to import the information as a TurningPoint participant list.

Uploading Grades to Canvas

  1. Login to Canvas.
  2. Mouse over Courses and select the appropriate course.
  3. Click Grades from the left panel.
  4. Click the Options drop-down menu and select Upload Scores (.csv).
  5. Click Choose File, select the file and click Open.
  6. Click Upload Data.
  7. Select the assignment type from the drop-down menu and click Continue.

    Enter the points possible for the new assignment.

  8. Click Save Changes.

    The assignment now appears in the gradebook.
