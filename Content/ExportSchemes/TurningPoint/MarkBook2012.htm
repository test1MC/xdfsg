TurningPoint 5® TurningPoint® Cloud and MarkBook 2012

This document outlines the steps for using TurningPoint with MarkBook 2012.

How does the integration work?

The participant information from MarkBook 2012 is downloaded and converted into a TurningPoint participant list that is loaded before you begin your polling session. Once the session is run and the results are saved, it can be opened in Results Manager. Session scores are then exported from TurningPoint in a specific format and imported into MarkBook 2012.

Downloading the Roster from MarkBook 2012

  1. Open MarkBook 2012.
  2. Select the appropriate class from the Select a Class drop-down menu and click Open.
  3. Click File and mouse over Exports.
  4. Select ...to TurningPoint.

    If necessary, click Enter Device IDs to add Device IDs to the participant list.

  5. Click Save File.

    Now that the CSV file has been saved, it is time to import the information as a TurningPoint Cloud participant list.

Uploading Grades to MarkBook 2012

  1. Login to MarkBook 2012.
  2. Click the appropriate class from the Select a Class drop-down menu and click Open.
  3. Click New Entry.
  4. Enter the Title of the assignment.
  5. In the Out Of field, enter the total points possible for the assignment.
  6. Select Import Marks.
  7. Locate the file and click Open.
  8. Select the second value under the This field has the ‘marks’ drop-down menu.

    The first value is the Student Number. The second value is the point total received by the first participant.

  9. Uncheck The first record of this file is a ‘header.’ It does not have any data to be imported.
  10. Click Import.

    The assignment now appears in the gradebook.
