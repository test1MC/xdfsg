TurningPoint 5® TurningPoint® Cloud and ANGEL®

This document outlines the steps for using TurningPoint Cloud (versions 5.2 and above) with the ANGEL Learning Management System (LMS).

How does the integration work?

Participants are instructed to register their response device through the Web Registration Tool hosted by Turning Technologies. The participant information from ANGEL is downloaded and converted into a TurningPoint participant list that is loaded before you begin your polling session. Once the session is run and the results are saved, it can be opened in Results Manager. In Results Manager, the participant list is updated with the latest registrations from the Web Registration Tool. Once the participant list is updated, the session scores are exported from TurningPoint in a specific format and then imported into ANGEL.

Downloading the Gradebook from ANGEL

  1. Login to ANGEL.
  2. Select the appropriate course below Courses.
  3. Select the Manage tab.

  4. Click Gradebook below Course Management.
  5. Click the Export Grades link.
  6. Choose Comma Separated and click Export.
  7. Save the file to a specified location and click OK.
  8. Click Done.

    Now that the CSV file has been saved, it is time to import the information as a TurningPoint Cloud participant list.

Uploading Grades to ANGEL

  1. Log in to ANGEL.
  2. Below Courses, select the appropriate course.
  3. Select the Manage tab.
  4. Below the Course Management section, click Gradebook.
  5. Click the Import Grades link.
  6. Click Browse... to locate the saved file. Choose Comma as the Delimiter.

  7. Click Next.

    The Import Grades Screen is displayed.

  8. Adjust the drop-down boxes as necessary and click Finish.

    The assignment now appears in the LMS.
