TurningPoint 5® TurningPoint® Cloud and WebCT/Blackboard Vista™

This document outlines the steps for using TurningPoint Cloud with the WebCT/Blackboard Vista Learning Management System (LMS).

How does the integration work?

Participants are instructed to register their response device through the Web Registration Tool hosted by Turning Technologies. The participant information from WebCT/Blackboard Vista is downloaded and converted into a TurningPoint participant list that is loaded before you begin your polling session. Once the session is run and the results are saved, it can be opened in Results Manager. In Results Manager, the participant list is updated with the latest registrations from the Web Registration Tool. Once the participant list is updated, the session scores are exported from TurningPoint in a specific format and then imported into WebCT/Blackboard Vista.

Downloading the Grade Book from WebCT/Blackboard Vista

  1. Login to WebCT/Blackboard Vista.
  2. Select the appropriate course below Course List.

  3. Click the Teach tab.
  4. Select Grade Book from the Instructor Tools menu.
  5. Click the Export to Spreadsheet icon.

    The Export to Spreadsheet window is displayed.

  6. Select the Visible Columns radio button below Columns to Export.

    Make sure that the Last Name, First Name and User ID columns are visible.

  7. Select the Comma-delimited radio button below Export Format.
  8. Select the Unicode (UTF-8) radio button below Character Set.
  9. Click Export.
  10. When prompted, choose Save File.

    Now that the CSV file has been saved, it is time to import the information as a TurningPoint Cloud participant list.

Uploading Grades to WebCT/Blackboard Vista

  1. Login to WebCT/Blackboard Vista and select the appropriate course below Course List.
  2. Click the Teach tab.
  3. Select Grade Book from the Instructor Tools menu.
  4. Click the Import from Spreadsheet icon.
  5. Click Browse... to locate the saved file. Choose Comma as the Separator and Unicode (UTF-8) as the Character Set.

  6. Click Upload.
  7. Name the new assignment and click Import.

    The assignment now appears in the LMS.
