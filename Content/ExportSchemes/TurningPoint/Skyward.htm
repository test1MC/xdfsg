TurningPoint 5® and Skyward®

This document outlines the steps for using TurningPoint with the Skyward software.

How does the integration work?

Participant information is entered into a TurningPoint participant list that is loaded before you begin your polling session. Once the session is run and the results are saved, it can be opened in Results Manager. Session scores are exported from TurningPoint in a specific format and then imported into Skyward.

Prior to using the Skyward integration, the Skyward .tpex file must be added to your TurningPoint Schemes folder. Contact Customer Support (information below) to obtain the file.

Downloading the Roster

  1. Login to Skyward.
  2. Click the Teacher Access tab in the upper left corner.
  3. Click the My Classes link.
  4. Locate the appropriate class and click the corresponding Class Options link.

    The Class Options page is displayed.

  5. Select Reports.
  6. Expand the Class Information folder and select Class Roster.
  7. Click Export to Excel.

    The roster is exported.

Creating a Participant List

  1. Open TurningPoint and select the Manage tab.

  2. Click the Participant List drop-down menu and select New.

    The Create Participant List window opens.

  3. Name the new participant list, select Create Manually and click Create List.

    The Participant List Editor is displayed.

  4. Drag and drop the downloaded Skyward roster (Excel file) into the Participant List Editor.

    It is also possible to copy and paste text from an Excel file directly into the Participant List Editor.

  5. Press Enter on the keyboard to add another participant.
  6. Click Save and Close to save the participant list and return to the Participant List Overview screen.

Uploading Grades to Skyward

  1. Login to Skyward.
  2. Click Open Educator Access Plus.
  3. Located within the Teacher Access folder, click My Gradebook.
  4. Click the Gradebook link.
  5. Mouse over Assignments and select Advanced Export/Import Assignments.
  6. Select Import Scores extracted from TurningPoint.
  7. Select the assignment.

    If participants have responded to an “online” assignment, scores cannot be imported. This is to prevent results data from being overwritten.

  8. Select the XML file exported from TurningPoint and click Import Scores from File.

    The results now appear in the gradebook.
