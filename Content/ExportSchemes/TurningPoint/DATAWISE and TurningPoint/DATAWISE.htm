TurningPoint 5® TurningPoint® Cloud and DATAWISE

This document outlines the steps for using TurningPoint with DATAWISE.

How does the integration work?

The participant information from DATAWISE is downloaded and converted into a TurningPoint participant list. The DATAWISE test is downloaded as an XML file and converted into a TurningPoint question list. The participant list and question list are both loaded before you begin your polling session. Once the session is run and the results are saved, it can be opened in Results Manager. Session scores are then exported from TurningPoint in a specific format and imported into DATAWISE.

Exporting the Roster from DATAWISE

  1. Login to DATAWISE.

  2. Click Test Delivery and select Give a Clicker Test.
  3. Select the Grade, Period, Group and Group Teacher.

  4. Click Export Roster.
  5. Click Save.

Exporting a Clicker Test from DATAWISE

  1. Login to DATAWISE.

  2. Click Test Delivery and select Give a Clicker Test.
  3. Select the test.

  4. Click Export Test for SMART.
  5. Save the test to the desired location. The test will be downloaded as an XML file.

Importing the Test into TurningPoint Cloud

  1. Open TurningPoint and select the Content tab. Open TurningPoint Cloud and sign in to your Turning Account.
  2. Select the Content tab.
  3. Click Content and select Import.

  4. Select the DATAWISE test and click Open.
  5. Click Yes to convert the XML file into a TurningPoint Cloud question list.

    The question list is now listed in the left panel. The question list can not be edited in TurningPoint Cloud.

Uploading Grades to DATAWISE

  1. Login to DATAWISE.
  2. Click Test Delivery and select Give a Clicker Test.

  3. Click Choose File, select the file that was exported from TurningPoint and click Load File.

  4. Click User Selected or Create New to provide a test schedule.
