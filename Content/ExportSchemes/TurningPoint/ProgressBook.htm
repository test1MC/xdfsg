TurningPoint 5® TurningPoint® Cloud and ProgressBook™

This document outlines the steps for using TurningPoint Cloud with ProgressBook.

How does the integration work?

The participant information from ProgressBook is downloaded and converted into a TurningPoint participant list that is loaded before you begin your polling session. Once the session is run and the results are saved, it can be opened in Results Manager. Session scores are then exported from TurningPoint in a specific format and imported into ProgressBook.

Downloading the Report from ProgressBook

  1. Open ProgressBook.
  2. Select an existing class.

  3. Below Reports, select Student Demographics by Class and click Run.

    Pop-ups must be enabled to download reports from ProgressBook.

  4. In the Report Viewer window, select the class from the Class drop-down menu. Leave the Set Optional Filters section blank. Select ONLY the following options from the Columns drop-down menu:
    • Student Last Name
    • Student First Name
    • Student ID
  5. Click View Report.
  6. Select CSV from the drop-down menu after the report is generated.
  7. Click Export.
  8. Choose Save File when prompted.

    Now that the CSV file has been saved, it is time to import the information as a TurningPoint participant list.

Uploading Grades to ProgressBook

  1. Open ProgressBook.
  2. Select an existing class.
  3. Click Add an Assignment below Assignments & Marks. You may be prompted to create an assignment type.
  4. Enter a description of the assignment, the assignment type, date and the total possible points.

  5. Click Save.
  6. Click the Marks tab.

  7. Click Import the Marks.

  8. Click Browse to locate the saved file. Select No for First row contains headers?
  9. Click Import the Marks.
  10. Click the Save button in the upper right corner of the screen.

    The assignment now appears in the gradebook.
