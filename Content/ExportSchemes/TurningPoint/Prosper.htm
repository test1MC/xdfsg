TurningPoint 5® TurningPoint® Cloud and Prosper

This document outlines the steps for using TurningPoint Cloud with Prosper.

How does the integration work?

The participant information from Prosper is downloaded and converted into a TurningPoint participant list that is loaded before you begin your polling session. Once the session is run and the results are saved, it can be opened in Results Manager. Session scores are then exported from TurningPoint in a specific format and imported into Prosper.

Exporting Data from Prosper

  1. Login to Prosper.
  2. Select the Analysis & Reporting tab.

  3. Select a program from the Program drop-down menu.
  4. Choose the appropriate class and click Next.
  5. It is not necessary to select a Test for step 2. Click Next.
  6. Select the Learner Roster Report from the Report drop-down menu and click Display.

     

  7. Click the Export icon.
  8. Select Text for the Export Format and Comma for the Text Delimiter. Browse to a specified location (...) and name the file.

    Now that the CSV file has been saved, it is time to import the information as a TurningPoint Cloud participant list.

Uploading Grades to Prosper

  1. Login to Prosper.
  2. Select the Data Collection & Scoring tab.

  3. Select the class from the Class drop-down menu.
  4. Select the program from the Program drop-down menu and click Next.
  5. Select the appropriate test to score from the Tests drop-down menu and click Next.
  6. Click File, mouse over Import and select Test Data.
  7. Browse to the saved file location and click OK.
  8. Click Finish.
