TurningPoint 5® TurningPoint® Cloud and Brain Honey™

This document outlines the steps for using TurningPoint with Brain Honey.

How does the integration work?

The participant information from Brain Honey is downloaded and converted into a TurningPoint participant list. The participant list is loaded before you begin your polling session. Once the session is run and the results are saved, it can be opened in Results Manager. Session scores are then exported from TurningPoint in a specific format and imported into Brain Honey.

Exporting Data from Brain Honey

  1. Open Brain Honey.
  2. Select the appropriate course below Courses.

  3. Click Export Scores.

     

    The Export Scores window opens.

  4. Select First Name, Last Name and Username below Student Information.
  5. Name the file and select Comma-separated file (.csv) as the file type.
  6. Click Export.
  7. Save the file.

    Now that the CSV file has been saved, it is time to import the information as a TurningPoint participant list.

Uploading Grades to Brain Honey

  1. Login to Brain Honey.
  2. Click the appropriate course below Courses.
  3. Click Browse... to locate the saved file.
  4. Select Header Row and click Continue.

    The Import Scores window opens.

  5. Select the appropriate Header for each column and click Import.

    The assignment now appears in the gradebook.
