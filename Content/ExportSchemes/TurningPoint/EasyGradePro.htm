TurningPoint 5® TurningPoint® Cloud and Easy Grade Pro®

This document outlines the steps for using TurningPoint Cloud with Easy Grade Pro.

How does the integration work?

The participant information from Easy Grade Pro is downloaded and converted into a TurningPoint participant list that is loaded before you begin your polling session. Once the session is run and the results are saved, it can be opened in Results Manager. Session scores are then exported from TurningPoint in a specific format and imported into Easy Grade Pro.

Exporting the Gradebook from Easy Grade Pro

  1. Open Easy Grade Pro.

  2. Choose an existing Gradebook.
  3. Click File and select Export.
  4. Under the Built-in Formats tab, select Export Standard Grades.

  5. Click Export.
  6. When prompted, choose Save File.

    Now that the TXT file has been saved, it is time to import the information as a TurningPoint Cloud participant list.

Uploading Grades to Easy Grade Pro

  1. Open Easy Grade Pro.
  2. Choose an existing Gradebook.
  3. Click File and select Import.
  4. Below the Built-in Formats tab, select ‘Easy Import’ of Scores and click Next.
  5. Locate the saved file and click Open.

  6. Choose the assignment to receive scores.
  7. Adjust the second column drop-down menus to match the first column fields.
  8. Click Next.
  9. Click Import Records.
  10. Click OK.

    The scores are now added to the assignment.
