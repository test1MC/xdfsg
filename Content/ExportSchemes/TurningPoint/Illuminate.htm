TurningPoint 5® and Illuminate™

This document outlines the steps for using TurningPoint with Illuminate.

How does the integration work?

The student/participant information from Illuminate is downloaded and converted into a TurningPoint participant list. The Illuminate test is downloaded as an XML file and converted into a TurningPoint question list. The participant list and question list is loaded before you begin your polling session. Once the session is run and the results are saved, it can be opened in Results Manager. Session scores are then exported from TurningPoint in a specific format and imported into Illuminate.

Exporting the Roster from Illuminate

  1. Login to Illuminate.
  2. Click the Tab Delimited with Header Row icon under the Class Roster widget.

    The roster is downloaded.

Uploading Grades to Illuminate

 

An assessment must be created in Illuminate.

  1. Login to Illuminate.
  2. Click the Assessments link.
  3. Select Import Assessment from the drop-down menu.
  4. Click Choose File and browse to the saved TurningPoint session file.
  5. Click Open.
  6. Click Import.
