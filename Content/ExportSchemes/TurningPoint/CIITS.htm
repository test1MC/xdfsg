TurningPoint 5® TurningPoint® Cloud and CIITS

This document outlines the steps for using TurningPoint (versions 5.2 and above) with CIITS.

How does the integration work?

The participant information from CIITS is downloaded and converted into a TurningPoint participant list that is loaded before you begin your polling session. Once the session is run and the results are saved, it can be opened in Results Manager. Session scores are then exported from TurningPoint in a specific format and then imported into CIITS.

Downloading the Student Analysis from CIITS

  1. Login to CIITS.
  2. Mouse over the Classrooms tab and select Student Performance.

  3. Select the Student Analysis tab and click Advanced:create your own report.

  4. Select Student Demographics from the Analyze By drop-down menu.
  5. Select Student ID from the Analysis Groups drop-down menu.
  6. Click Add Column.
  7. Click Export to Excel.
  8. Save the exported file.
  9. Change the extension from .xls to .csv.

    To change the extension from .xls to .csv, click File > Save As. Select CSV.

Uploading Grades to CIITS

  1. Log into CIITS.
  2. Mouse over the Assessment Admin tab and select Find a Test.
  3. Select the test from Test Central.
  4. Click Score Test from the Actions panel on the left.
  5. Click Upload Scores under the test name from the Search Results panel on the left.
  6. Select SchoolNet from the Choose File Format drop-down menu.
  7. Click Choose File, select the CSV file from the previous section and click Open.

    The scores are now uploaded into CIITS.
