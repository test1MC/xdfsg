PowerPoint Polling

The Real-Time Registration Tool provides a fast and simple way to assign response devices to a participant list. Registration only takes a few moments, thus eliminating the need for manually adding Device IDs into the permanent participant list. The steps below can be performed before each presentation to ensure each participant has the appropriate response device.

  1. Open TurningPoint Cloud and sign in to your Turning Account.

  2. Select the participant list, the content file (optional) and click PowerPoint Polling.
  3. Click the Real-Time Registration Tool on the TurningPoint ribbon/toolbar.

    The Real-Time Registration Tool opens.

  4. Click Play to open registration.
  5. Each participant must press a number on their response device that corresponds with the number to the left of their name or User ID.

    The Device ID for the participant will appear in the Device ID column.

    If more than 10 participants are in the participant list, click the forward or back buttons to scroll page by page to see the remaining participants.

  6. Click the Pause button to close registration once all participants have responded.
  7. Close the Real-Time Registration tool.
  8. Run the polling session.

  9. When you are finished polling, click Save in the Session section of the TurningPoint ribbon/toolbar.

    Saving the PowerPoint presentation only does not save the session data!

    Session files should be stored in the default save location. If the session is saved in another location it will then need to be placed in the TurningPoint Sessions folder or imported into TurningPoint.

  10. Name the session file and click Save.
  11. Close PowerPoint to return to the TurningPoint Dashboard.
  12. Save the registrations by closing the Real-Time Registration Tool.

    The following message appears.

    To permanently save the Device IDs to the participant list, click Yes. Otherwise, click No. If you click No, the software will not remember Device IDs for future use.

    If No is selected, the participant list is temporarily saved.

    If the registrations were not saved to the permanent participant list and the session is reset or polling is closed, the Device IDs are not saved.
