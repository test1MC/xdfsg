The Gallery

Gallery

The WorkSpace Gallery tool provides more than 4000 resources to supplement your lesson planning. The Gallery stores images, background pages, teaching tools and Flash content called Interwrite Sims.

The WorkSpace Content component must be installed prior to using the Gallery. If you have not installed it, visit www.turningtechnologies.com/responsesystemsupport/downloads.

Before using the Gallery for the first time, you will be prompted to build the Gallery. Click Yes to build the Gallery database.
