Downloading and Installing WorkSpace for Linux

WorkSpace can be installed from a disc, www.turningtechnologies.com/responsesystemsupport/downloads or from a direct setup file.

 

Verify that your system meets the Technical Requirements.

  1. Save the ZIP file to a specified location.
  2. Extract the saved file.

    If using Ubuntu 12.04 LTS:

    1. Open the file.
    2. Click Run or Run in Terminal.
    3. If you accept the terms in the End User License Agreement (EULA), click Accept.
    4. Reboot your computer after the installation is complete.

    If using Ubuntu 14.04:

    1. Open Terminal.
    2. The "gksu" utility is required. Enter the following command to install it:
      • sudo apt-get install gksu
      • Press Enter.
    3. Drag the extracted file into Terminal.
    4. Press Enter.
    5. If you accept the terms in the End User License Agreement (EULA), click Accept.
    6. Reboot your computer after the installation is complete.

Additional dependencies may be required. The installer will attempt to download the dependencies, however a stable internet connection is required.
