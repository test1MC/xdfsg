Annotations, Page Sorter and Screen Capture

Using the Page Sorter

The Page Sorter provides thumbnail previews of the pages in your presentation.

  1. From the main toolbar, click the Page Sorter icon.
  2. Within the Page Sorter, you can:
    • Rearrange a page by dragging and dropping it into a new position.
    • Delete a page by selecting it and clicking the Recycle Bin icon.
    • Create a duplicate page by selecting a page, clicking the Page Sorter Menu icon and selecting Create Duplicate Page.

    Click the Page Sorter Menu icon to view other editing options.

  3. Click the X in the upper-rightleft corner of the window to close the Page Sorter.

Saving Annotations

By default, a WorkSpace file is dated, time-stamped and saved to the following location: Documents\Interwrite WorkSpace.

  1. Click the Main Menu icon.
  2. Mouse over File and select Save or Save As.

    The Save File window opens.

    • If Save, the file is saved as a GWB file and can only be opened in WorkSpace.
    • If Save As, the file can be saved as a GWB file or an IWB file (Common File Format).
