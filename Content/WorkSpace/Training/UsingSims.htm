Using Sims

  1. Click the Blank Page icon.
  2. Click the Gallery icon.

    If the Gallery icon is not on your toolbar, click the Main Menu icon, mouse over Tools and select Gallery.

  3. Select the Content tab in the Gallery window.
  4. Open the Sims Flash Objects folder.
    1. Select a topic.
    2. Select a Flash object.
    3. Drag the object onto the blank page or right-clickcontrol-click and add to Favorites.
