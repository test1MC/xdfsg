Annotating Over the Desktop

  1. Open WorkSpace.
  2. Click the Main Menu icon, mouse over Pages and select Annotate Over Desktop.

    The desktop becomes an annotation area.

    Alternatively, click the Annotate Over Desktop icon from the toolbar to annotate the area. Refer to Adding a Tool to the Toolbar for more information about adding a tool/icon to the main toolbar.

  3. Use any of the annotation tools to annotate the desktop area.

    To manipulate any of the windows or applications on the desktop, click the Selection tool on the WorkSpace toolbar.

  4. Click the Save icon on the WorkSpace toolbar to save the annotation.
