Enabling Whiteboard Mode

Whiteboard Mode must be enabled through Device Manager.

  1. Plug the Whiteboard USB cable into an available USB port on your computer. Alternatively, use the Wireless Kit. See Setting Up the DualBoard via Wireless Kit for more information.
  2. Select Whiteboard Mode from the Device Manager Tools menu.

    An image of the whiteboard is displayed on the desktop.

  1. Plug in the whiteboard receiver.
  2. Open Device Manager.
  3. Select Whiteboard Mode from the Device Manager menu.

    An image of the whiteboard is displayed on the desktop.
