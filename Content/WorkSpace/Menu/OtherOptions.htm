Preferences

The Preferences window can be accessed from the Main Menu icon.

From the Preferences window, Toolbar settings can be adjusted and customized, WorkSpace Connect can be enabled and language settings can be defined - among other options.

For more information on the Preferences window, see Preferences.

Student Response System

Use the Student Response System window to interact with participants using the Student Response System (SRS). Create, edit and pose questions during a session while participants respond using the SRS.
