Name

Start PowerPoint/Excel/Word

Start the selected Microsoft Office application.

Insert all annotations into the current PowerPoint slide, Word document or Excel spreadsheet.

Insert selected annotations into the current PowerPoint slide, Word document or Excel spreadsheet.

Add a blank slide to the current PowerPoint slide show. This feature is available only when a PowerPoint Slide Show is running.

Save as Slide in PowerPoint

Save the current annotated screen as a slide in the open PowerPoint presentation. The slide will be appended to PowerPoint file.
