Deleting a Note

You can delete a note from the computer screen or permanently delete the note from the Mobi 360.

  1. To delete a Note from the computer screen:
    1. With the Mouse Pointer tool, click X in the top corner of the note.

    Deleting a note from the computer screen does not delete the note from the Mobi 360.

  2. To delete a Note from the Mobi 360:
    1. In Note View, tap the Scroll Left icon or the Scroll Right icon to find the desired note.
    2. Tap the Delete Note icon.

    Deleting a note from Note View permanently deletes the note. There is no Undo for this action, and the deleted note cannot be retrieved.

  3. To exit Note View, tap Back.
