Changing a Class

When you tap Teach Class on the Mobi 360, the class that is normally scheduled for the current date and time appears. If this is not the class you want to start, you can change to a different class.

Change Class works only if you have not already started the class. If you have already started a class and want to change, you must end the class first.

  1. Tap Teach Class.

    The currently scheduled class name, time and period is displayed.

  2. Tap Change Class.
  3. Tap the class you want to start.

    Tap down to view more classes.

  4. Tap Start Class.
