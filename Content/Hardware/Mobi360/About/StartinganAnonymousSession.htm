Starting an Anonymous Class Session

An anonymous class session can be used when you want to start an impromptu session or when a class has not been pre-configured in Insight 360.

During an anonymous class student devices are not associated with specific students. Insight 360 will save data for an anonymous class session and you can associate the data to a class after the session ends.

  1. From the Mobi 360 Home Screen, tap Teach Class.
  2. Tap Anonymous.
  3. Tap Start Class.
