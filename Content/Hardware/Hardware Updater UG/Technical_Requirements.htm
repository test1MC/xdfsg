Technical Requirements

The following are required for the Hardware Updater to properly function.

  • Microsoft Windows 32-bit or 64-bit XP, Vista, 7 or 8
  • Intel or AMD 600 Mhz class processor (1 GHz or higher recommended)
  • 256 MB RAM (256 MB or more of available RAM recommended)
  • 100 MB hard disk space (an additional 32 MB is required if Microsoft .NET Framework 2.0 is not installed)
  • Standard USB 2.0 port
  • Ethernet or 802.11 compatible wireless network card
  • Active Internet connection

Compatible Devices:

  • ResponseCard XR
  • ResponseCard NXT
  • ResponseCard AnyWhere
  • ResponseCard Programmer
  • RF Receiver (RRRF-02)
  • RF+ Receiver (RRRF-03)
  • RF Receiver (RRRF-04)
  • Proctor Receiver (RRRF-03-P)

    If using a firewall, be sure to allow updates.turningtechnologies.com permission to connect.
