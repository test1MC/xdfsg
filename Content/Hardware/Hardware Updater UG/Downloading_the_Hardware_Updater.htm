Downloading the Hardware Updater

Visit the Turning Technologies website to download the Hardware Updater.

Using an older version of the software may result in new ResponseCard firmware reverting to old firmware

  1. Go to www.turningtechnologies.com.
  2. Click on the Downloads icon in the upper right corner of the home page.
  3. Select Hardware Updater
  4. Enter the required information on the Software Downloads Registration and click Submit.
  5. Click the Download icon to start the download.
  6. Click Accept to agree to the terms and conditions or click X to cancel the download.
  7. When prompted to Save the setup file, it is recommended to save the file to a recognizable specified location (ie, Desktop or flash drive).
