Connecting via a Proxy Server

Some networks require the use of a proxy server to connect to external servers. If you are unsure if a proxy connection is required, contact your IT Administrator.

  1. Open the Hardware Updater.
  2. Select Connect using proxy server.
  3. Click Proxy Settings.
  4. Select Automatically detect proxy settings or Manually configure proxy settings.
  5. If entering proxy settings manually, enter the proxy server address and port number. If the proxy server requires authentication, select My proxy requires authentication and enter the Username, Password and Domain.
  6. Click Test to test the connection.
  7. Click Done.
