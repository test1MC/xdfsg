Updating a ResponseCard

ResponseCard XR, ResponseCard NXT, ResponseCard AnyWhere and the ResponseCard Programmer can receive firmware updates via the Hardware Updater. A mini USB cable is required for the firmware update.

A standard 4-pin Type A to a 5-pin Type B mini USB cable can be purchased from the Turning Technologies Student Store or at any major electronics retailer.

  1. Close out of all other Turning Technologies applications and unplug any other Turning Technologies hardware.
  2. Open the Hardware Updater.
  3. Connect the ResponseCard to the PC with a mini USB cable.

    Only plug in one device at a time.

    The version number will turn black if the device does not have the latest firmware version.

  4. Select the desired language.
  5. Click Update Firmware.

    DO NOT UNPLUG THE DEVICE DURING THE UPDATE!

  6. Unplug the device only after the prompt indicates that it is safe to do so.

    The device now has the latest firmware.
