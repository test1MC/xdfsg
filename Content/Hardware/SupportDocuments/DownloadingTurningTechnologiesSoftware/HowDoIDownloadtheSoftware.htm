How do I download the software?

  1. Go to www.turningtechnologies.com.
  2. Click the downloads tab at the top of the screen.

  3. Click the title of the software to expand the download choices.
  4. Click the link for the desired software. You will be taken to the Software Downloads Registration page.
  5. Enter the required information in the fields provided and click Submit.
  6. Click the download link to start downloading the software.
  7. Click Accept if you accept the End User License Agreement. You will not be able to download the software if you do not accept.

    If you do not see the Accept button, you may need to set your Windows Display Settings to 100% or adjust the zoom setting on your internet browser.

  8. When prompted, save the file to your computer.
  9. When the file has finished downloading you will need to install or open the program based on the type of file you chose to download.

Opening the No Install PC Version of the Software

  1. Double-click the downloaded zip file.
  2. Select the desired folder location and click Unzip.
  3. Open the unzipped folder and double-click TurningPoint.exe.

    DO NOT remove any files from the Application folder. If these files are removed the application will not function properly.

Opening the No Install Mac Version of the Software

  1. Double-click the downloaded DMG file.
  2. Drag and drop the software icon into the Applications folder.

    It is recommended to create an alias and add it to the Dock.

  3. Double-click the software to open it.

Installing the PC Version of the Software

  1. Double-click the downloaded setup file.
  2. Click Next. A window opens and displays the license agreement. Read the license agreement on the screen or click the Print button to send the text to a printer.
  3. Select the radio button labeled I accept the terms in the license agreement if you accept the terms.
  4. Click Next.
  5. Enter the name and organization in the fields provided and click Next.

    A window opens and displays a message that the computer is ready to install TurningPoint.

  6. Click Install. The computer installs TurningPoint and displays the Finish button on the page when the process is complete.
  7. Click Finish. Select the Launch TurningPoint checkbox to immediately open the application, or double-click the TurningPoint shortcut on the desktop to open the application.
