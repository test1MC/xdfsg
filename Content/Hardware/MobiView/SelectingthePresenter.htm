Selecting the Presenter

The “presenter” controls when Multi-User Mode begins and ends.

  1. Click the Turning Technologies icon from the notification area (PC)/Menu Extras area (Mac)/Dash Home (Linux).
  2. Select eInstruction Device Manager from the menu.

    Device Manager opens.

  3. Select the icon for the desired presenter device.
  4. From the menu bar, click Options and select Set Presenter.

    The selected device is now the presenter device.
