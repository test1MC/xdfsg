Accessing Settings

The following can be accessed from the Settings icon:

  • Auto Start - select the software that will open when the MultiHub is plugged in
  • Sleep Timer - increase or decrease display time
  • Calibrate Touch Screen - calibrate the touch screen when using Notes
  • Pair with Hub - connect MobiView to the MultiHub
  • Brightness - set the touchscreen brightness level
  • About - view the firmware and software versions
  • Language - select the MobiView language
  • Device Name - enter a name for the MobiView (up to eight characters)
