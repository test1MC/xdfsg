Annotating Content

Access WorkSpace tools to annotate content and collaborate with multiple users from anywhere in the classroom. The tools displayed on the Annotate screen can be customized.

Refer to the WorkSpace for PC User Guide or WorkSpace for Java User Guide for help using WorkSpace.

How to annotate content...

  1. WorkSpace opens in Lesson Mode upon selecting an annotation tool. Tap the Pen icon on the WorkSpace toolbar.
  2. Write on the MobiView’s active area to annotate the current screen or tap the Blank Page icon on the WorkSpace toolbar to annotate on a new page.
    • Use the WorkSpace toolbar tools and the WorkSpace Properties Bar to adjust the pen options.

How to customize the WorkSpace tools on the MobiView...

  1. Tap Annotate.
  2. Tap Customize Tools.

    Device Manager opens on your computer.

  3. Select a function row.
  4. Select the desired tool from the MobiView WorkSpace Annotation Tools drop-down menu.
  5. Click OK.
