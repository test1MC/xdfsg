Using Turning Technologies Receivers with CPS, Insight 360 and TurningPoint

When you have CPS, Insight 360 and TurningPoint 5 installed on your computer you will need to manually tell the receiver which program you are using. Only one application at a time can communicate with the receiver.

Close the CPS, Insight 360 and TurningPoint 5 applications before proceeding.

Using the Receiver with CPS

  1. Plug in the receiver.
  2. Close Insight 360 Director. (Press Ctrl + Shift + Q on your keyboard.)

    The Insight 360 Director starts every time you start your computer. The Insight 360 Director also automatically takes control of the Turning Technologies receiver.

  3. Click Device Manager from your system tray and select eInstruction Device Manager.

    The Device Manager application opens.

  4. From the Device Manager window, click File and select Preferences.
  5. Check the option labeled Scan for ResponseCard Devices and click OK.

    The Turning Technologies receiver is now connected.

  6. Open CPS.

Using the Receiver with Insight 360

  1. Plug in the receiver.
  2. Click Device Manager from your system tray and select eInstruction Device Manager.

    The Device Manager application opens.

  3. From the Device Manager window, click File and select Preferences.
  4. Uncheck the option labeled Scan for ResponseCard Devices and click OK.

    The Turning Technologies receiver is now disconnected.

  5. Open Insight 360.
