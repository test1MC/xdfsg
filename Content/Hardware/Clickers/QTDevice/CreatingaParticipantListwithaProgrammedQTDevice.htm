Creating a Participant List in TurningPoint 5 with a Programmed QT Device

Program Your ID on the QT Device

  1. Wake up the QT Device.
  2. Press the .
  3. Press the right arrow and navigate to Your ID .
  4. Press Enter.
  5. Enter the number corresponding to the sticker on the back of the device.
  6. Press OK.
  7. A checkmark on the screen indicates that the ID was successfully saved.
  8. Press twice to exit the Toolbox.

Creating a Participant List in TurningPoint 5

  1. Open TurningPoint 5.
  2. Click the Participant List drop-down menu and select New.

    The Create Participant List window opens.

  3. Name the participant list, select the Create Manually radio button and click Create List.

    The Participant List Editor is displayed.

  4. Select a cell and enter the participant information relevant to that particular category. For example, enter the participant’s last name under the Last Name column.

    Leave the Device ID column blank.

  5. In the User ID column, enter a number for each participant. For example, the first participant should be number 1, the second number 2, etc.
  6. Press Enter on the keyboard to add another participant.
  7. When finished, click Save and Close to save the participant list and return to the Participant List Overview screen.
