Setting the Program Key

The ResponseCard Programmer uses a program key to program all ResponseCards to their new Unit ID. The Program Key (0/J - 9/I on the ResponseCards) is the key that will be used to program the ResponseCards. It should be a key that is uncommonly used in the surrounding environment. 9/I is the default Program Key.

  1. Press Menu/OK to return to the Main Screen.
  2. Use the down arrow to highlight Setup.
  3. Press Menu/OK.
  4. Highlight Program Key and press Menu/OK.
  5. Use the up and down arrows to highlight a Program Key.
  6. Press Menu/OK to set the Program Key.
