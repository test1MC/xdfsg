Setting the Screen Contrast

The screen contrast on the ResponseCard Programmer can be adjusted.

  1. Press Menu/OK to return to the Main Screen.
  2. Use the down arrow to highlight Setup.
  3. Press Menu/OK.
  4. Use the down arrow to highlight Contrast.
  5. Press Menu/OK.
  6. Use the left and right arrows to adjust the contrast.
  7. Press Menu/OK to set the contrast.
