ResponseCard NXT, Pulse
and CPS

This guide is designed to familiarize you with some helpful tips for using ResponseCard NXT, Pulse and CPS.

 

ResponseCard NXT

Pulse

Sticker ID

NXT must have a programmed ID and not be on the factory ID.

Pulse clickers always operate on programmed ID (pad ID) and do not have the option to use factory ID.

Presentation Mode Stop Polling

If a participant attempts to answer a question but does not send the response before polling closes, the response will still be displayed on the ResponseCard until the participant sends it or backs out of the question/test. If the participant sends the response, a “Not Open” message will be displayed.

If participant attempts to answer a question but does not send the response before polling closes, the response is cleared on the clicker.

Self-Paced
(Excluded Question)

Questions appear as multiple choice.

Questions are skipped over.

Self-Paced
(End Test)

It is recommended that participants do a “final send” on the NXT prior to ending the test in the software. If a participant does not send the final test, the test must be deleted on the NXT card from the settings option.

The test automatically ends.

Join Class Count

Increases upon participant sending response.

Increases upon device powering on and connecting to receiver.

Multiple Choice
Question

NXT displays "Not an option" message when invalid response is selected.

Invalid response(s) buttons are disabled.

Answer Series
Question

NXT does not mask invalid responses. Response grid will show red “X” when invalid response is sent. Invalid responses are not recorded.

Invalid response(s) buttons are disabled.

Yes/No Question

Shows as True/False.

Shows as A/B.

True/False Question

Shows as True/False.

Shows as A/B.

Short Answer
Question

NXT allows for an answer up to 16 characters in length.

Pulse allows for an answer up to 20 characters in length.

Essay Question

NXT allows for an answer up to 160 characters in length. CPS only supports 140 character answers. For responses over 140 characters, the response grid will show a red “X” when answer is sent. Answer will not be recorded.

Pulse allows for an answer up to 140 characters in length.

Numeric Question

NXT allows for an answer up to 16 characters in length. For responses over 12 characters, the response grid will show a red “X” when answer is sent. Answer will not be recorded.

Pulse allows for an answer up to 12 characters in length.
