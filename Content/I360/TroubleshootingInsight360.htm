Troubleshooting Insight 360 Cloud

Insight 360 Cloud Does not Open

If the Insight 360 Cloud software does not open, check the following:

  • Double-click the Insight 360 Cloud icon on the classroom computer desktop.
  • Is another instance of Insight 360 Cloud already open, but hidden behind other windows on your computer?
  • If there is no icon on the desktop, was the software installed correctly or were there any errors? Contact your IT Administrator if you experienced errors during installation.
  • Did you have Admin (administrative) rights on your computer when you installed Insight 360 Cloud? Contact your IT Administrator if you are unsure.
  • Is Insight 360 Cloud listed as an installed program on your Start menu (Windows) or in your Applications folder (Mac)?
  • Does your computer have an anti-virus or firewall software that would prevent you from installing new software? Be sure you disabled any anti-virus software on your system before installing Insight 360 Cloud. Contact your IT Administrator if you are not sure whether your system has anti-virus or firewall software.

Possible Solutions

  • Uninstall and reinstall the Insight 360 Cloud software.Temporarily shut down your anti-virus software before installing Insight 360 Cloud. Contact your IT Administrator about firewalls and security permission required to install new software.
 Instructor Device Does not Connect

If your instructor device (such as a tablet) does not connect to the Insight 360 Cloud system, check the following:

  • You are using the Insight 360 Teacher app downloaded from the appropriate app store.
  • The Enable Insight 360 Teacher app check box is selected in My Settings > Mobile Devices.
  • The computer running Insight 360 Cloud is shown in the Pair Computer (iOS) Pair Computer and Tablet (Android) list.
  • An Insight 360 Teacher app is not already connected. Only one Insight 360 Teacher app can be connected to Insight 360 Cloud at a time.
  • The password used to pair the computer and device is the same as the password typed into My Settings > Mobile Devices.

Possible Solutions

  • Download and install the Insight 360 Teacher app from the appropriate app store.
  • Enable the device in Insight 360 Cloud.
  • Pair the device with the computer running Insight 360 Cloud. Create a new password if necessary.
  • Manually connect the device with the IP address. Contact your IT Administrator if you are not sure how to do this.
 Student Application Does not Connect

If an Insight 360 Student app does not connect to the Insight 360 Cloud system, check the following:

  • The classroom computer is running Insight 360 Cloud Mobile Edition.
  • The instructor’s Insight 360 Teacher app has been correctly set up.
  • The student device has downloaded and installed the Insight 360 Student app from the appropriate app store.
  • The session code, Student ID or Sticker ID used to connect to the session is correct.
  • If instructor started an anonymous class, the student selected Join as Guest User (may have to tap Back and then Join as Guest User).
  • The number of allowable connected student devices (100 as of Release 2.0) has not been exceeded.
  • The student was not denied upon attempting to rejoin the class.

Possible Solutions

  • Click About in the Insight 360 Cloud software on the classroom computer to determine which edition of the software is installed. If Edition shows anything other than Mobile, contact your sales representative to purchase Mobile Edition or an Insight 360 Cloud Mobile USB Stick, which provides Mobile functionality while the USB stick is inserted into a computer with Insight 360 Cloud.
  • Download the Insight 360 Student app from the appropriate app store.
  • Disconnect any non-essential student devices from the session (power off or use the Insight 360 Teacher app to disconnect Student applications).
  • In the Session Running dialog box displayed on the classroom computer, deselect and reselect the Require Session Code to Join box. Ensure the correct Session Code is entered in the Session Code box when the student is attempting to join.
  • Restart the class.
 Mobi 360 Does not Connect

If the Mobi 360 does not connect, check the following:

  • The Mobi 360 is turned ON using the blue power button on the top edge of the device.
  • The Mobi 360 receiver is inserted into a working USB port on the computer running the Insight 360 Cloud software.
  • The Mobi 360 battery icon shows at least one bar or charging. If the battery icon shows red, charge the Mobi 360.
  • Does your computer have an anti-virus or firewall software that would prevent communication between the Mobi 360 and the Insight 360 Cloud software? Check with your IT Administrator if you are unsure.
  • If you are running other Turning Technologies devices, make sure the Mobi 360 and Mobi 360 receiver do NOT show up in Device Manager.

The Device Manager is a separate program that is NOT used in Insight 360 Cloud but is similar to Insight 360 Cloud’s Device Management section in My Settings.

Possible Solutions

  • Charge the Mobi 360 using the charging stand. You can use the Mobi 360 while it is charging.

    Do NOT plug the charging stand into a computer to charge the Mobi 360. Either plug the Mobi 360 into the computer using the USB cable or plug the charging stand into a standard wall outlet.

  • Update firmware on the Mobi 360 and/or the Mobi 360 receiver.
  • Close Device Manager (right-click on the Device Manager icon and select Exit).

Mobi 360 Unable to Connect Error Message

If you see the message “Mobi 360 is unable to connect…” check the following:

  • The Mobi 360 receiver is inserted into a working USB port on the computer.

    Many USB receivers look the same. Ensure that you have the Mobi 360 receiver and not a receiver from another Turning Technologies product.

  • The computer is the same computer on which the Insight 360 Cloud software is installed.
  • The computer is turned ON.
  • Both the Mobi 360 Receiver and the Mobi 360 are listed as Connected in Device Management.
 Student Clicker Does not Connect

Student clickers cannot connect to Insight 360 Cloud unless a class has been started with the Mobi 360 or Insight 360 Teacher app. If a class has been started, clickers with the same Join Code as entered in Device Management should automatically connect to the class session.

Multiple Clickers Unable to Connect

If none of the clickers in your classroom can connect to Insight 360 Cloud, check the following:

  • The appropriate receiver is inserted into a working USB port on the classroom computer that is running the Insight 360 Cloud software.

    When you are running Insight 360 Cloud with a Mobi 360 and the clickers, TWO receivers must be inserted into USB ports on the same computer: the Mobi 360 receiver and the clicker receiver.

  • The Join Code as entered in Device Management is the same as the Join Code entered on the individual clickers.
  • The clickers are Pulse 360, ResponseCard NXT or QT Device clickers, or have been upgraded to work with the Insight 360 Cloud system.

    If a clicker receiver was recently upgraded, the firmware on the individual clickers must be updated. The update process starts in Device Management and must be completed for each clicker. Clickers will not connect until the firmware update is finished.

Single Clicker Unable to Connect

If only one or two clickers are unable to connect, but others are able to connect, check the following:

  • The clicker is turned ON.
  • The battery level indicator shows at least one bar. If the battery indicator is empty or the device cannot turn ON, replace the batteries.
  • The Join Code entered on the clicker is the same as configured in Device Management.
  • The Sticker ID shown in the clicker’s menu is the same as the number on the sticker, and is also the same as the Sticker ID configured for the student on the My Classes tab.
  • If joining by Student ID, ensure the Student ID entered is the same as the Student ID configured for the student on the My Classes tab.
  • The clicker has the latest firmware and does not need updating. If there is a triangular icon in the device’s row in Device Management, the clicker requires a firmware update.

    The update process must be completed for each clicker. Clickers will not connect until the firmware update is finished.

 Pulse/Spark 360 Receiver Not Found Error

The "Pulse/Spark 360 Receiver not found" error message displays on the Mobi 360 if Insight 360 Cloud cannot communicate with the Pulse/Spark 360 receiver when you start a self-paced test or Verbal or Tagged question. Check the following:

  • The Pulse/Spark 360 receiver is inserted into a working USB port on the classroom computer.

    Many USB receivers look the same. Verify that you have the Pulse/Spark 360 receiver and not a receiver from another Turning Technologies product. When running Insight 360 Cloud with the optional student response system, you should have TWO receivers inserted into USB ports on your computer: the Mobi 360 receiver and the Pulse/Spark 360 receiver.

  • The computer is the same computer on which the Insight 360 Cloud software is installed.
  • The computer, clickers and Mobi 360 are all turned ON.
  • Both the Pulse/Spark 360 receiver and one or more clickers are listed as "Connected" in Device Management.
  • Pulse/Spark 360 receiver has the most current firmware update.
 Firmware Update is Available Message

Periodically you will see the message "One or more of your devices has a firmware update available."

This message indicates that updates are available for download to your Insight 360 Cloud system. The update could be for your Mobi 360, Mobi 360 receiver, clickers or clicker receiver. You may continue using the Insight 360 Cloud system now and update the firmware at a later time.

 How Do I Know if the Receiver is Working?

A blue light on the receiver glows when it is fully inserted and turned ON. On the Mobi 360 receiver, the blue light is in the middle of the receiver. On the Pulse/Spark 360 receiver, the blue light is at the end closest to the USB port.

 Insight 360 Cloud Does Not Show My Classes or Students

If you cannot see any classes or students that you had previously configured in the Insight 360 Cloud software, check the following:

  • The Insight 360 Cloud database is installed on the local machine rather than on a network.
  • You did not move the Mobi 360 to another classroom.

  • Ensure the database was not moved and that permissions were not changed.
  • Your Insight 360 Cloud software is activated and has been recently updated.

Contact your IT Administrator for more guidance on these topics.

 My Pen Does Not Work

If your Mobi 360 digital pen does not work, check the following:

  • The red protective tape has been completely removed from the battery section of the digital pen. The thin plastic tape keeps the battery from discharging during storage and shipment. It must be completely removed upon hardware set up. Occasionally a small piece of tape remains behind. To determine whether any tape is stuck between the pen sections, gently press the tab on the back of the pen, slide the two sections apart, and check around the battery for remnants of red tape.
  • The digital pen proximity indicator shows on the Mobi 360 when the pen tip nears the Mobi 360 writing surface. If this indicator is red, the digital pen needs recharging. Place the digital pen in the Mobi 360 holder, ensuring it clicks into place. The pen icon on the Mobi 360 shows charging when the pen is correctly situated. You can use the Mobi 360 while charging the pen, but should place the Mobi 360 on its charging stand when it is not in use to charge both the Mobi 360 and its digital pen. The digital pen requires eight hours to fully charge a drained battery.
  • If the digital pen proximity indicator does not show at all, the pen battery may need to be replaced.
 Student Clickers Do Not Receive Feedback

There are two ways to send feedback to clickers:

  • To always send feedback at the end of Verbal, Tagged, or ExamView questions, select the Send correct/incorrect feedback to student devices at the end of each question checkbox in Preferences.
  • To send scores at the end of self-paced tests, select the Send scores to response pads at the end of the test option when starting the self paced test in My Tests.

If these options are selected and clickers do not show feedback during these occasions, check the following:

  • Ensure the Pulse/Spark 360 receiver is plugged into a USB port on the computer running the Insight 360 Cloud software.
  • Clickers and Pulse/Spark 360 receiver firmware is current and does not need to be updated. If the firmware needs updated, there will be a message on the Mobi 360 and in the Message Center on the Insight 360 Cloud Home page.
  • If only one clicker does not receive feedback, ensure the student entered the proper Join Code and Sticker ID or Student ID (if required). If these are correct, you will be able to see the student’s response on the Student Performance Table in the Results Zone on the Mobi 360 for Verbal or Tagged questions. For self-paced tests, you can look up the student’s score on the Session Response Grid report in My Results in the Insight 360 Cloud software.
 Computer Does Not Meet Minimum Requirements Error

When you turn ON the Mobi 360 mobile interactive whiteboard for the first time, it connects to the Insight 360 Cloud software to confirm system specifications. If Insight 360 Cloud detects that your computer does not meet minimum requirements, a message will appear on the Mobi 360 screen.

Tap the error message on the Mobi 360 screen to view system deficiencies. Contact your IT Administrator for assistance in correcting system deficiencies.
