Technical Requirements

Windows
  • Microsoft Windows® 7 or 8 (32- or 64-bit)
  • Intel or AMD 600 MHz class processor (1 GHz or higher recommended)
  • 2 GB RAM
  • 4 GB hard disk space for installation
  • 1024 x 768 resolution at 32-bit color or higher
  • Adobe® Acrobat® Reader version 9.0 or equivalent PDF reader
  • DVD Drive with dual layer support for installation
  • Standard USB 2.0 port (2)
  • Active internet connection
  • Turning Technologies Device Manager 6.71 or higher for users running other Turning Technologies software
 Mac
  • Mac® OS X 10.7, 10.8, 10.9 or 10.10
  • Intel processor
  • 2 GB RAM
  • 4 GB hard disk space for installation
  • 1024 x 768 resolution at 32-bit color or higher
  • Adobe® Acrobat® Reader version 9.0 or equivalent PDF reader
  • DVD Drive with dual layer support for installation
  • Standard USB 2.0 port (2)
  • Active internet connection
  • Turning Technologies Device Manager 6.71 or higher for users running other Turning Technologies software
 Insight 360 Cloud Teacher app
  • iPad® with iOS 6 or higher
  • Android™ tablet with OS 4.0 or higher and 7 inch minimum screen
  • It is recommended that the network support the use of Internet Group Management Protocol (IGMP)
 Insight 360 Student app
  • iPad with iOS 6 or higher
  • Android tablet with OS 4.0 or higher
  • Local high-speed, enterprise class Wi-Fi connection for tablet
  • It is recommended that the network support the use of Internet Group Management Protocol (IGMP)
