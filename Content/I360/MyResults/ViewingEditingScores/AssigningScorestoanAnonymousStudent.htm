Assigning Scores to an Anonymous Student

Occasionally you may have a student in class who is not assigned to the Insight 360 Cloud class roster. If that student answers questions with a student response device, Insight 360 Cloud will assign responses from that device as anonymous.

  • Anonymous students appear with blank student names in My Results
  • You can assign anonymous results to actual students in the Scores section of My Results
  • Before you can assign a student to the anonymous scores, the student must be added to the class roster
  2. Click next to the device for which you want to assign the scores.

    The Assign Anonymous Student box opens.

    If the Assign Anonymous Student box is empty, you must add the student to the class roster.

  3. Click the student name to assign to the device.
  4. Click Accept.

    Assigning the student to the Device ID for this session does not affect past or future Device ID assignments. To permanently change future Device ID assignments, change the ID in My Classes. For more information, see Editing Student Details.
