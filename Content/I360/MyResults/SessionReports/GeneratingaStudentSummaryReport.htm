Generating a Student Summary Report

A Student Summary report allows you to view proficiency data based on each individual student.

The Student Summary report is divided into seven columns consisting of:
  • Student name
  • Student ID
  • Number of questions answered
  • Number of questions answered correctly
  • Number of questions answered incorrectly
  • Number of questions not responded to
  • Score
