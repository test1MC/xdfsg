Generating a Response Summary Report

A Response Summary report allows you to view proficiency data based on the percentage of correct answers.

The Response Summary report is divided into four columns consisting of:
  • Question
  • Correct answer
  • Percentage correct
  • Number of students who did not respond
  • Other responses
