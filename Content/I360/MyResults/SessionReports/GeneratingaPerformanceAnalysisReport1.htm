Generating a Performance Analysis Report

A Performance Analysis report allows you to determine each student’s proficiency for a specific test session.

The Performance Analysis report can be shown in chart or grid format. The chart provides the following information for the selected test session:
  • Class average score
  • Number of questions
  • Each student’s individual score color-coded in green (at or above proficiency level) or yellow (below proficiency level)

In addition to the information displayed on the chart format, the grid version of this report provides the Student ID and each student’s exact score.
