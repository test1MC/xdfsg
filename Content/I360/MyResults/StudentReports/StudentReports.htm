Student Reports

Student reports provide summary information on a student’s progress towards meeting educational objectives. Insight 360 Cloud provides four types of Student reports:

  • Performance Summary - The Performance Summary report provides student proficiency data as compared to the class average over multiple class sessions.
    • Options include choosing a student, setting a date range and changing the proficiency setting
    • Available in grid or chart views
    • Each class session is listed on a separate line (grid view) or bar (chart view)
    • Compares the student’s score to the class average and proficiency setting
    • On chart view only – designates Answer Key tests with Key
    • Can be exported to CSV file format
  • Session Results - The Session Results report provides an overview of a selected student's proficiency for a particular class session.
    • Options include selecting the student and displaying only incorrect answers
    • Provides the Class Average Score and Student Score
    • Can be exported to CSV file format
  • Standards Analysis - The Standards Analysis report provides an overview of student’s proficiency toward a single US state educational standard.
    • Options include choosing the student and standard, setting a date range and changing the proficiency setting
    • Displays only the sessions with test or question results aligned with the selected standard
    • Available in grid or chart views
    • Each class session is listed on a separate line (grid view) or bar (chart view)
    • Shows the student’s proficiency on questions linked to the selected standard in each session
    • On chart view only – designates Answer Key tests with Key
    • Can be exported to CSV file format
  • Multiple Standard Analysis - The Multiple Standards Analysis report provides the proficiency percentage of each student answering test questions that align with multiple US educational standards.
    • Options include selecting a student, setting a date range and changing the proficiency setting
    • Displays only the sessions with test or question results aligned with standards
    • Lists percentage of students above and below proficiency level for each standard in each session
    • Can be exported to CSV file format
