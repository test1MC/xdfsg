Connecting the Insight 360 Student App

You may join the class session using a Sticker ID or Student ID, depending on the configuration your teacher selected.

  1. Turn on your device and tap .

    The Connect to Desktop screen is displayed.     

  2. Tap your teacher's computer name to connect.
    1. Most recently connected desktops appear in the Previous Desktops area at the top of the screenHistory area on the right of the screen. Swipe from right to lefttop to bottom to access additional desktops when applicable.     
    2. If you know the IP address of your teacher’s computer, you can tap Connect by IPIP Connect, type in the IP address, then tap NextConnect.

  3. On the Choose a Login Method screen, tap Join as known user if your Student ID or Device ID is recorded as part of the class roster. Otherwise, tap Join as guest, which allows you to participate as an Anonymous student.    

  4. Enter your Sticker ID, Student ID, or name in the box provided.

    The label on the box will change depending on whether Sticker ID or Student ID was selected as the Student Devices Join Configuration in Preferences, or if you selected Join as a guest user in step 3.

  5. To enter a session code, enter the four-digit session code into the Session Code box.
  6. Keep the Remember Me box checked if you will always use this ID and session code on this device.
  7. Tap Join.

    A welcome message is displayed.

  8. If the name is correct, tap OK to join the active class session. Otherwise, tap That’s Not Me to return to the Connect to Desktop screen.

    If your device displays a message stating that you have already joined the session, the Student ID or Sticker ID has been entered on another device during the same session. Check both devices to ensure the correct ID is entered on the correct device. The second device will not be allowed to connect.

    When the Insight 360 Student app has successfully connected to a computer desktop, the desktop and connection information appears in the Previous Desktop list at the top of the Connect to Desktop screen the next time the app is opened.
