Asking Verbal Questions

Verbal questions allow you to set up an impromptu question and receive feedback through student response devices. You can ask the following types of questions through the Verbal Questions tab: True/False, Yes/No, Numeric, Short Answer (up to 70 characters) and Multiple Choice/Multiple Select (up to six options).

  1. Tap .

    The Ask a Question screen is displayed.

  2. Tap .

    The Question Properties window opens.

  3. Tap the desired question type and answer option.

    If asking a Numeric or Short Answer question, enter the answer(s) in the appropriate fields.

  4. Tap OK.
  5. Tap the Question Type.

    The Select Question Type window opens.

  6. Tap the desired question and answer bullet (ABC or 123) types and tap Back.
  7. Tap Answer.
  8. Tap or enter the appropriate answer information based on the selected question type and tap Back.
  9. To set the timer, slide the timer bar to on. Tap or to set the timer with more or less time.   
  10. Ask your students the question and include all options (if multiple choice), or display the question using a projector connected to your computer. Instruct students to answer by pressing the corresponding key on their devices.

    You may want to write the question and options using Whiteboard mode.

  11. Tap Start.

    Tap to display the results on the classroom computer screen.

    Tap to display the timer on the classroom computer screen.

    Tap to display the response grid on the classroom computer screen.

  12. If all students answer the question before the timer is finished, tap Stop.      
  13. To start another question, tap New.
