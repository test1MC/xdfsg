Ending a Class Session

A class session does not end until you tap End Class. If the teacher device hibernates or is turned off, the class will resume when the device is turned on.

Insight 360 Cloud saves class session information while the session is active. When a class session is ended, the Insight 360 Cloud software makes the session information available for review and editing. This information may include session date and start time, screenshots and results.

You can retrieve session information from the My Results tab in Insight 360 Cloud.
