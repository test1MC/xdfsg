Starting a Class with the Insight 360 Teacher App

When a class session is started, the Insight 360 Teacher app communicates with the Insight 360 software to associate devices with the students in your class roster.

 

You must connect the Insight 360 Teacher app to start a class. For more information, see Connecting the Insight 360 Teacher App.

  1. On the Teach a Class screen, tap the appropriate class from the class list.
  2. Confirm the class name, section, period number and days are correct.
  3. Tap Start Class.
