Ending a Self-Paced Test

The Self-Paced Test command bar is visible on the classroom computer screen when a self-paced test (such as an Answer Key or ExamView test) has been started from Insight 360 Cloud.

  1. Click End Test on the Self-Paced Test command bar.

    The Test Ended window opens.

  2. Select one of the following options:
    • Return to Test - Closes the Test Ended window and returns to the Self-Paced Test command bar in its most recent configuration.
    • Close - Closes the Test Ended window and the Self-Paced Test command bar.
    • Close and View Results - Closes the Test Ended window and the Self-Paced Test command bar; displays the test results on the My Results tab in Insight 360 Cloud.
