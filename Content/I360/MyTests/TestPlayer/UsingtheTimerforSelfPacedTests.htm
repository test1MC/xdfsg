Using the Timer for Self-Paced Tests

The Test Player provides an optional timer that adds or subtracts minutes during the test. While the timer is running, students may change their answers using their student devices. Once the timer reaches zero, answers are no longer accepted.

Answers are not accepted if the timer is paused.

  1. To set the timer in Test Player:
    1. Open a test in Test Player. For information about opening a test in Test Player, refer to Starting a Self-Paced Test.
    2. Below Additional Options, select the box labeled Use Timer.
    3. Enter the number of minutes in the Minutes box or click the arrows to set the number.
  2. To use the timer during the test:
    1. Start a self-paced test in Test Player.
    2. On the Self-Paced Test command bar, click . The timer begins.
      • To add more time, click .
      • To subtract time, click .
      • To pause the timer, click .
      • To restart the timer, click .
    3. When the timer reaches zero, click End Test.
