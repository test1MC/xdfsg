Creating Multiple Choice Questions

A multiple choice question has one correct answer from a number of options.

  1. Create an Answer Key or Tagged question.
  2. From the Question Type drop-down menu, select the Multiple Choice type matching the number of answer options to present to the student.

    Multiple Choice A-C has three options: A, B, and C

    Multiple Choice A-F has six options: A, B, C, D, E, and F

  3. To present numeric answer options, select 123.
  4. Below Select one or more answers, click the letter or number representing the correct answer. If the question is a poll with no correct answer, click No Correct Answer.
