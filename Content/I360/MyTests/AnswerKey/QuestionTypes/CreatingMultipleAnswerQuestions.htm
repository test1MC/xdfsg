Creating Multiple Answer Questions

A multiple answer question has two or more correct answers among a number of options. Students must enter all of the correct answers (and none of the incorrect answers) for their response to be marked as “correct.”

  1. Create an Answer Key or Tagged question.
  2. From the Question Type drop-down menu, select the Multiple Choice type matching the number of answer options to present to the student.

    Multiple Choice A-C has three options: A, B, and C

    Multiple Choice A-F has six options: A, B, C, D, E and F

  3. To present numeric answer options, select 123.
  4. Below Select one or more answers, click the letters or numbers representing the correct answers.
