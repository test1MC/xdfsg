Question Types

Insight 360 Cloud allows you to create the following types of questions:

  • Multiple Choice Multiple Choice (one correct answer, up to six options)
  • Multiple Answer Multiple Answer (more than one correct answer, up to six options)
  • Numeric Numeric
  • Short Answer Short Answer
  • True/False True/False
  • Yes/No Yes/No

Responses can be scored as correct or incorrect, providing statistical feedback on individual and class progress.
