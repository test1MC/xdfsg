Attaching a Test File Document

Insight 360 Cloud does not store individual question or answer options in the Answer Key tests. However, you can attach a test file document containing the individual questions and answer options to the corresponding Answer Key test.

This feature is only available when your students are using the Insight 360 Student app as a response device.

With the Insight 360 Student app, the attached test file document and test question displays on the joined Insight 360 Student app while the test is in progress. If a document is not attached to the Answer Key test, the Insight 360 Student app does not display the test questions.
