Editing an Answer Key Test

An Answer Key test can only be edited if it has not been used.

  2. Select the Answer Key test to edit.
  3. To change the Answer Key Test name:
    1. Delete the text in the Test Name field.
    2. Enter a new name for the Answer Key test. The test name is changed immediately in the test list.
  4. To change the associated test file:
    1. Click to navigate to the test file.
    2. Click Open to attach the file to the test.
  5. To edit an Answer Key Test question:
    1. Select the question to edit.
    2. Change the Short Description, Question Type or Correct Answer as applicable.

    See Question Types for more information on setting up each type of question.

    1. (USA only) If applicable, change the Standard.
  6. To delete a question from an Answer Key Test:
    1. Select the question to delete.
    2. Click .

      A warning message is displayed.

    3. Verify that this is the question you want to delete and click Yes.
  7. To associate a new standard to the question, click Standard and select the applicable standard.
