Adding a Question to an Answer Key Test

Answer Key tests only contain answer text for the question, NOT the question text.

The following types of questions can be added to an Answer Key test:

  • Multiple Choice (one correct answer, up to six options)
  • Multiple Answer (more than one correct answer, up to six options)
  • Numeric
  • Short Answer
  • True/False
  • Yes/No

  3. In the Questions area, click .

  4. In the Short Description field, describe the question in 20 characters or less.

    First President

    T/F

    This description identifies the question in various Insight 360 Cloud reports and the Scores area.

  5. Below Question Type, select the type of question from the drop-down.

    The question options change depending on the question type selected.

  6. Select the options for the question type.
