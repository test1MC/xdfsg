  4. Name the Answer Key Test and click Save.

    The default save location is:

    PC - C:\Program Files (x86)\eInstruction\ExamView\Tests

    Mac - Applications > eInstruction > ExamView > Test

 

The Answer Key Test is exported as an XML file. You can now save this file to a flash drive and import the XML version of the Answer Key Test on another computer that has Insight 360 Cloud installed. For more information, see Importing an Answer Key Test.
