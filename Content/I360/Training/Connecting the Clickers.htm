Connecting the Clickers

The Insight 360 software is compatible with the Pulse 360/Spark 360 and ResponseCard NXT clickers, as well as the QT Device.

  1. Start a class session.
  2. Plug in the clicker receiver.
  3. If you are using the Pulse 360/Spark 360 clickers, press the power button to turn on the clicker.
  4. If prompted, enter the Join Code (JC).

    If you are using the ResponseCard NXT or the QT Device, the Join Code (JC) is the same as the Channel Number.
