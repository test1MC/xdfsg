Preferences

The Preferences category allows you to determine the Student Devices Join Configuration and the Student Device Settings.

Student Devices Join Configuration

Assign students to student devices using Sticker ID - Select this option if each student has been assigned a Sticker ID in the class roster. Use the Starting Sticker ID menu to determine the starting ID number.

Student will input ID into their student devices - Select this option to require each student to input his or her unique Insight 360 Cloud ID prior to joining the session.

Student Device Settings

Select the box labeled Send correct/incorrect feedback to student devices at the end of each teacher led question to send feedback to student devices after the polling question has ended.

Select the box labeled Prompt teacher to confirm student reconnect if you want to confirm dropped connections.
