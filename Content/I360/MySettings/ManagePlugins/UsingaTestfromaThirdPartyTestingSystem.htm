Using a Test from a Third-party Testing System

When you administer an imported test, the results automatically upload to the third-party testing system once the test ends. If you choose to give an ExamView or Answer Key test, the results must be manually uploaded to the third-party testing system.

  1. Select the My Tests tab.
  2. Select a class from the class list.
  3. From the Select Test Type drop-down menu, select the third-party testing system's test.
  4. Below Select Test,select the criteria needed from the drop-down menus.
  5. Below Additional Options, you can:
    • Enter the Session Name
    • Select the box labeled Use Timer to set a timer
    • Send individual scores to student devices at the end of the test. Select the box labeled Send scores to student device at the end of test.
  6. Click Start Test.

    Preview test will not function for imported tests.

    An imported test can be linked to a test file (containing the questions) by using Attach Content File.
