Importing a Class from a Third-Party Testing System

  1. Select the My Classes tab.
  2. Click Import from Plugin.
  3. Click Start Import next to the plugin that you want to use.

    If your plugin is not listed, go back to Getting Started.

    You may need to select additional information to complete the import.

  4. Click Import.

    A progress status bar is displayed and a confirmation message appears when there is a successful import.

  5. Click OK.
