Viewing and Editing an Imported Class

  1. To view your classes, select the My Classes tab and click Classes.
    • All of your classes are listed, including any that have been created within Insight 360 Cloud.
    • An icon of the third-party testing system appears next to each imported class.
    • If you click a class, the students’ names appear.
  2. To edit a class, select the class and click Edit Class.

    Fields that contain gray text cannot be edited.
