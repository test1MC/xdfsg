Device Management

The Device Management category lists information for every device that has connected to your computer through Insight 360 Cloud.

  1. From the Select Device Type drop-down menu, select the device type.
  2. The following information is displayed:
    • Device Type
    • Connection
    • Firmware
    • Join Code
    • Last Used
  3. Optionally, select the box labeled Only show devices that have updates available to be notified of new device firmware.
