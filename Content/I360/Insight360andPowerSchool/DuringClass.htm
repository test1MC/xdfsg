Exporting Results from Insight 360 Cloud

  1. Open Insight 360 Cloud and select the My Results tab.

  2. Click Reports in the upper left corner.

     

  3. Expand Session and select Student Summary.
  4. Select a class from the Select a class drop-down menu and a session from the Select a session drop-down menu.

  5. Select the box labeled Show Advanced Export Options.
  6. Select the box labeled Export Percentages as Whole Numbers.
  7. Configure the assignment based on one of the following:
    • Percentage export - configure columns 1 and 2 as:

    • Points export - configure columns 1 and 2 as:

    The Student ID and score choice must be exported to meet PowerTeacher requirements. Optionally, additional columns can be exported. If more than one numeric column is chosen, the score column must be identified during the import process.

  8. Click Export to CSV and save the CSV file to a specified location.
