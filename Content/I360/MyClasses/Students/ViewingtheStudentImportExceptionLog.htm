Viewing the Student Import Exception Log

If there are errors in the import process, the student or students containing errors (also called exceptions) will not be imported. Instead, the affected student or students are placed in the Student Import Exception Log. You can view and edit students in the Exception Log.

  1. Select the My Classes tab and click Import from CSV.
  2. Click View Exceptions.

    If the student import did not generate any errors, you will not see the View Exceptions option.
