Deleting a Class

Use the Delete Class feature to remove unwanted classes from the Insight 360 Cloud database.

Once the class is deleted from the Insight 360 Cloud database, all of the session data associated with the class including test scores, screenshots and reports will also be deleted. Students enrolled in the class will not be deleted from the database, but student results relating to the deleted class will be deleted.
