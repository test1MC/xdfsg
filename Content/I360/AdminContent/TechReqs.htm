Technical Requirements

Software Requirements

Windows

  • Microsoft Windows XP SP3, Vista, 7, 8
  • Intel or AMD 600 MHz class processor (1 GHz or higher recommended)
  • 2 GB RAM
  • 4 GB hard disk space for installation
  • 1024 x 768 resolution at 32-bit color or higher
  • Adobe Reader version 9.0 or equivalent PDF reader
  • DVD Drive with dual layer support for installation
  • Standard USB 2.0 port (2)
  • Active internet connection
  • Device Manager 6.71 or higher for users running other Turning Technologies software

Mac

  • Mac OS X 10.6.8, 10.7, 10.8 or 10.9
  • Intel processor
  • 2 GB RAM
  • 4 GB hard disk space for installation
  • 1024 x 768 resolution at 32-bit color or higher
  • Adobe Reader version 9.0 or equivalent PDF reader
  • DVD Drive with dual layer support for installation
  • Standard USB 2.0 port (2)
  • Active internet connection
  • Device Manager 6.71 or higher for users running other Turning Technologies software

Insight 360 Student app for iOS

  • iPad with iOS 6 and higher

Insight 360 Cloud Teacher app

  • iPad with iOS 6 and higher
  • Local high-speed Wi-Fi for connection of both host desktop computer and iPad
  • Network must support use of Internet Group Management Protocol (IGMP)

Insight 360 Student app for Android

  • Android OS 4.0 and higher
  • Local high-speed Wi-Fi for connection of both host desktop computer and iPad
  • Network must support use of Internet Group Management Protocol (IGMP)
