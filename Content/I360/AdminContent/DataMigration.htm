Data Migration

User data should not be shared among computers, but can be transferred when necessary. Insight 360 Cloud files are cross-platform compatible between PC and Mac.

Insight 360 Cloud Data

This directory contains files that house:

  • class rosters set up in Insight 360 Cloud
  • favorites for Insight 360 Cloud and Mobi 360
  • answer key tests
  • assessment results
  • screenshots
  • user preferences

Before moving, these two files will need to be removed from the data folder as they contain information unique to the system: filters.dat and Mac.dat. Once deleted, the data folder can be copied in its entirety to the new location.

The data folders can be found here:

  • Windows Vista, Windows 7, Windows 8: C:\Users\<<username>>\AppData\Local\Insight360\Data\
  • Windows XP: C:\Documents and Settings\<<username>>\Local Settings\Application Data\Insight360\Data\
  • Mac OS X: Users > [username] > Library > Application Support > Insight360 > Data

When you have completed the transfer, the user will need to re-register with the original SRC code to obtain an activation code for the new system within 30 days.
