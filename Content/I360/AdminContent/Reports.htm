Modifying Exported Reports

Insight 360 Cloud allows you to modify the format of an exported report. This can be done on a single computer, or the IT Department can define the format and send the file format to multiple computers.

Single Computer Export

  1. Open Insight 360 Cloud and click the My Results tab.
  2. Select a report.
  3. To modify the format of the Export to CSV report, select Show Advanced Export Options.
  4. You can now select which columns to include or exclude. You will see one drop-down box for each column that will appear in the report. The first drop-down will appear in the first column of the report. The second drop-down will appear in the second column, and so on. Select where you would like each column to appear, or select None if you do not want that column to appear in the report.
  5. The settings are saved until you go back into the Show Advanced Export Options and modify the format.

Multiple Computer Export

Similar to defining the export for a single computer, an IT Department could define an export format and push that format out to multiple instructor computers.

  1. Follow steps 1 through 4 above.
  2. You have now created this file: CsvExportMap.xml. It is located in the DATA_PATH directory. If you have installed Insight 360 Cloud in its default location, the file would be located at AppData\Local\Insight360\Data
  3. This file can now be pushed out to the instructors’ computers and placed in the same location as above.
  4. When the instructor clicks Export to CSV, the data will be exported in the format you have defined.
  5. If the instructor goes into Show Advanced Export Options and modifies the export format, the data will no longer export in the format that you defined.
