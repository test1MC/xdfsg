Uninstalling

When you uninstall the software, the user preferences and data will not be removed so that they are available if the software is reinstalled.

Windows

  1. Click Start and select Control Panel.
  2. For Windows 7 and 8, select Programs and Features. For Windows XP, select Add or Remove Programs.
  3. Select Insight 360 Cloud from the list of programs and click Uninstall.
  4. If prompted, select Yes to confirm the uninstall.
  5. After the uninstall is complete, select ExamView Learning Series and click Uninstall.
  6. After the ExamView Learning Series uninstall is complete, select ExamView Assessment Suite and click Uninstall.
  7. If prompted, select Yes to confirm the uninstall.
  8. Restart the computer.

Mac OS X

  1. Open the Applications folder.
  2. Open the eInstruction folder.
  3. Move the Insight 360 Cloud folder to the Trash.
  4. Go to hard drive root and move the ExamView Pro folder to the Trash.
  5. Empty the Trash.

Remove User Preferences and Data

To remove user preferences, log files and user data, delete the following directories and contents. If needed, back up the data before deleting as this contains the user preferences and data, such as class rosters and grades.

Windows Vista, Windows 7, Windows 8

  • C:\Users\<<username>>\AppData\Local\Insight360

Windows XP

  • C:\Documents and Settings\<<username>>\Local Settings\Application Data\Insight360

Mac OS X

  • Users > [username] > Library > Application Support > Insight360
  • Users > [username] > Library > Logs > Insight360.log
  • Users > [username] > Library > Preferences > com.eInstruction.Insight-360-Director.plist

Older log files are named Insight360.logYYYYMMDD. For example, an archived log from January 1st, 2012 would be named Insight360.log20120101.
