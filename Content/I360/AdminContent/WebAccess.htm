Web Access

Insight 360 Cloud Web Access allows students to participate in an Insight 360 Cloud session using nearly any laptop, smartphone, or other device with an internet browser.

If the student device goes to sleep, they will be removed from the class and will need to rejoin. For more information, please visit: https://support.einstruction.com/entries/21571904-Mobile-Devices-using-Web-Access-will-be-dropped-from-the-class-if-device-goes-into-sleep-mode

Minimum Browser Requirements

  • Internet Explorer 9 or higher
  • Firefox 14 or higher
  • Chrome 21 or higher
  • Safari 5.1 or higher
  • Opera 12 or higher
  • BlackBerry 7.0 or higher
  • iOS Safari, all versions
  • Android Chrome, all versions
  • Windows Phone 7.5 or higher
  • Android, all versions

How It Works

  1. The instructor opens the Insight 360 Cloud software on their computer and enables Web Access in My Settings > Mobile Devices.
  2. The instructor starts a class using a Mobi 360 or the Insight 360 Teacher app.
  3. The student opens a web browser (e.g., Internet Explorer, Firefox, Chrome) and goes to Insight360.einstruction.com. The Insight Online tab displays Join Class at the top.
  4. The student enters the Student ID and Session Code into the corresponding boxes.

    Only the Session Code is required. The teacher can display the Session Code by tapping Class/Session Code on the Mobi 360 or Insight 360 Teacher app.

  5. The student clicks Join.

    If a Student ID was entered in Step 4, a welcome message appears with the student's name. If the name is correct, click Proceed to Class. Otherwise, click That’s Not Me and repeat steps 4-5 with the correct information.

    If a Student ID was not entered and the student wants to join anonymously, click Yes, I’m Sure. This allows the student to join the class but their answers will not be credited to their name or Student ID.
