Install by Silent (or Network) Installation

Follow the steps below to perform a silent or network installation. For additional information, please refer to your specific product documentation.

Windows

  1. Insert the Insight 360 Cloud DVD.
  2. Open a command prompt.
  3. Type the letter of your DVD drive followed by a colon and press Enter (e.g., “d:”).
  4. Type cd \Installer to change to the directory that contains the installer executable.
  5. Run the following command: Insight_360_Installer.exe /s /v”/qn /norestart”
  6. To run the silent uninstall, follow steps 1-3 above and use the following command: Insight_360_Installer.exe /x /s /v”/qn /norestart”

Mac

  1. Insert the Insight 360 Cloud DVD and open Terminal.
  2. Type cd /volumes/"Insight 360 Cloud" and press Enter.
  3. Run the following command: sudo installer –pkg “Insight_360.pkg” –target /
  4. If prompted, type in your sudo password.
