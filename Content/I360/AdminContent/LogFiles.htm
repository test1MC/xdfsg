Log Files

Log files are created in default directories. These log files contain information that may be useful for troubleshooting when contacting Technical Support.

  • The main log file is named Insight360.log.
  • Older log files are named Insight360.logYYYYMMDD.

    An archived log from January 1st, 2012 would be named Insight360.log20120101.

  • Log files can be opened with any text editor, such as Notepad. Each record is time stamped with a date and time, warning level and the log message. The latest information should always be at the bottom of the log file.

Default Log Paths

Windows Vista, Windows 7, Windows 8

  • C:\Users\<<username>>\AppData\Local\Insight360\Insight360.log

Windows XP

  • C:\Documents and Settings\<<username>>\Local Settings\Application Data\Insight360\Insight360.log

Mac OS X

  • Users > [username] > Library > Logs > Insight360.log

Multiple Users

If multiple users log into the same machine with one kit, it will store the individual user data local to their log in. In addition, since no data is stored on the Insight 360 Teacher app or Mobi 360, the devices can be used among the users and they will reflect the favorites and configurations of the logged in user.
