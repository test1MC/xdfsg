Licensing

Insight 360 Cloud is available as a single-user license or multiple-user license. If two individuals are using Insight 360 Cloud at the same time, they must have two licenses.

A single-user license can be installed on two computers.

You can install Insight 360 Cloud on two computers (such as a desktop and a laptop), but the software cannot be used at the same time.

You can install Insight 360 Cloud on a single computer under different user accounts, but the software cannot be used at the same time.

Additional license for evaluation and training

The license also allows for an additional copy of Insight 360 Cloud to be installed in a testing lab for evaluation, or for training purposes, as long as these copies are not used for instructional purposes with students.
