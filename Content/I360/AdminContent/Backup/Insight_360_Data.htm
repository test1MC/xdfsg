Insight 360 Cloud Data

This directory contains files that house:

  • class rosters set up in Insight 360 Cloud
  • favorites for Insight 360 Cloud and Mobi 360
  • answer key tests
  • assessment results
  • screenshots
  • user preferences

Windows Vista, Windows 7, Windows 8

  • C:\Users\<<username>>\AppData\Local\Insight360\

Windows XP

  • C:\Documents and Settings\<<username>>\Local Settings\Application Data\Insight360\

Mac

  • Users > [username] > Library > Application Support > Insight360
