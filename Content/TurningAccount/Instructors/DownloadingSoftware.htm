Downloading Software

 

You must have Administrative privileges on your computer to download the Install version of the software.

Prior to downloading, verify with your institution which version of the software is supported.

  1. Sign in to your Turning Account.
  2. Select Downloads from the left menu.
  3. Select the appropriate program.

    Please note TurningPoint Cloud requires participant or instructor licensing for individual reporting and exporting.. Due to the added security encryption available in TurningPoint Cloud these data files cannot be opened in TurningPoint 5.

  4. Save the file to a specified location.
  5. Open the file and follow the on-screen instructions.
