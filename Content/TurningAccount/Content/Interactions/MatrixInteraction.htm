Matrix Interaction

The matrix assignment interaction offers a two-dimensional table for assignment between multiple options.

  2. Enter a Title in the area provided and select Matrix Interaction.

  3. Click OK.

    The Edit question screen appears.

  5. Select the type of matrix interaction.
    • Single Choice - The participant may only select one option in the row.
    • Multiple Choice - The participant may choose more than one option in the row.
  7. Enter the source elements in the areas provided.

    Source and destination elements may include tables, images, audio or formulas.

  8. To add another source element, click the below a source element box.
  9. Enter the destination elements in the areas provided.
  10. To add another destination element, click the to the right of a destination element box.
  11. To remove an element, click the icon in the upper right corner of the element box.
  12. To set a correct answer for a single choice matrix interaction, select the radio button that corresponds to the correct source and destination elements. To set a correct answer for a multiple choice matrix interaction, select the check boxes that correspond to the correct source and destination elements.
