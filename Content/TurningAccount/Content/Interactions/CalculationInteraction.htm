Calculation Interaction

The calculation interaction is an extended text entry question, designed especially for numerical input with parameters.

  2. Enter a Title in the area provided and select Calculation Interaction.

  3. Click OK.

    The Edit question screen appears.

  4. Enter the statement in the area provided.
  5. To add a gap, place the cursor at the desired location and click Add gap. To change a word in your statement into a choice, highlight the word and click Transform text in gap.
  6. In the Solution area, enter the value or click Calculated value to select a variable that defines the solution calculation.
  7. Enter the score for the question. The default point value is 1.
  8. Select the precision value.
    • Exact - The participant must enter the exact numeric value.
    • Absolute - You must enter the upper and lower bounds values.
    • Relative - You must enter the percentage values for the upper and lower bounds.
  9. Enter the number of characters for the gap size.
  10. Enter the placeholder text.

    The placeholder text is displayed until the participant enters their own text.

  11. To remove a gap, place your cursor in front of the [...] and press the delete key on your keyboard.
  12. To add question feedback, select the Feedback tab.
    • To add feedback for a correct answer, enter the text in the Question has been answered correctly feedback area.
    • To add feedback for an incorrect answer, enter the text in the Answer was wrong feedback area.
    • To add feedback for specific responses, click Add new feedback at the bottom of the page.
      1. Set the conditions. Select the operator ( =, >, >=, <, <=) and set the score value.
      2. To add another condition, click +. To remove the condition, click -.

        The same feedback is displayed for multiple conditions. To add separate feedback text for each condition you must add a new feedback.

      3. Enter the feedback text for the condition in the area provided.
      4. To add feedback for another condition, click Add a new feedback and repeat the steps above.
      5. To remove individual feedback, click Delete to right of the feedback area.
