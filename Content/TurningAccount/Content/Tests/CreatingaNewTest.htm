Creating a New Test

  1. From the Welcome screen, click Create Test. Alternatively click New Test from Content.

    The Create a new test window opens.

  2. Enter a Title for the test in the area provided.
  3. To add Meta data, click the Meta data tab.
    • The General tab allows you to change the Title, add a Description and Keywords. You may also specify the Language. Click Show details to view Identifier information.
    • The Lifecycle tab contains information about the Author, Created date and Modified date.
    • The Rights tab allows you set access rights for users you add to your test. For more information, see Managing Access Rights for Tests.
  4. Click OK at the bottom of any of the tabs to continue setting up the test.

    The new test is displayed on your screen. There are three columns:

    • Easy access to question bank - From this column, you can add questions from your question bank to your test. For more information, see Adding Questions From a Question Bank to Your Test
    • Test structure - The Test structure column allows you to add sections and questions to the test. You can also delete sections and questions. The More option allows you to cut, copy, paste, delete, move up or move down questions and sections. For more information, see Setting Up the Test Structure.

      Performing any of the More options with a Section selected will perform the action on the entire Section.

    • Edit test - The title of this column and the editing items change based on the item selected in the Test structure column.
  5. Optionally, click Preview to view your test. For more information, see Previewing a Test.
  6. Click Save test to save your changes.
