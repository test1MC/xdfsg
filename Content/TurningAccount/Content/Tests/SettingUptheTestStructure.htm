Setting Up the Test Structure

  1. To add a section:
    1. In the Test structure column, select the location for the new section. You can add a section at the test level, and you can add a section within a section.
    2. Click the Add Section icon.
    3. Enter a name for the section in the Title box.
    4. Enter the Stimulus in the area provided.
  2. To add a single question.
    1. Select the section you would like to add the question to.
    2. Click the Add Question icon.
    3. Select the Question type and click OK.
    4. Enter the information on the Question & Answer tab. For more detailed information on creating the different question types, see Interactions (Question Types).
  3. To delete a question or section, select the item and click the Delete icon.
