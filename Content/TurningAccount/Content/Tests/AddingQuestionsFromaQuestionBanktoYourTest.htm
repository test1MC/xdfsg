Adding Questions From a Question Bank to Your Test

 

You must have a test open for editing.

  1. In the Test structure column, select the section you want to add the questions to.
  2. Select a question bank from the Choose a question bank drop-down in the Easy access to question bank column.
  3. Select the desired questions.

    Hover the mouse over a question to view the question metadata.

  4. Click Add an element.

    The questions are copied into the selected section.
