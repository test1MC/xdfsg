Managing Access Rights for Tests

  1. Search for the desired user by entering the user name in the area provided and click Start search.
  2. Select the user from the Search results.
  3. Select the rights to assign the user.
    • Read - The user will have permission to view the test.
    • Export - The user will have permission to export the test. A user assigned export rights is automatically assigned read rights.
    • Edit - The user will have full permission to edit the test. A user assigned edit rights is automatically assigned read and export rights.
    • Delete - The user will have permission to delete the test. A user assigned delete rights is automatically assigned read rights.
  4. Click Assign access rights.
  5. To remove a user's rights, select the user from the list below Assigned access rights and click Delete.
  6. Click OK.
