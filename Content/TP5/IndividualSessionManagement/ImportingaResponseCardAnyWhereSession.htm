Importing a ResponseCard AnyWhere Session

ResponseCard AnyWhere session files may be imported into TurningPoint versions 5.2 and higher.

  1. Connect ResponseCard AnyWhere to the computer via a mini USB cord.
  2. From the Manage tab, click the Session drop-down menu and select ResponseCard AnyWhere.

    The following prompt is displayed.

  3. Click Yes.
  4. The session is displayed under Auto in the left panel.

 

Now that the ResponseCard AnyWhere session has been uploaded, it can be edited. For more information on editing a session, see Editing a Session.
