RemotePoll

This section contains the preferences for enabling RemotePoll.

Enable RemotePoll Support

Check this option to enable RemotePoll Support via TurningPoint.

Current IP Address

Displays the presenting computer’s IP Address to establish communication between the RemotePoll software and TurningPoint.

RemotePoll Port

Enter the presenting computer’s network port number. Default is 1025.

For more information about RemotePoll, please visit www.turningtechnologies.com/user-guides.
