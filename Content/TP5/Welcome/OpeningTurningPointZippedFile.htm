Opening TurningPoint (Zipped File)

The zipped file must be extracted before TurningPoint can be opened.

  1. Double-click the downloaded ZIP file.
  2. Select the desired folder location and click Unzip.
  3. Open the unzipped folder and double-click TurningPoint.exe.

    Create a shortcut for TurningPoint.exe on the desktop for quicker access to the application.

    DO NOT remove any files from the Application folder. The application will not function properly if files are removed.
