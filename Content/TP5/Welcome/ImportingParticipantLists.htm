Importing Participant Lists

Participant lists created with previous Turning Technologies products can be imported into TurningPoint. There is no need to recreate existing participant lists.

  2. Click the Participant List drop-down menu and select Import.
  3. Click Choose File... and browse to the location of the participant list to be imported.
  4. Select the participant list and click Open.

    The participant list is now displayed in the left panel.

  5. Repeat steps 2 - 4 to import additional participant lists.
