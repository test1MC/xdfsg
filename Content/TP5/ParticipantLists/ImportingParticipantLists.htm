Importing Participant Lists

Exported participant lists can be imported into TurningPoint on another computer. Exported participant information from an integration can also be imported as a participant list.

Participant lists used in previous Turning Technologies products can be imported into TurningPoint.

  2. Click the Participant List drop-down menu and select Import.

    The Import Participant List window opens.

  3. Select one of the following options:
    • TurningPoint Participant List
    • File from Integration - Select the scheme from the drop-down menu.
  4. Click Choose File.
  5. Select the file and click Open.

    The participant list appears in the left panel.

 

To edit participant list information, see Modifying Participant Lists .
