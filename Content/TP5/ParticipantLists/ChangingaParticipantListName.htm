Changing a Participant List Name

A participant list name may be changed at any time. Participant list names are displayed under the Polling tab so that they are easily identifiable for polling sessions.

  2. Select a participant list and click the pencil icon at the top left of the Participant List Overview screen.

    The Edit Participant List window opens.

  3. Enter a name for the participant list in the box provided and click Save.

    The new participant list name is displayed in the left panel.
