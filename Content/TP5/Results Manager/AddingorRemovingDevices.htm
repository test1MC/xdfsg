Adding or Removing Devices

Device IDs can be added to a participant from Results Manager.

  2. Select a participant row.
  3. Expand the Devices category from the right panel.
  4. To add a device, select the Device ID and click the ( + ).
  5. To remove a device, select the Device ID and click ( - ).

    Select one of the following options:

    • Yes - The device is removed from the participant as well as all grade entries assigned with that device.
    • Remove Device Only - The device will not be registered for future sessions, but current scores associated with the device are kept.
