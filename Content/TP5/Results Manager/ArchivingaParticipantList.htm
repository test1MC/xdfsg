Archiving a Participant List

An archived participant list can be saved in any location, such as a shared network drive or flash drive.

  1. From the Manage tab, select the appropriate participant list, click the Participant List drop-down menu and select Create Archive.

  2. Browse to the desired save location and click Save.

    The participant list saves as a TPRX file. The following message is displayed.

  3. Click Keep to leave the participant list visible under the Manage tab or click Delete to remove the participant list.

    Keeping the participant list and session files will allow them to remain active. This allows the archive to function as a file backup for later restoration if necessary.
