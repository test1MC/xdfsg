Creating a Participant List (Optional)

A participant list enables you to track individual results for grading and assessment purposes.

  1. From the TurningPoint Dashboard, select the Manage tab.
  2. Click the Participant List drop-down menu and select New.

  3. Name the participant list and click Create List.

  4. Enter the Device IDs found on the back of the ResponseCards or the ResponseWare IDs.

    Also, enter first and last names and any other relevant information in the appropriate cells.

    Press Enter for additional rows, or Tab to move between fields.

    The User ID field is optional. A User ID typically refers to a Student ID.

  5. Click Save and Close to save the participant list.
