Question List: Priority Ranking

A Priority Ranking Question weighs the participants’ choices and then ranks them accordingly. By default, the first answer choice is weighted with 10 points, second answer choice with nine points, third answer choice with eight answer points and so on.

  1. From the Content tab, click the Content drop-down menu, mouse over New and select Question List.
  2. Name the question list and click Save.

  3. Select Priority Ranking from the Question Type drop-down menu.
  4. Select answer choices from the Answer Choices drop-down menu.
  5. Select the number of responses permitted.
  6. Double-click the Enter question text box.
  7. Enter the question text and answer choices.
  8. Optionally, expand Question Options and adjust the response weights.
  9. Click Close.
  10. Click Save and Close.
