Downloading the Software

  1. Visit www.turningtechnologies.com/downloads to download TurningPoint.

    Choose the install version or the no install version of the software (Mac is no install only).

  2. Opening the no install version of the software:
    • PC - Double-click the downloaded zip file. Select the desired folder location and click Unzip. Open the unzipped folder and double-click TurningPoint.exe.
    • Mac - Double-click the DMG file. Drag and drop the icon into the Applications folder.
