Getting Started

Downloading the Software

  1. Visit www.turningtechnologies.com/downloads to download TurningPoint.

    Choose the install version or the no install version of the software.

  2. Opening the no install version of the software:
    1. Double-click the downloaded zip file.
    2. Select the desired folder location and click Unzip.
    3. Open the unzipped folder and double-click TurningPoint.exe.
    4. Drag and drop the file into the Applications folder.
    5. Create a shortcut by dragging the icon from the Applications folder to the dock.
    6. Click the TurningPoint icon.
