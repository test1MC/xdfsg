Short Answer Question

A short answer slide contains a question text box, a chart and a list of the five most common responses.

Creating a Short Answer Slide and Setting Correct Answers

  1. Click the New icon from the TurningPoint ribbon/toolbar and select Short Answer.
  2. Type the question.
  3. Click outside of the answer box.

    These are the required steps to build a basic slide. Setting correct answers and adding objects to the slides are optional.

  4. Click to expand Scoring Options from the Slide Preferences Pane.
  5. Select the answer value from the Answer Value drop-down menu.

  6. Enter the keyword or keywords in the box provided.

    Polling Options

    • Show Correct Keyword - After polling closes, the correct keyword(s) is displayed on the slide.

    Scoring Options

    • Correct Keyword(s) - Enter keyword(s) in the box provided. Multiple keywords should be separated with a semicolon.
    • Enforce Capitalization - If a response does not have proper capitalization the response is considered incorrect.
