Numeric Response Question

A Numeric Response slide contains a question text box, a chart and a list of the five most common responses.

Creating a Numeric Response Slide and Setting Correct Answers

  1. From the TurningPoint ribbon/toolbar, click the New icon and select Numeric Response.
  2. Type the question.
  3. Click outside of the answer box.

    These are the required steps to build a basic slide. Setting correct answers and adding objects to the slides are optional.

  4. Click to expand Scoring Options from the Slide Preferences Pane.
  5. Select the answer value from the Answer Value drop-down menu.

  6. Enter the value in the Acceptable Value field and if desired, enter the minimum and maximum range in the provided fields.

    Polling Options

    • Show Correct Answer - After polling closes, the correct answer(s) is displayed on the slide.

    Scoring Options

    • Answer Value - The answer value can be set by selecting “Correct” or “No Value” from the drop-down box.
    • Acceptable Value - Enter a specific value in the box provided.
    • Acceptable Range - If a specific value is not required for a correct answer, a range can be set. Enter a minimum value and a maximum value in the boxes provided.

    A numeric response can have up to 16 characters in all three polling environments.
