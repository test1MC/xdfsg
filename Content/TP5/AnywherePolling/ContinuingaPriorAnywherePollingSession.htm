Continuing a Prior Anywhere Polling Session

Anywhere Polling can resume a session from a saved session file. New session information is appended to the open session file. When saved, the polling results are saved in one session file.

  1. Click Options.
  2. Mouse over Session and select Continue Prior Session.
  3. Select the session file to be continued and click Open.

    The previously saved session file is now open.

 

Now that the session has been continued, it can be polled. For more information, see Running a Session.
