TurningPoint for Mac 5.2 and Higher Tips

Below are tips for using TurningPoint for Mac 5.2 and higher.

Installing the PowerPoint Library - There are two scenarios for installing the library:

  • You are a user with administrative privileges.
    1. Click the Install PowerPoint Library icon on the PowerPoint Polling environment.
    2. When prompted, enter your credentials to proceed. This action is only required once.
  • You are a user without administrative privileges. The PowerPoint library needs to be installed by an Administrator. Please refer to the PowerPoint Library Install guide.

Presenter View vs. Mirror Show - Presenter View allows the presenter to view an alternate version of the presentation on his/her monitor while the audience sees the full screen presentation. Mirror Show displays the full screen presentation on both the presenter’s monitor and the projector screen. NOTE Presenter View is the default view for PowerPoint 2011.

To use Mirror Show, you must manually select it via the PowerPoint toolbar:

  1. On the PowerPoint 2011 toolbar, select the SlideShow tab.
  2. Click Mirror Show prior to starting the Slide Show.

    Alternatively, you may choose to mirror your entire display through System Preferences. Follow the path: System Preferences > Displays > Mirror Display.

Toggle Compact Mode - When using a projector in the PowerPoint Polling environment, the TurningPoint toolbar may exceed the width of the screen.

  • To view the full toolbar while plugged into a projector, click View from the TurningPoint menu bar and select Toggle Compact Mode. Be sure TurningPoint is selected as the focus item. If it is not the focus item, click the TurningPoint icon on the Dock.

Refresh - When creating a presentation in TurningPoint for Mac, a slide needs refreshed anytime a change is made to the slide’s content.

If you create a slide with four answers choices (A, B, C, D), and later add a fifth answer choice (E), you must click the Refresh button to immediately see that “E” has been added as an answer choice.

Alternatively, you may click the Reset button and select Session. This option resets and refreshes ALL slides in the presentation.

Additionally, when editing a slide, or a slide’s preferences, verify that the Preferences pane reflects the appropriate slide number. If it does not, select the desired slide and click once on the Preferences pane. The Preferences pane heading now coincides with the selected slide.

Tabbing - When creating a question list, if any non-text field items are skipped over while tabbing, verify that All Controls is selected in the System Preferences. Follow the path: System Preferences > Keyboard > Keyboard Shortcuts. Selecting All Controls allows tabbing to non-text fields.

Floating Windows - If this option is checked in the Anywhere Preferences, normal windows such as the presentation window, or ResponseWare login window, no longer take focus away from other applications. Use this setting when polling in the Anywhere environment over a full screen application such as PowerPoint or Keynote.

 

 

Basic PowerPoint for Mac Keyboard Shortcuts

Below are some common shortcuts that may be helpful to you.

To Press
Cut text Cmd+X
Copy text Cmd+C
Paste text Cmd+V
Duplicate Cmd+D
Increase font Cmd+Shift+>
Decrease font Cmd+Shift+<
Bold Cmd+B
Underline Cmd+U
Italicize Cmd+I
Undo Cmd+Z
Create new presentation Cmd+N
Insert new slide Cmd+Shift+N OR Control+M
Zoom in Cmd++ (plus sign)
Zoom out Cmd+- (minus sign)
Open presentation Cmd+O
Close presentation Cmd+W
Print Cmd+P
Save Cmd+S
Show or hide pointer A OR Equal sign
Close active window Cmd+W
