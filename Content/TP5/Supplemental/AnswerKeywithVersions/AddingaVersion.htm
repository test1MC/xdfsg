Adding a Version

Question lists can be generated into different versions for Self-Paced Polling. A master list of the question list is generated for quick editing of the question and answer choices.

Versions containing more than 250 questions cannot be used for Self- Paced Polling.

 

A question list must be open. If a question list is not open, select the Content tab, select a question list and click Edit Question List on the Question List Overview screen.

  1. Click Version and select Add.

    The Add Version window opens.

  2. Enter a New Version Code. A version code must be a numeric value.

    When the participants take the Self-Paced Polling test they will be prompted for a version code. The version code corresponds to the question list version they received.

  3. Select one of the following options:
    • Add with New Questions - A new version of the question list is created and the new questions are added to the master list.
      1. Enter the number of questions in the box provided.
      2. Select the default question type from the drop-down menu.
    • Add with Existing Questions - A new version of the question list is created with the question from the master list.
      1. Select the questions to be included in the new version.
      2. Optionally, check the box to randomize questions.
      3. Optionally, check the box to randomize answers.
  4. Click Add Version.

    The Version Warning dialog box is displayed.

  5. Click OK on the PC or Add Version on the Mac to create the new version.
  6. Optionally, repeat steps 1 - 5 to create additional versions.
  7. To edit a specific version, select the version from the drop-down menu in the upper right of the window.
  8. Click Save and Close to save the changes and return to the Question List Overview screen.
