Exposing the Auxiliary Session Save Setting for Mac

Prior to editing the TurningPoint 5 settings, verify that version 5.3 of the TurningPoint 5 software is installed and that you are logged in as the Administrator.

 

TurningPoint 5 must be closed.

  1. Open Terminal.
  2. Copy and paste ONE of the following commands:

    All Users:

    sudo defaults write /Library/Preferences/com.turningtechnologies.turningpoint TTSettingsKeyExposeAuxiliarySavePreference -boolean YES

    Current Users:

    defaults write com.turningtechnologies.turningpoint TTSettingsKeyExposeAuxiliarySavePreference -boolean YES

  3. Enter your password if prompted and press Return.

    The secondary save location now appears in the Preferences.

  4. Open TurningPoint 5 and click the Preferences icon in the lower right corner of the Dashboard.
  5. Select Software from left menu and locate Auxiliary Session Save Location.
  6. Click ... to select a save location.
  7. Click Close.
