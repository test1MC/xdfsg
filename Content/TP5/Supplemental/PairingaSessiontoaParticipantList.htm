Pairing a Session to a Participant List

This document provides steps to link a session ran in Auto to a participant list.

How to pair a session to an existing participant list...

  1. Open TurningPoint and select the Manage tab.
  2. Expand Auto from the left panel.

  3. Select the appropriate session and drag and drop it into the appropriate participant list.

    The session is now linked to the participant list.

How to pair a session to a new participant list...

  1. Open TurningPoint and select the Manage tab.
  2. Expand Auto from the left panel.
  3. Right-click/Control-click the appropriate session and select Move to New Participant List.

    The Create Participant List window is displayed.

  4. Enter a name for the new participant list and click Create List.

    A new participant list is created.

  5. Click Edit Participants to add participants to the new participant list.
