Using Data Slicing

After receiving response data, TurningPoint can cross-reference the responses of one question to another set of responses.

  1. While on the question that contains the data to be sliced, click the Data Slice icon from the showbar.

    The Data Slicing Setup window opens.

  2. To slice by Question Data:
    1. Select the Question Data tab.
    2. Expand the question that contains the information by which to slice the data.

    3. Select the answer choice and click Select.
    4. The chart now displays the results of the participants who responded to the answer choice from step c.

      At the top center of the screen there is a data slice drop-down menu that allows for the selection of a different answer choice. There is also an option to display the results of participants who answered the question selected in step b correctly or incorrectly.

  3. To slice by Participant Groups:
    1. Select the Participant Groups tab.
    2. Expand the group that contains the information by which to slide the data.

    3. Select the desired group and click Select.
    4. The chart now displays the results of the current question for the participants in the selected group.

      At the top center of the screen there is the Data Slice drop-down menu that allows for the selection of a different group.

  4. Advance the slide to continue the presentation.

    Click the View the Original Chart icon to reset the chart.
