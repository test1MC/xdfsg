Continuing a Prior PowerPoint Polling Session

TurningPoint can resume a presentation from a saved session file. New session information is appended to the open session file. When saved, the polling results are saved in one session file.

  1. Click the Continue Prior Session icon from the TurningPoint ribbon/toolbar.
  2. Select the session file to be continued and click Open.
  3. TurningPoint opens the previously saved session file.

    If a new presentation is opened while continuing a session, click the Refresh button prior to making any edits to the new presentation.

 

Now that the previous session is open, it can be continued. For more information on running a session, see Running a Slide Show.
