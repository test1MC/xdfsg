Inserting a Response Counter

During the Slide Show, the response counter displays the number of responses. For example, if there are 32 participants in the participant list, the response counter displays “0 of 32.” It continues to count as the responses are received and closes polling when the 32nd response is received.

  1. Select the polling slide.
  2. From the TurningPoint ribbon/toolbar click the Objects icon and select Response Counter.

    A placeholder is inserted for the response counter.

  3. To remove the response counter, simply delete the object from the slide.

    The response counter can be formatted like regular PowerPoint objects. The prompt text can be adjusted in the preferences.
