Editing a Version

Questions can be rearranged or removed from a specific version.

 

A master list must be open. If a master list is not open, select the Content tab, select the master list and click Edit Question List on the Question List Overview screen.

  1. Select the version to be edited from the Version drop-down menu in the upper right corner of the question list editor.
  2. From the Question List Editor toolbar, click Version and select Edit.

    The Edit Version window is displayed.

    • The left panel (Available Questions) displays questions from the master list that are not being used in the current version.
    • The right panel (Current Version) displays the questions being used in the current version.
  3. To remove a question from the current version:
    1. Select a question from the Current Version panel.
    2. Click the bottom arrow between the panels.
  4. To add a question from the master list:
    1. Select a question from the Available Questions panel.
    2. Click the upper arrow between the panels.
  5. To rearrange the order of the questions in the current version:
    1. Select the question to be moved from the Current Version panel.
    2. Click the up arrow or down arrow to the right of the panel to move the question to the desired place in the list.
  6. Click Save EditsChanges to close the window and save the changes.
  7. Optionally, repeat steps 1 - 6 to edit to additional versions.
