Matching

Matching questions may have up to 10 answer choices. To set up a matching question, a list of matches is paired with a list of choices. Participants submit the choices in the order they correspond to the matches.

Matching questions are NOT supported in the PowerPoint Polling environment.

  3. Select the number of matches from the Matches drop-down menu.
  4. Select the number of choices from the Choices drop-down menu.
  5. Enter the correct answers in the box provided.

    Each correct answer choice needs to be separated by a semicolon. For example, “a; c; d”.

    Correct answers may also be assigned by selecting the correct answer choices that corresponds to the matched item.
