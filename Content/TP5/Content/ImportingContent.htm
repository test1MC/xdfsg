Importing Content

The following file types can be imported into TurningPoint: question list files (TPQX and TPQ), answer key files from TurningKey (TKY), QuestionPoint files (TQZ) and PowerPoint presentations (PPTX and PPT). When a file is imported, it is copied from its current location to the Content folder in TurningPoint.

Drag and drop files and folders directly under the Content tab.

  2. Click the Content drop-down menu and select Import.
  3. Select the file or files and click Open.

    Select more than one file by holding down the Shift or ControlCommand keys.

 

To place content into a folder, select the content and drag it to the desired folder. For more information on folders, see Organizing Content Into Folders.
