Downloading Turning Technologies Software

How do I know which type of software I will need to download?

Mac users will need the Mac No Install Version of the Turning Technologies software.

PC users have two choices, the Install Version and the No Install Version. You must have administrator privileges on the computer in order to run the install version of the software. The no install version of the software is downloaded as a zipped file. The software can be run from a flash drive, network drive or from your desktop.

How do I download the software?

  1. Go to www.turningtechnologies.com.
  2. Click the downloads tab at the top of the screen.

  3. Click the title of the software to expand the download choices.

  4. Click the link for the desired software. You will be taken to the Software Downloads Registration page.
  5. Enter the required information in the fields provided and click Submit.
  6. Click the download link to start downloading the software.

  7. Click Accept if you accept the End User License Agreement. You will not be able to download the software if you do not accept.

    If you do not see the Accept button, you may need to set your Windows Display Settings to 100% or adjust the zoom setting on your internet browser.

  8. When prompted, save the file to your computer.
  9. When the file has finished downloading you will need to install or open the program based on the type of file you chose to download.
