Downloading TurningPoint Cloud

The TurningPoint Cloud software is accessible through the Turning Account website.

Please verify that your computer meets the Technical Requirements on page 1 before downloading TurningPoint Cloud.

  1. Go to https://account.turningtechnologies.com and sign in to your Turning Account.

    Alternatively, the Turning Account website can be accessed from a link via www.turningtechnologies.com/downloads.

  2. Sign in to your Turning Account.
  3. Select Downloads from the left menu.
  4. Below TurningPoint Cloud, select the PCMac Install/No Install version.
  5. Save the file to a specified location.
