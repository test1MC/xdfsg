Configuring Automatic Updates

Automatic updates provides users the option to update the software without first uninstalling it.

An internet connection must be available to check for software updates.

  2. Click the TurningPoint Cloud logo located at the bottom of the Dashboard.

    The About TurningPoint Cloud window is displayed.

    The About TurningPoint Cloud window can also be accessed from Help on the TurningPoint Cloud PowerPoint Polling ribbontoolbar.

  3. Click Check For Updates.

    If an update is available, TurningPoint Cloud will download the update.

  4. Optionally, check the box to automatically check for updates.

    TurningPoint Cloud will check for updates each time the application is opened. You may experience a slight lag between the application and the Turning Technologies server.

    TurningPoint Cloud will check for updates once every 24 hours. You may experience a slight lag between the application and the Turning Technologies server.

  5. To turn off automatic updates, click in the lower right corner of the Dashboard. Select Software from the left menu and uncheck Automatically Check for Updates.
