Importing Session Files

Session files created with other Turning Technologies products may be imported via the import function on the Manage tab. TurningPoint Cloud will automatically convert them to the new format.

  3. Click the Session drop-down menu and select Import.
  4. Browse to the location of the session.
  5. Select the session or sessions and click Open.
  6. Select one of the following options:
    1. Convert and Replace - This option converts the session file to the new format and deletes the original session file. The new session file is saved in the TurningPoint Cloud Sessions folder.
    2. Convert and Keep - This option adds the session to the sessions list, but the file remains in its original location. The file must remain in this location to view reports and edit the session. If the file is moved, it must be relocated in the session overview screen.

      The import summary displays the status of each session or sessions selected for import.

  7. Click OK.

    The session or sessions are displayed in the left panel.
