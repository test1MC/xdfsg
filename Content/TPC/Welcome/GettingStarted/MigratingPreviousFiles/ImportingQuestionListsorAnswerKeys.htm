Importing Question Lists or Answer Keys

Question lists and answer keys created in TurningPoint 5, TurningPoint Anywhere and TurningKey can be imported into TurningPoint Cloud. There is no need to recreate existing content.

  3. Click the Question List drop-down menu and select Import.

    Answer key files (TKY) are considered question list files in this case.

  4. Browse to the location of the file(s) to be imported.
  5. Select the question list and click Open.

    A dialog box appears. Click Yes to convert the selected file to the new format.

 

For more information about updating question list preferences, see Updating Question Preferences within the Question List Editor on page 1.
