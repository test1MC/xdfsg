Importing Participant Lists

Participant lists created with previous Turning Technologies products can be imported into TurningPoint Cloud.

Participant lists from earlier versions of Turning Technologies products do not contain Turning Account licenses. Sessions associated with these participant lists cannot be viewed with participant based reports.

  3. Click the Participant List drop-down menu and select Import.
  4. Click Choose File and browse to the location of the participant list to be imported.
  5. Select the participant list and click Open.

    The participant list is now displayed in the left panel.

  6. Repeat steps 2 - 4 to import additional participant lists.
