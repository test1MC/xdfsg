Converting Graph Charts to Excel Charts

TurningPoint versions prior to 6.2 used MS Graph charts. TurningPoint versions 6.2 and higher use Microsoft Excel charts on interactive polling slides.TurningPoint Cloud can convert Graph charts to Excel charts. Converting the charts is optional, as Graph charts will continue to work, however newly inserted slides will contain Excel charts by default.

  1. Open TurningPoint Cloud and sign in to your Turning Account.

  2. Click PowerPoint Polling.

    PowerPoint opens with the TurningPoint Cloud ribbon.

  3. Open a TurningPoint presentation created in a version prior to 6.2.

    A warning message appears.

  4. Click Yes to convert the Graph charts to Excel charts.

    Some Graph charts may not be compatible for the conversion and may require manual conversion. For more information, see Inserting a Chart/Changing the Chart Type on page 1.

  5. Save the presentation.

    Save your PowerPoint presentation in the TurningPoint Cloud Content folder. The presentation is listed in the Content section of the Polling tab for quick access.
