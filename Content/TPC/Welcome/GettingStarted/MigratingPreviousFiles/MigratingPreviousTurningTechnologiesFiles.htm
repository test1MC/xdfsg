Migrating Previous Turning Technologies Files

Questions lists, participant lists and session files created with previous Turning Technologies products can be imported into TurningPoint Cloud.

It is recommended to make copies of session files, participant lists, question lists, answer keys and presentations before migrating them to TurningPoint Cloud. Once the files have been converted to the new format, they cannot be used with previous versions of Turning Technologies products.

This section covers the following tasks:

  • Importing Participant Lists
  • Importing Session Files
  • Importing Question Lists or Answer Keys
  • Converting TurningPoint 2008 Presentations
