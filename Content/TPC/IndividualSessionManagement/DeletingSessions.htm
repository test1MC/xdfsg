Deleting Sessions

A session can be deleted from the computer or the sessions list in the Manage tab.

  2. From the Manage tab, select a session from the left panel.
  3. Click the Session drop-down menu and select Delete.

    The Delete Session window opens.

  4. Select one of the following options:
    • Remove Session from TurningPoint Cloud- Remove the session from the list, but leave the associated file in its current location on the computer.
    • Delete Session- Remove the session from the list and move the file to the Recycle BinTrash.
  5. Click OK.

    A session deleted from its permanent location, but not from the TurningPoint Cloud Manage tab, will still appear in the sessions list. However, generating reports or session editing will no longer be available for that session.
