Running a Session With a Question List

One or more question lists can be used in an Anywhere Polling session. If a question list is not selected from the Dashboard, you may load one at any time the polling is closed.

For more information on creating a question list, see Creating a New Question List on page 1.

  1. Plug in the receiver or establish a ResponseWare session.

    For information on how to establish a ResponseWare session, see Enabling ResponseWare on page 1.

  3. Select a question list below Content.

    If you do not select a question list at this time, one can be loaded later from the showbar. Clicking the Options menu, mouse over Question List and select Load Question List.

  4. Click Anywhere Polling.

    The TurningPoint Cloud Dashboard closes and Anywhere Polling opens. For more information on the Anywhere Polling showbar, see About the Anywhere Polling Showbar on page 1.

    The question list is visible on the Anywhere Polling Showbar.

  5. Click .

    Participants may now respond to the question.

  6. Click after all the responses have been received.

    The chart window displays if Show Results is enabled for the question. For more information about the chart window, see About the Chart Window on page 1.

  7. Repeat steps 5 and 6 for the remaining questions.
  8. To change a question list:
    1. Click , mouse over Question List and select Change Question List.

    2. Select the question list and click Open.

      Polling must be closed to load or change a question list.

    The question list is visible on the Anywhere Polling Showbar.

  9. Repeat steps 5 and 6 for the remaining questions.
  10. Repeat step 8 to select an additional question list.
  11. When finished, close the Anywhere Polling application.

    A message appears stating that the current session contains unsaved response data.

  12. Click Save.
  13. Name the session and click Save.

    The Anywhere Polling application closes and the TurningPoint Cloud Dashboard opens.

 

To edit session data or manage results, see Individual Session Management on page 1 or Results Manager on page 1.
