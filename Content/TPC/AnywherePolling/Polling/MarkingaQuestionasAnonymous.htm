Marking a Question as Anonymous

If using a participant list, all responses are associated with a participant as they respond during a session. Follow the steps below if anonymous responses are desired. Questions created in the question list with the Anonymous option checked do not need to be selected again.

Once enabled, this feature cannot be undone for the current question poll. If enabled by mistake, close polling and reopen on the same question.

  1. Click .
  2. Select Mark Question as Anonymous.

    Repeat the steps above any time anonymous polling is required.
