Setting Up Self-Paced Polling

The following instructions explain how to open the Self-Paced Polling environment, how to select the content and participant list, and how to set up the test so participants may log in.

  1. Plug in the receiver, open TurningPoint Cloud and sign in to your Turning Account.
  2. Select a participant list and a question list.
  3. Click Self-Paced Polling.

    The TurningPoint Cloud Dashboard closes and the polling application opens.

  4. Click .
    1. Select the login option from the Participant Login drop-down menu.
    2. Select the test time from the Time Limit drop-down menu.

    Select either duration or end time.

    1. Select the message to be sent to participants from the Completed Test Message drop-down menu.

    The default login options can be set in the preferences. For more information, see Self-Paced on page 1.

  5. Click OK.

    At this time the participants may log into the test.

 

Now that participants have logged into the test, it is time to begin Self-Paced Polling on page 1.
