Self-Paced Polling

If the instructions in Setting Up Self-Paced Polling on page 1 have already been completed, skip to step 6.

  1. Plug in the receiver, open TurningPoint Cloud and sign in to your Turning Account.
  2. Select a participant list and a question list.
  3. Click Self-Paced Polling.

    The TurningPoint Cloud Dashboard closes and the polling application opens.

  4. Click .

    If Show Login Window is unchecked in the preferences, skip to step 6.

    1. Select the login option from the Participant Login drop-down menu.
    2. Select the test time from the Time Limit drop-down menu.
    3. Select the message to be sent to participants from the Completed Test Message drop-down menu.
  5. Click OK.

    At this time participants may log into the test.

  6. Click .

    Participants may now begin the test. Once they have finished the test, participants can submit their results from the response device.

  7. Click once all the results have been received.
  8. Click to save the session results.
  9. Name the session and click Save.

    Session files should be stored in the default save location. If the session is saved in another location it will then need to be imported into TurningPoint Cloud.

  10. Close the application to return to the TurningPoint Cloud Dashboard.

 

The session file can now be used to manage results or generate reports. For more information, see Individual Session Management on page 1 or Results Manager on page 1.
