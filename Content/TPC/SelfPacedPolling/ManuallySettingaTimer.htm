Manually Setting a Timer

Test time limits may be set manually or changed if the time limit was incorrectly set.

  1. From the menu bar, click ToolsWindow and select Timer.

    The Timer Options window opens.

  2. Select one of the following options:
    • Duration- Enter the hours and/or minutes for the length of the test.
    • End Time- Set a specific end time for the test.
    • No Timer- The test will have no timer and polling must be closed manually.
  3. Click Set.
