Email Template

Below you will find an email template that can be used to notify your students about the use of ResponseWare mobile polling in your classroom this semester. You will need to edit the following area, if not already edited by your Account Executive: School Code.

Send this email one to two (1 - 2) weeks prior to the start of classes.

 

Hello Students,

We will be using Turning Technologies ResponseWare mobile polling in our class this semester. You will need to do the following prior to the first day of class:

Download the free app from the following locations:

  • iPhone, iPad, iPod Touch: www.itunes.com
  • Android: www.googleplay.com
  • All other web-enabled devices can gain access to the software by visiting responseware.turningtechnologies.com

Purchase a Turning Account license at one of the following locations:

  • Campus Bookstore (if applicable)
  • store.turningtechnologies.com

NOTE: The Turning Account license includes a free ResponseWare license.

The School Code for store.turningtechnologies.com is: _____________

Create a Turning Account and enroll in the course at account.turningtechnologies.com.

Log in to your LMS (Blackboard, Canvas, D2L, Moodle, Sakai) and follow the links for Turning Technologies to create a Turning Account, purchase the license (if you have not already done so), and enroll in the course.

If applicable, update an existing ResponseWare account by changing your password. The new website for web-enabled connection is now responseware.turningtechnologies.com.

Please purchase the Turning Account license and create a Turning Account by the first Friday of class.

I hope you have found this information helpful. If you have any questions, please let me know.
