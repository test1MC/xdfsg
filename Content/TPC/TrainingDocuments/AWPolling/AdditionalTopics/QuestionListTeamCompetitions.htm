Question List: Team Competitions

 

  1. From the Content tab, click the Content drop-down menu, mouse over New and select Question List.
  2. Name the question list and click Save.

  3. Select Demographic Assignment from the Question Type drop-down menu.
  4. Select answer choices from the Answer Choices drop-down menu.
  5. Double-click the Enter question text box.
  6. Enter the question text and answer choices.
  7. Expand Demographic Options and check Use in Competition.

    Verify that correct answers are set within the question lists so that teams can accumulate points.

  8. Click Close.
  9. Click Save and Close.
