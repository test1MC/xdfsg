Manually Assigning Participants to Teams

Creating a participant list allows you to assign participants to specific teams. Assigning participants to teams via a participant list allows you to create more than 10 teams.

  2. Click Participant List and select New.
  3. Name the participant list, select Create Manually and click Create List.
  4. Once the Participant List Editor is displayed, select a cell and enter the participant information.

    If there is a CSV or TXT file that contains participant information, drag and drop the file into this window. It is also possible to copy and paste text from an Excel file directly into the Participant List Editor.

  5. Press Enter to add another participant.
  6. Click the Demographic icon and select Add.

    The Add Demographic window is displayed.

  7. In the Demographic Grouping box, enter a group
    (i.e. Teams, Competition).
  8. In the Group Options box, enter the team names
    (i.e. One, Two, East, West).
  9. After all team names are entered, check the box
    labeled Use for Competitions and click OK.

    The new column now allows you to assign each
    participant to a team by selecting a team from the drop-down menu next to each participant.

  10. Click Save and Close to save the participant list and return to the Dashboard.
