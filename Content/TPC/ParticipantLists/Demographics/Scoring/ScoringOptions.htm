Scoring Options

There are multiple scoring options on the Slide Preferences Pane that can factor into the participant/team score.

  • A recognized point value is 1 - 10000.
  • A recognized point value is 1 - 10000.
  • Speed Scoring - Point values will decrease relative to the time polling is kept open when this option is selected.

    Countdown timers work well on slides where speed scoring is utilized.

  • Show Advanced Scoring - When this option is selected, both the point value and answer value are displayed. The point value can be changed by entering a new value in the box provided.
