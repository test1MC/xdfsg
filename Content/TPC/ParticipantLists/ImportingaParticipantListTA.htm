Importing a Participant List

A participant list created on the Turning Account Registration website can easily be imported into the TurningPoint Cloud software.

  1. Open TurningPoint Cloud and sign in to your Turning Account.
  2. Click the Manage tab.
  3. Click the Participant List drop-down menu and select New.

    The Create Participant List window is displayed.

  4. Select the Download from Integration radio button and click Create List.

    The Connect to Integration window is displayed.

  5. Select Turning Account Registration from the Integration drop-down menu.

    The Server Address, Username and Password are automatically populated.

  6. Click Connect.

    The Import Participant List window is displayed.

  7. Select the participant list(s) to import.
  8. Click Import.

    The participant list(s) is imported and placed in the left panel.

    The imported participant list contains the following information: Device ID(s), License Status, First Name, Last Name, User ID and Email.
