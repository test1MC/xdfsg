Participant Lists

A participant list is a list of participant names, User IDs, Device IDs and other custom categories. The use of a participant list allows for detailed reporting on each participant from a session. In TurningPoint Cloud, participant lists that are imported from either the Turning Account Registration website or from an LMS include each participant's license status. Participants must have an active license or their scores will not be recorded in any TurningPoint Cloud participant reports. Additionally, unlicensed participant scores will not export to an LMS.

TurningPoint Cloud sessions are grouped based on the participant list with which they are associated. For more information on participant list-based session management, see Results Manager on page 1.
