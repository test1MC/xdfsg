Hiding the Auxiliary Session Save Setting for PC

The secondary save location may be hidden so that users cannot choose their own save location.

 

TurningPoint Cloud must be closed.

  1. Open the TurningPoint Cloud folder.
  2. Double-click the Application folder.
  3. Locate and open the TurningPointexe.settings file. It is recommended to open the file with Notepad.
  4. In Notepad, click Edit and select Find.
  5. Search for secondary.
  6. Press Enter three times until the command
    <DisplaySecondarySaveLocation datatype="Boolean">True</DisplaySecondarySaveLocation> is displayed.
  7. In the above command, change True to False.
  8. Click File and select Save.

    The setting is now hidden, but will still apply.
