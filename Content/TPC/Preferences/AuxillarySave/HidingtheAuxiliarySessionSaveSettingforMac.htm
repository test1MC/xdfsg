Hiding the Auxiliary Session Save Setting for Mac

The secondary save location may be hidden so that users cannot choose their own save location.

 

Turning Point Cloud must be closed.

  1. Open Terminal.
  2. Copy and paste ONE of the following commands:

    All Users:

    sudo defaults write /Library/Preferences/com.turningtechnologies.TurningPoint-Cloud TTSettingsKeyExposeAuxiliarySavePreference -boolean NO

    Current Users:

    defaults write com.turningtechnologies.TurningPoint-Cloud TTSettingsKeyExposeAuxiliarySavePreference -boolean NO

  3. Enter your password if prompted and press Return.

    The setting is now hidden.
