Exposing the Auxiliary Session Save Setting for PC

Prior to editing the TurningPoint Cloud settings, verify that version 6.0 or higher of the TurningPoint Cloud software is installed and that you are logged in as the Administrator.

 

TurningPoint Cloud must be opened once and then closed prior to performing the steps below. If the software is not opened and then closed, the TurningPointexe.settings file will not be visible.

  1. Open the TurningPoint Cloud folder.
  2. Double-click the Application folder.
  3. Locate and open the TurningPointexe.settings file. It is recommended to open the file with Notepad.
  4. In Notepad, click Edit and select Find.
  5. Search for secondary.
  6. Press Enter three times until the command
    <DisplaySecondarySaveLocation datatype="Boolean">False</DisplaySecondarySaveLocation> is displayed.
  7. In the above command, change False to True.
  8. Click File and select Save.

    The secondary save location now appears in Preferences.

  9. Open TurningPoint Cloud and click the Preferences icon in the lower right corner of the Dashboard.
  10. Select Software from left menu and locate Auxiliary Session Save Location.
  11. Click ... to select a save location.
  12. Click Close.
