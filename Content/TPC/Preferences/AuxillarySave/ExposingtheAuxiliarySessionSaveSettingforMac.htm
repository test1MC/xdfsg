Exposing the Auxiliary Session Save Setting for Mac

Prior to editing the TurningPoint Cloud settings, verify that version 6.0 or higher of the TurningPoint Cloud software is installed and that you are logged in as the Administrator.

 

TurningPoint Cloud must be closed.

  1. Open Terminal.
  2. Copy and paste ONE of the following commands:

    All Users:

    sudo defaults write /Library/Preferences/com.turningtechnologies.TurningPoint-Cloud TTSettingsKeyExposeAuxiliarySavePreference -boolean YES

    Current Users:

    defaults write com.turningtechnologies.TurningPoint-Cloud TTSettingsKeyExposeAuxiliarySavePreference -boolean YES

  3. Enter your password if prompted and press Return.

    The secondary save location now appears in the Preferences.

  4. Open TurningPoint Cloud and click the Preferences icon in the lower right corner of the Dashboard.
  5. Select Software from left menu and locate Auxiliary Session Save Location.
  6. Click ... to select a save location.
  7. Click Close.
