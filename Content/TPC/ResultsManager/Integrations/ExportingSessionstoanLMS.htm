Exporting Sessions to an LMS

LMS integrations allow for the sharing of data between TurningPoint Cloud and the LMS. Sessions can be exported to the LMS grade book directly from TurningPoint Cloud.

 

An LMS Integration must be configured on an LMS server. Confirm with your IT department that the integration is properly setup before attempting to use the integration in TurningPoint Cloud.

    The Connect to Integration window opens.

  4. Select the Integration from the Integration drop-down menu.
  5. Enter the Server Address, Username and Password in the appropriate fields.
  6. If applicable, select an Institution from the Select an Institution drop-down menu.
  7. Click Connect.

    Optionally, click Remember this information to have the Integration, Server Address and User Name stored for future use.

    The Update with Integration window opens.

  8. Select Export Session(s).
  9. Check the session or sessions to be exported.
    • Check Select All to export all sessions associated with the participant list.
    • Check Active Participants Only to include only participants who have an active device in the session or have grade overrides within a column.

      Unlicensed participants will not export to the LMS.

  10. Click Export.

    A prompt will appear displaying the number of columns being exported to the LMS.

  11. Click Export.
  12. Click OK on the confirmation box.

 

Login to the LMS to verify the sessions correctly uploaded.
