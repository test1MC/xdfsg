Exporting a Report

Reports can be exported as a CSV, Excel or HTML file.

Java must be installed to export a report as a CSV or Excel file.

  1. While in the Reports window, click and select one of the following options when available:
    • CSV- CSV files are plain text files that can be opened with various programs.
    • Excel- Saves the file as a Microsoft Excel spreadsheet with native Excel charts for manual manipulation.
    • HTML- HTML files can be opened with an Internet browser.
  2. Name the file and click Save.
