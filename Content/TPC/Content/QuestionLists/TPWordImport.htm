Importing a Word Document into TurningPoint

Formatting a Word Document for Import

TurningPoint allows you to import a Word document into your presentation, creating interactive slides from each individual question. If you have additional support questions, please contact Turning Technologies Technical Support at 1.866.746.3015 or visit www.turningtechnologies.com/technical-support.

Formatting the Word Document...

  1. Open the document in Microsoft Word.
  2. Remove all question numbers from the questions and bullets from the answer choices.
  3. Format all questions as Heading 1.
  4. Format all answers as Heading 2.

    Turn on the Show/Hide option in Word to detect any unnecessary spacing. Delete all extra spaces to ensure a successful import.

  5. Save and close the document.

Importing the Word Document into TurningPoint...

  1. Open TurningPoint and select the Content tab.
  2. Click the Content drop-down menu, mouse over New and select Question List.
  3. Name and save the question list.
  4. Click the Question icon and select Import.
  5. Browse to the saved Word document and click Open.
  6. Select the questions to import.

    The questions appear in the Question List Editor.

  7. Click Save and Close.
