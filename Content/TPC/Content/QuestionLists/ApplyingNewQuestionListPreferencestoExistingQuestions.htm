Applying Question List Preferences to an Existing Question File

Question list preferences can be applied to an external file that contains questions.

 

A saved question list from a legacy version of TurningPoint or a question list made with a supported file type is needed.

  3. Click the Content drop-down menu, mouse over New and select Question List.
  4. Name the Question List.
  5. Expand the Preferences tab and adjust the preferences as needed.
  6. Click Save.

    The Question List Editor opens.

  7. Click and select Import.
  8. Select the desired file and click Open.
  9. Check the questions to be imported.
  10. Click Import.
