Renaming a Version

 

A question list with versions must be open. If a question list is not open, select the Content tab, select the master list and click Edit Question List on the Question List Overview screen.

  1. From the Question List Editor toolbar, click and select Rename.

    The Rename Versions window is displayed. All of the versions associated with the question list are listed in the window.

  2. Double-click the version name to be edited.
  3. Enter the new version name and click OK.
