Inserting a Countdown Timer

A countdown timer provides participants with a visual cue that polling is about to close as it counts down to zero. When the countdown expires, the countdown timer disappears and polling is automatically closed. A countdown timer can be made to appear on each polling slide that is created. For more information on how to automatically include a countdown timer on each polling slide, see PowerPoint on page 1.

  1. Select more than one slide by holding down the CtrlCmd or Shift key on the keyboard.

  2. From the TurningPoint Cloud ribbontoolbar click Objects, mouse over Countdown and select the desired countdown timer.

    There are five countdown timers available.

    Square

    Letter

    Clock

    Gemstone

    Adjustable

    The Adjustable countdown timer inserts a placeholder on the slide. During the slideshow it displays as above.

  3. To change the current countdown on the slide, select a different countdown from the Objects menu.
  4. To remove the countdown, simply delete the object from the slide.
  5. To adjust the time or add a sound to the countdown timer, click the countdown timer.

    Adjust the time from the Slide Preferences Pane.

    1. To adjust the time, enter a value into the Countdown Seconds box. A valid entry is 1 - 999.
    2. To add a sound, click ..., browse to the location of the sound file and click Open.

    Countdown timers can be formatted like regular PowerPoint objects. Right-click on the object and select a formatting option from the menu.

    1. To change the countdown time, insert a new countdown timer. Changing the time on the existing object will not alter the time value.

    To adjust the time on all countdown timers, click on the TurningPoint Cloud ribbontoolbar. Select PowerPoint from the left menu, check Countdown timer, adjust the time and click Apply All.
