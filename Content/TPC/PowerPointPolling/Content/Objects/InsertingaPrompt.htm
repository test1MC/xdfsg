Inserting a Prompt

A prompt provides participants with a visual cue that the slide is a polling slide to which they can respond. A prompt can be made to appear on each polling slide that is created. For more information on how to automatically include a prompt on each polling slide, see PowerPoint on page 1.

  1. Select more than one slide by holding down the CtrlCmd or Shift key on the keyboard.

  2. From the TurningPoint Cloud ribbontoolbar click Objects, mouse over Prompt and select the desired prompt.

    There are four prompts available.

    Bevel

    Explosion

    Speech

    Marquis

    Prompts can be formatted like regular PowerPoint objects. The prompt text can be adjusted in the PowerPoint on page 1.

  3. To change the current prompt on the slide, select a different prompt from the Objects menu.
  4. To remove the prompt, simply delete the object from the slide.
