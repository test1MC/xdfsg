Inserting a Response Counter

During the Slide Show, the response counter displays the number of responses. For example, if there are 32 participants in the participant list, the response counter displays “0 of 32.” It continues to count as the responses are received and closes polling when the 32nd response is received.

If there is no participant list, the response counter does not close polling on the first slide but rather builds a list of response devices. The list of devices then becomes the participant list and the expected number of response devices for the remainder of the session.

  1. Select more than one slide by holding down the CtrlCmd or Shift key on the keyboard.

  2. From the TurningPoint Cloud ribbontoolbar click Objects and select Response Counter.

    A placeholder is inserted for the response counter.

     

    The response counter can be formatted like regular PowerPoint objects.

  3. To remove the response counter, simply delete the object from the slide.
