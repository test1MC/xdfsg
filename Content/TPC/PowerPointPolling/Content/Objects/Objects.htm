Objects

Objects may be added to each slide in several ways. This section covers adding objects to a slide through the TurningPoint Cloud ribbontoolbar.

  • Objects such as charts, countdown timers, response grids and prompts may be set as default objects that are automatically inserted on each slide. To set default objects, see PowerPoint on page 1 for more information.
  • Objects may also be set during question list creation. For more information, see Content on page 1.
  • Objects can be added to a slide through the TurningPoint Cloud ribbontoolbar.
