Inserting Stats

Stats objects reveal statistics about responses after polling is closed. The value of the statistic is displayed in the stats object during a presentation after polling has been closed.

  1. Select more than one slide by holding down the CtrlCmd or Shift key on the keyboard.

  2. From the TurningPoint Cloud ribbontoolbar click Objects, mouse over Stats and select the desired stat.

    There are four stats available.

    Multiple stats may be placed on a single slide.

    Stats can be formatted like regular PowerPoint objects.

  3. To change the current stat on the slide, select a different stat from the Objects menu.
  4. To remove a stat, simply delete the object from the slide.
