Saving Custom Chart Templates

Excel charts can be edited and saved to PowerPoint to create new chart templates to insert as TurningPoint polling slides.

  1. From the TurningPoint Cloud ribbon, click New and select Multiple Choice.

    The template applies to the chart type, not the slide type. To change the default chart type when inserting a TurningPoint slide, see PowerPoint on page 1 in the TurningPoint Preferences.

  2. To change the chart type, click Objects, mouse over Charts and select the chart type.

    If you create a horizontal or offset template that does not use the same basic chart type as the default, the chart will not align with the answer choices.

  3. On the slide, click Enter Answer Text and enter 10 answer choices.

    Creating a template with 10 answer choices is recommended. Templates created with lower answer choice count questions do not apply the settings to higher answer choice count questions using the template.

  4. Click outside of the answer text box to update the chart.
  5. Click the chart.
  6. Edit the chart with the PowerPoint Chart Tools.

    Colors in the custom template default to that of the template's color scheme.

    The Automatically Format Chart feature is not compatible with custom templates.

  7. When you have finished editing your chart template, click the Design tab from Chart Tools.
  8. Click Save As Template.

    The Save Chart Template window appears.

  9. Enter the file name corresponding to the chart type and click Save.
    Chart TypeFile Name
    VerticalTP_Vertical
    HorizontalTP_Horizontal
    3D PieTP_3DPie
    Distributed PieTP_DistributedPie
    OffsetTP_Offset
    DoughnutTP_Doughnut

    The file is saved as a CRTX file.

    The chart template is applied to any newly inserted charts of the same chart type.
