Converting a PowerPoint Slide into a Multiple Choice Polling Slide

An ordinary PowerPoint slide can be converted into a multiple choice polling slide by adding a chart to a slide that is formatted as Title and Text or Title and Content. There can be up to 10 items in the list of answers. The question/title and the list of answers/numbered list must be in two separate boxes in order for the conversion to work properly.

  1. Select the slide or slides in the presentation that you want to convert into a polling slide or slides.
  2. From the TurningPoint Cloud ribbontoolbar click Objects, mouse over Charts or Animated Chartsand select the desired chart type.

    The answer list is re-numbered according to the TurningPoint Cloud preferences. The bullet format can be modified in the preferences. For more information, see Questions on page 1.

  3. Repeat step 2 to convert additional PowerPoint slides.
  4. When finished, save the presentation by selecting File > Save/Save As.

 

Enhance the presentation by adding Objects on page 1 or Competition on page 1 slides to help increase participation. Also, try using the Tools on page 1 available on the TurningPoint Cloud ribbon.
