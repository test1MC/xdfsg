Continuing a Prior PowerPoint Polling Session

TurningPoint Cloud can resume a presentation from a saved session file. New session information is appended to the open session file. When saved, the polling results are saved in one session file.

  1. Click Continue from the TurningPoint Cloud ribbontoolbar.
  2. Select the session file to be continued and click Open.

    TurningPoint Cloud opens the previously saved session file.

    If a new presentation is opened while continuing a session, click the Refresh icon prior to making any edits to the new presentation.

 

Now that the previous session is open, it can be continued. For more information on running a session, see Running a Slide Show on page 1.
