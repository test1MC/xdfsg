Opening the PowerPoint Polling Environment

The PowerPoint Polling environment is embeddedintegrated directly into PowerPoint.

You may be prompted to install the PowerPoint library. If you are a user with administrative privileges, click the Install PowerPoint Library icon on the PowerPoint Polling environment. When prompted, enter your credentials. This action is only required once. If you are a user without administrative privileges, the PowerPoint library must be install by an IT Administrator. Contact Turning Technologies Customer Support Team to receive a document on how to install the library.

  1. Open TurningPoint Cloud.

    Optionally, select a participant list, question list or PowerPoint presentation from the left panel.

    If using a question list, TurningPoint Cloud will automatically parse the content into PowerPoint slides. This may take a few moments if there are a large number of questions in the question list. For more information on participant lists, see Participant Lists on page 1. For more information on question lists, see Question Lists on page 1.

  2. Click PowerPoint Polling.

    PowerPoint opens with TurningPoint Cloud.

    If using a display setting of 1024 x 768, the TurningPoint Cloud toolbar may exceed the width of the screen. To view the full toolbar, ensure that TurningPoint Cloud is selected, click View from the menu bar and select Toggle Compact Mode.

 

Polling slides can be created directly in PowerPoint. There are also tools available for enhancing the presentation. For more information, see Content on page 1.
