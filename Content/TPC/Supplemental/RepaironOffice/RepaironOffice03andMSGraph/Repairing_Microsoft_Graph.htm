Repairing Microsoft Graph

  1. Click the Windows Start icon.
  2. Select Control Panel.
  4. After all programs populate, select Microsoft Office 2003.
  5. Click Change.

  6. Select Add or Remove Features and click Next.

  7. Select Choose advanced customization of applications and click Next.

  8. Click the + to the left of “Office Tools.”

  9. Click the disk to the left of Microsoft Graph, select Run all from My Computer and click Update.
