Generating an Attendance Report

  1. From the Manage tab, select a session from the left panel.

    The Session Overview screen is displayed.

  2. Click Reports at the bottom of the Session Overview screen.

    Double-clicking the session name also opens the Reports window.

  3. Select Attendance from the Report Type drop-down menu in the upper right corner.

    Points are not given for the Attendance Poll or in the Attendance report. Participation points can be assigned in Results Manager.For more information, see Setting Participation Points on page 1.
