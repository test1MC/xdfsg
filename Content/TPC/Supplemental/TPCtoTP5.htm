TurningPoint Cloud to TurningPoint 5

Moving to from TurningPoint 5 to TurningPoint Cloud is intended to be a one-way migration. However, if you need to return to TurningPoint 5, the steps below will assist in the process.

Copies are made of files converted from TurningPoint 5 to TurningPoint Cloud. The source file remains in its original location.

Content

If content was created in TurningPoint Cloud, it must be exported into TurningPoint 5. If content was brought into TurningPoint Cloud from TurningPoint 5, skip these steps and use the original TurningPoint 5 content.

  1. Open TurningPoint Cloud and sign into your Turning Account.
  2. Click the Content tab.
  3. Click the Content drop-down menu and select Export.
  4. Select the content to export.
  5. Click the ... button to save the file to a specified location.
  6. Open TurningPoint 5 and select the Content tab.
  7. Click the Content drop-down menu and select Import.
  8. Select the file to be imported and click Open.

Participant List

Participant list files cannot be directly exported back to TurningPoint 5 because of the encryption of personal information that is applied in TurningPoint Cloud. Follow the workaround process below to return personal data to TurningPoint 5.

  1. Open TurningPoint Cloud and sign into your Turning Account.
  2. Click the Manage tab, select the appropriate participant list and click Edit Participants.
  3. Click Participant, mouse over Export and select CSV.
  4. Name and save the file.
  5. Click Save and Close and close TurningPoint Cloud.
  6. Open TurningPoint 5 and select the Manage tab.
  7. Click the Participant List drop-down menu and select New.
  8. Name the participant list and click Create List.
  9. Click Participant, mouse over Import and select CSV.
  10. Locate the CSV file and click Open.

    The Import List window opens.

  11. Within the Data section, select Contains Headers.
  12. Click Import.

Session Data

Session files also include personal identifying information that is encrypted in TurningPoint Cloud. To return the data back to TurningPoint 5, it must be manually entered following the steps below.

  1. Open TurningPoint 5 and select the Manage tab.
  2. Select the appropriate participant list and click Results Manager.
  3. Click Columns and select Add Manual Column.

    The Manual Column window opens.

  4. Name the column and adjust performance/attendance points, if necessary.
  5. Click Add.
  6. Open TurningPoint Cloud and sign in to your Turning Account.
  7. Click the Manage tab.
  8. Select the appropriate participant list and click Results Manager.
  9. Manually copy the scores.
  10. Paste the scores into the Results Manager manual column created in TurningPoint 5.

    The data is not viewable or accessible in TurningPoint reports. However, the data will be included for all participant scoring for the overall course.
