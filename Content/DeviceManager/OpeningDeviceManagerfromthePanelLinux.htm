Opening Device Manager from Dash Home (Linux Ubuntu 14.04)

In order to use Turning Technologies whiteboards (DualBoard, Touch Board and Touch Board Plus), Linux users must install and run the Ubuntu GNOME environment. Refer to Opening Device Manager in Ubuntu GNOME.

  1. Click Dash Home and search for Device Manager.
  2. Select eInstruction Device Manager.

    If File > Exit is selected, Device Manager closes.
    To reopen Device Manager, search for the program in Dash Home or follow the path: Home > eInstruction > Device Manager > Device Manager.
