MOBI Reports

The MOBI reports tab is only visible if Flow is installed.

This tab is only available for Mobi receivers.

If using a MobiView, the following reports can be accessed from the MOBI Reports tab:

  • Question Status
  • Response per Person
  • Response Grid
  • Answer Distribution
  • Session Results by Student
  • Session Summary

If a report is not selected, it will not be viewable on the MobiView.

If using CPS, navigate to CPS Settings\Delivery Options and click the KWIK Screen tab.
