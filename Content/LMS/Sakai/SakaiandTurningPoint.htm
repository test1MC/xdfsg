Sakai® and TurningPoint® Cloud

The integration with Sakai allows for Turning Technologies users to leverage response devices in class to easily collect student achievement data. Very simply one can import from Sakai a participant list of students into TurningPoint Cloud and then export assessment data into Sakai.

Users can import participant information from Sakai.

  6. Select Sakai from the Integration drop-down menu and enter the Server Address in the box provided. Depending on your institution's application protocol, enter http:// OR https:// before the URL.

  7. Enter your Sakai Username and Password.

    The Import Participant List window is displayed.

  8. Select the course(s) in which the participant list(s) is to be imported and click Import.

    The imported participant list contains the following information: Device ID(s), License Status, First Name, Last Name, User ID and Email.

A participant list from Sakai can be loaded into TurningPoint Cloud to track participant results.

  1. Open TurningPoint.
  3. From the Polling tab, select the participant list, the content file (optional) and the polling environment.

    A Sakai participant list must be loaded prior to running a session.

  6. Close the polling environment to return to the TurningPoint Dashboard.

Participant lists can be updated with the latest participant information from Sakai through the export function in Results Manager.

  6. Select Sakai from the Integration drop-down menu and enter the Server Address in the box provided. Depending on your institution's application protocol, enter http:// OR https:// before the URL.

  7. Enter your Sakai Username and Password.

  8. The following message is displayed if there are any unlicensed participants.

  9. Click Close.

Exporting a Session to Sakai

Sessions can be exported to Sakai via Results Manager.

  6. Select Sakai from the Integration drop-down menu and enter the Server Address in the box provided. Depending on your institution's application protocol, enter http:// OR https:// before the URL.
  7. Enter your Sakai Username and Password.

  8. Select Export Session(s).

    Click Update List to update the participant list with the latest participant list information available from Sakai.

  10. Click Export.

 

Login to your Sakai account to view the exported results data.
