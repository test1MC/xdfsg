Sakai® Install Guide for TurningPoint® Cloud

Before installation, ensure that there are no existing Turning Technologies tools installed. (components\turningtool-pack, shared\lib\turningtool-api-0.1.jar, shared\lib\turningtool-webservices-0.1.jar, webapps\turningtool-tool.war, and webapps\turningtool-tool)

  1. Extract the .zip file in the Sakai root directory (components,shared, webapps, directories).

    If using Shibboleth, copy and paste the text below:
    sakai.properties:#Turning Technologies Shibboleth
    config: turningtool.shibbolethEnabled=true

    If using CAS, copy and paste the text below:
    sakai.properties:
    #Turning Technologies CAS config: turningtool.casBaseUrl=<CAS_SERVER_PATH>

  2. Restart the Sakai Server.
  3. Log in to Sakai as an Instructor.
  4. Select the desired course to add the LTI link.
  5. Select Site Info.
  6. Select Edit Tools.
  7. Select Basic LTI.
  8. Select Continue.
  9. Enter a Title.
  10. Select Continue.
  11. Select Finish.
  12. Select the Basic LTI link from the left menu.
  13. Select Edit in the top right portion of the window.
  14. Enter the Remote Tool URL. The Tool Base URL below is an example of what should be entered: https://account.turningtechnologies.com/account/lti?lmsUrl=http://sakai.university.com/
  15. Enter the Remote Tool Key provided by Turning Technologies.

    Requests should be made to support@turningtechnologies.zendesk.com .  Please include the Sakai URL with the request. 

  16. Enter the Remote Tool Secret provided by Turning Technologies.
  17. Set Button Text to the desired name.
  18. Set Tool Title to the desired name.
  19. Check Open in a New Window.
  20. Check Send Names to the External Tool.
  21. Check Send Email Addresses to the External Tool.
  22. Enter the following Custom Launch Parameter:
    • lms_type=Sakai
  23. Select Update Options.
