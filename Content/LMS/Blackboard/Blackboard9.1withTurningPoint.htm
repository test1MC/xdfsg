Blackboard® 9.1 and TurningPoint® Cloud

The integration with Blackboard® allows for Turning Technologies users to leverage response devices in class to easily collect student achievement data. Very simply one can import from Blackboard a participant list of students into TurningPoint Cloud and then export assessment data into Blackboard.

Downloading a Participant List

Users can import participant information from Blackboard.

  6. Select Blackboard from the Integration drop-down menu and enter the Server Address in the box provided. Depending on your institution's application protocol, enter http:// OR https:// before the URL.

  7. Enter your Blackboard Username and Password.

    The Import Participant List window is displayed.

  8. Select the course(s) in which the participant list(s) is to be imported and click Import.

    The imported participant list contains the following information: Device ID(s), License Status, First Name, Last Name, User ID and Email.

A participant list from Blackboard can be loaded into TurningPoint Cloud to track participant results.

Exporting a Session to Blackboard

Sessions can be exported to Blackboard via Results Manager.

  2. Open TurningPoint Cloud and sign in to your Turning Account.
  3. Click the Manage tab on the TurningPoint Cloud Dashboard.

  4. A Turning Account license is mandatory. If a participant does not purchase a Turning Account license, his or her score will show as an asterisk in all TurningPoint Cloud participant reports. Furthermore, his or her score will export as "0" to an Excel workbook and will not be exported to an LMS.

  6. Select Blackboard from the Integration drop-down menu and enter the Server Address in the box provided. Depending on your institution's application protocol, enter http:// OR https:// before the URL.
  7. Enter your Blackboard Username and Password.

  8. Select Export Session(s).

    Click Update List to update the participant list with the latest participant list information available from Blackboard.
