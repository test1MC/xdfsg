Blackboard® 8 Installation Guide

This guide provides Blackboard® System Administrators with step-by-step instructions to install the Turning Technologies building blocks.

Installing the Building Blocks for Blackboard 8

  1. Log into Blackboard 8 as the Administrator.
  2. Select the System Admin page and click Building Blocks.

  3. Click Install Building Blocks.

  4. Click Browse and select BBTP-v2.1.war, click Browse again and select Reg_BB8_v1.3.war and click Submit.

  5. Change both tools to Available for Availability & Course Defaults.

    If the school does not want the tool to be available to all courses by default, leave Course default as ‘Unavailable’ for registration block.

  6. Click Properties for Blackboard-TurningPoint.
  7. Enter the Mail Server URL and valid UserID and Password.

    This information is necessary to enable instructors to e-mail student grades via the Turning Technologies Integration Wizard.


    Optionally, the mail server settings may be left blank, however the next step IS required.
  8. Click Submit. DO NOT CLICK CANCEL.
