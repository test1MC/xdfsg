Blackboard® Device ID Search Tool
for Administrators

The Administrator role is system wide. Regardless of the course role, the Turning Technologies Registration Tool will always display the Device ID Search Tool.

Using the Device ID Search Tool

  1. Log into Blackboard as the Administrator.
  2. Click the Courses tab and select a course.
  3. Click the Tools link from the left panel.
  4. Select the Turning Technologies Registration Tool.

  5. Enter the Device ID in the box provided and click Search.
  6. To delete a Device ID, check the box that corresponds to the appropriate student name and click Delete.

  7. To delete all Device IDs, check the box next to Select and click Delete.
