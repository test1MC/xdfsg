Blackboard® 9.1 Install Guide for TurningPoint® Cloud

  1. Log in to Blackboard 9.1 as the Administrator.
  2. Select the System Admin tab in the upper right-corner.
  3. Locate the Building Blocks section and click the Building Blocks link.
  4. Click Installed Tools.
  5. Select the Upload Building Blocks tab.
  6. Click the Browse button to locate the TurningTechnologies.war file.
  7. Click Submit.
  8. Change the Turning Technologies Building Block from Inactive to Set Available.
  9. Click Approve in the upper right-corner of the Privileges page.

Installing the LTI Link

  1. Locate the Building Blocks section and click the Building Blocks link.
  2. Click LTI Tool Providers.
  3. Select the Register Provider Domain tab.
  4. In the Provider Domain field, enter ‘account.turningtechnologies.com.’
  5. Change Default Configuration to Set Globally.
  6. Enter the Tool Provider Key provided by Turning Technologies.
  7. Enter the Tool Provider Secret provided by Turning Technologies.

    Requests for the Tool Provider Key and Tool Provider Secret should be sent to support@turningtechnologies.zendesk.com. Please include the Blackboard URL with the request.

  8. Enter the following in Tool Provider Custom Parameters. Each parameter must be on its own line.
    • lms_type=Blackboard
    • user_id=@X@user.id@X@
  9. To the right of Send User Data, select Send user data over any connection.
  10. To the right of User Fields to Send, select Role in course, Name and Email Address.
  11. To the right of Show User Acknowledgment Message, select No.
  12. Click Submit.
  13. Mouse over account.turningtechnologies.com, click the drop-down arrow and select Manage Placements.
  14. Select Create Placement.
  15. In the Label field, enter the desired tool name.

    'Turning Account Registration' is preferred.

  16. In the Handle field, enter the same name as in the Label field.

    Optionally, add an icon which displays to the left of the Tool name in the course.

  17. Enter the URL in the Tool Provider URL field. The LMS URL below is an example of what should be entered:https://account.turningtechnologies.com/account/lti?lmsUrl=http://blackboard.university.com/
  18. Click Submit.

Enabling the LTI Link for All Courses

  1. Select the System Admin tab in the upper right-corner.
  2. Locate the Tools and Utilities section and click the Tools link.
  3. Set Course and Organization availabilities for Turning Account Registration as desired.

    Always on for both Course Tool and Organization Tool is recommended to allow any instructor in the institution access to the direct integration software. This setting also allows any participant, at any time, to register a ResponseCard or a ResponseWare Device ID.
