Blackboard® 9.1 Installation Guide
(Service Pack 8+)

This guide provides Blackboard® System Administrators with step-by-step instructions to install the Turning Technologies building block.

Installing the Building Block for Blackboard 9.1

  1. Log into Blackboard 9.1 as the Administrator.
  2. Select System Admin in the upper right corner.

  3. Click the Building Blocks link under the Building Blocks section.

  4. Select Installed Tools.

  5. Click Upload Building Blocks.

  6. Click Browse to locate the TurningTechnologies.war file.

  7. Click Submit.

  8. Change the Turning Technologies building block from Inactive to Set Available.

  9. Click Approve in the upper right corner of the Privileges page.

  10. Select Settings from the Turning Technologies building block drop-down menu.

  11. Enter the mail server URL and valid User ID and Password.

    This information is necessary to enable instructors to e-mail student grades via the Turning Technologies Integration Wizard.

    Optionally, the mail server settings may be left blank, however the next step IS required.

  12. Optionally, if configuring a proxy server, enter the Proxy Host and Proxy Port.
  13. Below Configure ResponseWare, select Enabled or Disabled from the Enable ResponseWare drop-down menu.
  14. Click Submit. DO NO CLICK CANCEL.

  15. Select System Admin.
  16. Click the Tools link under the Tools and Utilities section.
  17. Set Course and Organization availabilities for Turning Technologies as desired.

    'Always on’ for both Course Tool and Organization Tool is recommended to allow any instructor in the institution access to the direct integration software. This setting also allows any participant to register a ResponseCard or a ResponseWare Device ID at any time.
