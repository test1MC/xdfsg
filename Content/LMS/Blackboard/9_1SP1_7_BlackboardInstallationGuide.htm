Blackboard® 9.1 Installation Guide
(Service Packs 1-7)

This guide provides Blackboard® System Administrators with step-by-step instructions to install the Turning Technologies building block.

Installing the Building Block for Blackboard 9.1

  1. Log into Blackboard 9.1 as the Administrator.
  2. Select SystemAdmin in the upper left corner.
  3. Click Building Blocks under the Building Blocks section.

  1. Select Installed Tools.

  1. Click Upload Building Blocks.

  1. Click Choose File to locate the TurningTechnologies.war file.

  1. Click Submit.

  1. Change the Turning Technologies building block from Unavailable to Available.

  1. Click Approve in the upper right corner of the Privileges page.

  1. Set Course/Org Default for Turning Technologies as desired.

‘Available’ is recommended to allow any instructor in the institution access to the direct integration software. This setting allows any participant to register a ResponseCard or ResponseWare Device ID at any time.

  1. Click the Settings button for the Turning Technologies building block.

  1. Enter the mail server URL and valid User ID and Password.

    This information is necessary to enable instructors to e-mail student grades via the Turning Technologies Integration Wizard.


Optionally, the mail server settings may be left blank, however the next step IS required.
  1. Click Submit. DO NO CLICK CANCEL.
