Moodle™ for TurningPoint® Instructors

Registering a Device ID

  1. Log into Moodle.
  2. Click the appropriate course.
  3. Click Turn editing on in the top right corner.
  4. Select Turning Technologies from the Add a block drop-down menu.
  5. Click the Turning Technologies Device ID Search Tool link.

  6. Click the Register ResponseCard Device ID link associated with the participant with whom you would like to register.

  7. Enter the Device ID in the provided field and click Register.

Editing a Participant's ResponseCard Device ID

If a participant entered an incorrect ResponseCard Device ID, the instructor has the option to edit the Device ID. A ResponseWare Device ID cannot be edited.

  1. Click on a participant’s Device ID.

  2. Edit the hexadecimal characters and click Update when finished.

    The Device ID is changed.

Downloading a TurningPoint Participant List

The instructor can download a participant list from Moodle and import it into TurningPoint.

  1. On the course page, click the Download TurningPoint Participant List icon.
  2. Select Save File and click OK.
  4. Click the Participant List drop-down menu and select Import.
  5. Select TurningPoint Participant List and click Choose File...
  6. Browse to the saved location and click Open.

    The participant list now appears under the Manage tab.

    Use the E-mail Unregistered Students feature to send an e-mail reminder to participants who have not yet registered their devices (ResponseCards and ResponseWare).
