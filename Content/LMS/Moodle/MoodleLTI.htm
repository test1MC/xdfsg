Moodle™ Install Guide for TurningPoint® Cloud

Perform a backup of mod/lti on the server before following the below installation instructions. This mod contains 'lti' and 'turningtech' directories. The LTI directory contains modifications to existing LTI functionality in order to pass the 'Moodle User Name.'  In IMS LTI this is the lis_person_sourcedid which is not an option using the default LTI tool. 

Ensure that block/turningtech is not installed on the system and is not in the directory prior to installation.

  1. Enable Mcrypt and SOAP in PHP.
  2. Unzip the turningtech.zip file.
  3. Move the lti and turningtech folders into your Moodle Server mod directory.
  4. Allow the LTI directory to be overwritten.
  5. Log in to Moodle as the Administrator and select Notifications in Site Administration.
  6. Follow the prompts to make the mods available.

Installing the LTI

  1. Log in to Moodle as an Instructor or Administrator.
  2. Navigate to the desired course.
  3. Click the Turn editing on link.
  4. Click the Add an activity or resource link.
  5. Select External Tool from the Activities menu.
  6. Click Add.
  7. Enter a name in the Activity Name field, i.e. Turning Account Registration.
  8. To the right of External tool type, select +.
  9. Enter a name in the Tool Name field, i.e. Turning Account Registration.
  10. Enter the Tool Base URL.  The Tool Base URL below is an example of what should be entered:

    https://account.turningtechnologies.com/account/lti?lmsUrl=http://moodle.university.com/ 

  11. Enter the Consumer Key provided by Turning Technologies.
  12. Enter the Shared Secret provided by Turning Technologies.

    Requests for Keys and Secrets should be sent to support@turningtechnologies.zendesk.com.  Please include the Moodle URL with the request.

  13. Enter the following Custom parameter:
    • lms_type=Moodle
  14. Change the Default Launch Container to New window.
  15. Change Share launcher's username with tool to Always.
  16. Change Share launcher's name with tool to Always.
  17. Change Share launcher's email with tool to Always.
  18. Change Accept grades from the tool to Always.

    Optionally, select the box labeled Force SSL.

  19. Click Save changes.
