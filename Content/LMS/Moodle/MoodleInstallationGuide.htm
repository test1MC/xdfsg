Moodle™ Installation Guide

This guide provides Moodle System Administrators with step-by-step instructions to install the Turning Technologies plugin.

Installing the Plugin for Moodle

  1. Enable Mcrypt and SOAP in PHP.
  2. Unzip the turningtech.zip file.
  3. Move the mod folder into your Moodle Server root directory.
  4. Log in as the Moodle Administrator and select Notifications in Site Administration.
  5. Follow the prompts to make the mod available.
  6. Move the block folder into your Moodle Server root directory.
  7. Select Notifications in Site Administration.
  8. Follow the prompts to make the plugin available.
  9. If you wish to change any of the default configurations, go to Modules > Activities > Turning Technologies in the Site Administration block.

Adding the Turning Technologies Block to a Course

  1. Log into the desired course as a user with course editing rights and Turn Editing On.
  2. From the Blocks drop-down menu, select the Turning Technologies block.

    The Turning Technologies block is now available in the course and ready for use.

    Do not add the activity TurningTech to a course. Instructors and students will not see the option to add to courses.
