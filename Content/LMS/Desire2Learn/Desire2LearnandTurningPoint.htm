Desire2Learn® and TurningPoint® Cloud

The integration with Desire2Learn® allows for Turning Technologies users to leverage response devices in class to easily collect student achievement data. Very simply one can import from Desire2Learn a participant list of students into TurningPoint Cloud and then export assessment data into Desire2Learn.

Users can import participant information from Desire2Learn.

  6. Select Desire2Learn from the Integration drop-down menu and enter the Server Address in the box provided. The Server Address is the URL of the registration link normally residing in the News section. Depending on your institution's application protocol, enter http:// OR https:// before the URL.

  7. Click Connect.

    The Desire2Learn Login window is displayed.

  8. Enter your Desire2Learn Username and Password and click Login.

    The Import Participant List window is displayed.

  9. Select the course(s) in which the participant list(s) is to be imported and click Import.
  10. Click OK.

    The imported participant list contains the following information: Device ID(s), License Status, First Name, Last Name, User ID and Email.

A participant list from Desire2Learn can be loaded into TurningPoint Cloud to track participant results.

  1. Open TurningPoint.
  3. From the Polling tab, select the participantlist, the content file (optional) and the polling environment.

    A Desire2Learn participant list must be loaded prior to running a session.

  6. Close the polling environment to return to the TurningPoint Cloud Dashboard.

Participant lists can be updated with the latest participant information from Desire2Learn through the export function in Results Manager.

  6. Select Desire2Learn from the Integration drop-down menu and enter the Server Address in the box provided.   7. Click Connect.

  8. At the Connect to Integration window, enter your Desire2Learn Username and Password and click Login.

Exporting a Session to Desire2Learn

Sessions can be exported to Desire2Learn via Results Manager.

  6. Select Desire2Learn from the Integration drop-down menu and enter the Server Address in the box provided. The Server Address is the URL of the registration link normally residing in the News section. Depending on your institution's application protocol, enter http:// OR https:// before the URL.
  7. Click Connect.
  8. At the Connect to Integration window, enter your Desire2Learn Username and Password and click Login.

  9. Select Export Session(s).

  11. Click Export.

 

Login to your Desire2Learn account to view the exported results data.
