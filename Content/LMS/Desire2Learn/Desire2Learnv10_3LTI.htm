DESIRE2LEARN® v10.3 LTI Install Guide for TurningPoint® Cloud

Setting Up the LTI

  1. Log in to Desire2Learn with an administrative account.
  2. Click the Settings icon in the upper right-corner of the home page.
  3. Click the External Learning Tools link.
  4. Click the New Link button.
  5. Enter a name for the registration tool in the Title field.
  6. Enter the URL in the URL field. The LMS URL below is an example of what should be entered: https://account.turningtechnologies.com/account/lti?lmsUrl=http://school.turningtechnologies.com/ 

    Optionally, enter a description of the tool in the Description field.

  7. Select the box labeled Allow users to view this link.
  8. Select the box labeled Sign messages with key/secret with.
  9. Enter the Key provided by Turning Technologies
  10. Enter the Secret provided by Turning Technologies.

    Requests for Keys and Secrets should be sent to support@turningtechnologies.zendesk.com.

  11. Click the Add Custom Parameters link.
  12. Enter the following parameters:
    • Name lms_type
    • Value D2L
  13. Select each option below Security Settings.
  14. Select the box labeled Current Org Unit.
  15. Click Save.
  16. Click My Home.
  17. Locate the News section, click the down arrow and select the + icon (New News Item).

  18. Enter the name of the registration tool in the Headline field.
  19. Click the Insert Quicklink icon.

  20. Select External Learning Tools from the Category drop-down menu.
  21. Select the appropriate external learning tool from the External Learning Tools drop-down menu.
  22. Enter 'Register Your Turning Technologies Device Here' (or any equivalent caption) in the Link Caption field.
  23. Select New Window.
  24. Click Insert.
  25. Below Availability, deselect Always show start date.
  26. Click Now.
  27. Click Publish.

    The student role must have the Launch External Learning Tool Links permission enabled.
