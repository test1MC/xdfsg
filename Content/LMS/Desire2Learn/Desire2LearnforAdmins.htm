Desire2Learn® for Administrators

This document explains the Administrator login process for Desire2Learn®.

Logging into Desire2Learn

  1. Enter the middleware server URL followed by /admin. This is the same URL as the External Learning Tool in Desire2Learn. For example:
    http://'yourschool'.turningtechnologies.com/registrationsystem/admin

  2. Enter admin in the Login ID field and admin in the Password field.
  3. Click Login.
  4. Once in Desire2Learn, administrators have the option to do the following:
    • Create a new user
    • View current registered users
    • Search current users
    • Remove inactive devices
    • Change the account password
