Performance Matters™ and
TurningPoint® Cloud

The integration with Performance Matters™ allows for Turning Technologies users to leverage response devices in class to easily collect student achievement data. Very simply one can import from Performance Matters a participant list of students into TurningPoint Cloud and then export assessment data into the Performance Matters gradebook.

Users can import participant information from Performance Matters.

  6. Select Performance Matters from the Integration drop-down menu and enter the Server Address in the box provided.
  7. Enter your Performance Matters Username and Password.

    The Import Participant List window is displayed.

  8. Select the course(s) in which the participant list(s) is to be imported and click Import.

A participant list from Performance Matters can be loaded into TurningPoint Cloud to track participant results.

  1. Open TurningPoint.
  3. From the Polling tab, select the participant list, the content file (optional) and the polling environment.

    A Performance Matters participant list must be loaded prior to running a session.

  6. Close the polling environment to return to the TurningPoint Cloud Dashboard.

Participant lists can be updated with the latest participant information from Performance Matters through the export function in Results Manager.

  6. Select Performance Matters from the Integration drop-down menu and enter the Server Address in the box provided. Depending on your institution's application protocol, enter http:// OR https:// before the URL.
  7. Enter your Performance Matters Username and Password.

  9. The following message is displayed if there are any unlicensed participants.

  10. Click Close.

Exporting a Session to Performance Matters

Sessions can be exported to Performance Matters via Results Manager.

  6. Select Performance Matters from the Integration drop-down menu and enter the Server Address in the box provided.
  7. Enter your Performance Matters Username and Password.

  8. Select Export Sessions.
  9. Select the column(s) to be exported and click Export.

  10. Click Export.

 

Login to your Performance Matters account to view the exported results data.
