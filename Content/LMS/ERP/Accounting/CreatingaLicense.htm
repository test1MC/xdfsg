Creating a License

  1. Sign in to ERP.
  2. Click Licenses.
  3. Enter the email address of the user for whom you are creating the license.
  4. Click View Licenses.

    One of three scenarios occurs:

    • The user already has a Turning Account and a license. The option to create a license is not available. In this scenario:
      1. Select the license.
      2. Click Update License to edit the license information.
    • The user has a Turning Account but not a license. In this scenario:
      1. Click Create License.
      2. Select the product type from the Product drop-down menu.
      3. Select the license type from the Type drop-down menu.
      4. If applicable, enter the number of seats in the Seats field.
      5. Set the license expiration date using the drop-down menus below Expires.
      6. If the license is sharable, click Yes. If it is not, click No.
      7. Click Create License.
    • The user does not have a Turning Account or a license. The option to create a license is not available. In this scenario:
      1. Click Create Invite.
      2. Select the product type from the Product drop-down menu.
      3. Select the license type from the Type drop-down menu.
      4. If applicable, enter the number of seats in the Seats field.
      5. Set the license expiration date using the drop-down menus below Expires.
      6. If the license is sharable, click Yes. If it is not, click No.
      7. Click Create Invite.

        A Turning Account invite is sent to the user.
