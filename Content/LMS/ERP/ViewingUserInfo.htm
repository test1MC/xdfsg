Viewing User Info

  1. Sign in to ERP.
  2. Click View User Info.
  3. Enter a participant email address and click View User Info.

    The User Info page is displayed.

    The following information is available:

    Email addressThe email address used to create the Turning Account.
    Turning IDThe ID assigned by Turning Technologies to the account.
    User IDThe ID entered by the user during Turning Account creation.
    ResponseWare IDThe user's unique ResponseWare identifier.
    LMS IDThe user's LMS ID as defined by the LMS.
    LMS URLA list of all learning management systems to which the user is associated.
    ResponseCard IDsIf the user registered a ResponseCard in his/her Turning Account, the ResponseCard ID is displayed here.

    The following additional information is displayed:

    License IDThe ID assigned by Turning Technologies to a particular license.
    ProductThe Turning Technologies product the user is utilizing.
    TypeThe type of license associated to the product.
    SeatsThe number of seats that the license contains.
    SharableIf the license is sharable, the text is displayed as True. If the license is not sharable, the text is displayed as False.
    ExpiresThe license expiration date.
