Acuity® and Self-Paced Polling for PCMac

The integration with Acuity® allows for Turning Technologies users to leverage response devices in class to easily collect student achievement data. Very simply one can import from Acuity a participant list of students into Self-Paced Polling and then export assessment data into the Acuity gradebook.

Acuity is only compatible with the Self-Paced Polling environment.

Downloading an Acuity Question List

Users can import an Acuity question list, which also contains participant information.

  1. Open TurningPoint.
  2. Select Auto below the Participants section of the Polling tab.

  1. Select None below the Content section of the Polling tab.

  1. Click Self-Paced Polling.

    The following message is displayed.

  1. Click Continue to Self-Paced Polling.
  2. Click ToolsWindow and select Connect to Integration.

  1. Select Acuity from the Integration drop-down menu and enter the Server Address in the box provided.
  2. Enter your Acuity Username and password. Select the institution name from the Institution drop-down box if available and click Connect.
  3. Select Import Participant List and Acuity Question List and click Continue.

    The following window is displayed.

  1. Select the district, school, class, subject and assessment from each drop-down menu.
  2. Select the assignment(s) and click Download.

    A prompt is displayed stating that the dated imported successfully. Click OK.

    The data is displayed in Self-Paced Polling.

Using Self-Paced Polling

Listed below are the basic steps for running Self-Paced Polling. Please refer to the TurningPoint user guide for more information regarding Self-Paced Polling.

  1. Plug in the receiver.
  2. Click Accept Logins.
    1. Select the Participant Login option from the drop-down menu.
    2. Select the Time Limit from the drop-down menu.
    3. Select the Completed Test Message from the drop-down menu.
  3. Click OK.

    At this time participants may login to the test.

  4. Click Start Polling.

    Participants may now begin the test. After they have finished the test, they can submit their results from the response device.

    While a test is being administered, click Live Results on the Self-Pace Polling toolbar to view individual and overall progress.

  5. Click Stop Polling after all the results have been received.
  6. Click Save to save the session results.
  7. Name the session and click Save.

Exporting a Session to Acuity

Sessions can be exported to Acuity via Self-Paced Polling.

  1. Click ToolsWindow and select Connect to Integration.
  2. Select Acuity from the Integration drop-down menu and enter the Server Address in the box provided.
  3. Select Submit Student Responses and click Continue.
  4. Select either the Current Session or Choose Session radio button. If Choose Session is selected, click Browse to locate the saved session file.
  5. Click Export.

 

In 24 hours, login to your Acuity account to view the exported results data.
