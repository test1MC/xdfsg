About the Web Registration Tool

Turning Technologies’ Web Registration Tool is the most basic registration offering available.  It is free, easy-to-use and provides the ability to create participant lists with TurningPoint.  Participants are able to register their devices via a central portal while presenters can create a basic participant list to sync with registrations.

Before using the Web Registration Tool, the following are required:

  • Participant registration on Turning Technologies’ hosted web registration site
  • A participant list with participants’ first and last names and User IDs
  • A saved PowerPoint Polling, Anywhere Polling or Self-Paced Polling session ran with a participant list including first and last names and User IDs
