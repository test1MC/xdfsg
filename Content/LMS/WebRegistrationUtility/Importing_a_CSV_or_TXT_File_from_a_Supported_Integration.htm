Importing a CSV or TXT File from a Supported Integration

Presenters can import a CSV or TXT file from a supported Integration such as Blackboard, D2L, Canvas, etc.

  1. Open TurningPoint.
  2. Select the Manage tab.
  3. Click the Participant List drop-down menu and select Import.
  4. Select the File from Integration radio button and choose the appropriate integration from the Integration drop-down menu.

  1. Click Choose File...
  2. Locate the saved CSV or TXT file and click Open.
  3. Click Save and Close.

For more information about exporting CSV files from Integrations into TurningPoint participant lists, refer to the TurningPoint User Guide.
