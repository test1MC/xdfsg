Importing a CSV or TXT File

Presenters can import a CSV or TXT file into a TurningPoint participant list.

  1. Open TurningPoint.
  2. Select the Manage tab.
  4. Name the new participant list and click Create List.

  1. Click the Participant drop-down menu, mouse over Import and select CSV or TXT.

  1. Locate the saved CSV or TXT file and click Open.
  2. Click Save and Close.

For more information about importing a CSV or TXT file into a
participant list, see Importing a CSV or TXT File in the TurningPoint User Guide.
