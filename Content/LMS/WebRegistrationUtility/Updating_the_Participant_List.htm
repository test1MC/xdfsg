Updating the Participant List

After a session is ran and saved, the Web Registration Tool syncs Device IDs to participants and User IDs.

  1. Select the Manage tab.
  2. Select the participant list associated with the saved session and click the Results Manager icon in the bottom right corner.
  3. Click the Integrations icon on the Results Manager toolbar.

  1. Select Web Registration Tool from the Integration drop-down menu.
  2. Leave the server URL as http://webreg.turningtechnologies.com/ unless otherwise instructed by your IT Administrator and click Connect.
  3. Click Update List. Click Close once the sync is complete.
  4. The Device IDs will now be associated to the participants who registered at
    http://webreg.turningtechnologies.com.
  5. Device IDs for participants who did not register will appear under Unassigned Devices.

 

If necessary, use the Web Registration Tool to update your participant list after each session.
