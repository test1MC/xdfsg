Participant List

A participant list can be manually created or imported from a CSV or TXT file, or from a supported Integration.

Manually Creating a Participant List

Presenters can manually create a participant list that consists of first and last names and User IDs.

  1. Open TurningPoint.
  2. Select the Manage tab.
  4. Name the new participant list and click Create List.

  1. Select a cell and enter the participant information.

Only the First Name, Last Name and User ID columns are required.

  1. Click Save and Close.
