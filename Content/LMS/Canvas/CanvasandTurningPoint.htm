Canvas® and TurningPoint® Cloud

The integration with Canvas® allows for Turning Technologies users to leverage response devices in class to easily collect student achievement data. Very simply one can import from Canvas a participant list of students into TurningPoint Cloud and then export assessment data into Canvas.

Registering a Response Device

A ResponseCard or ResponseWare Device ID must be registered to receive credit in class.

  1. Enter your ResponseCard or ResponseWare Device ID in the provided field.
  2. Enter the Captcha in the provided field.
  3. Click Register Device.

    The following window is displayed.

  4. Verify the information that you entered is correct. Click Final Submission to register the Device ID. Click Cancel to start over.

A participant list from Canvas can be loaded into TurningPoint Cloud to track participant results.

  1. Open TurningPoint.
  3. From the Polling tab, select the participant list, the content file and the polling environment.

    A Canvas participant list must be loaded prior to running a session.

  6. Close the polling environment to return to the TurningPoint Cloud Dashboard.

Participant lists can be updated with the latest participant information from Canvas through the export function in Results Manager.

  6. Select Canvas from the Integration drop-down menu and enter the Server Address in the box provided. The Server Address is the URL of the registration link normally residing in the course’s Modules section. Depending on your institution's application protocol, enter http:// OR https:// before the URL.
  7. Click Connect.

  8. At the Connect to Integration window, enter your Canvas Username and Password and click Login.

  9. The following message is displayed if there are any unlicensed participants.

  10. Click Close.

Exporting a Session to Canvas

Sessions can be exported to Canvas via Results Manager.

  6. Select Canvas from the Integration drop-down menu and enter the Server Address in the box provided. The Server Address is the URL of the registration link normally residing in the course’s Modules section. Depending on your institution's application protocol, enter http:// OR https:// before the URL.
  7. Click Connect.
  8. At the Connect to Integration window, enter your Canvas Username and Password and click Login.

  9. Select Export Session(s).

  11. Click Export.
