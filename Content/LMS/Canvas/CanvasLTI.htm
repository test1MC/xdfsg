Canvas® Install Guide for TurningPoint® Cloud

Setting Up the LTI

  1. Log in to Canvas with an administrative account.
  2. Mouse over Courses.
  3. Click the appropriate account name below Managed Accounts.
  4. Click Settings from the left menu.
  5. Select the Apps tab.
  6. Click View App Configurations.
  7. Click Add New App.

    If any External Apps were previously enabled,  this option will be located under View Installed Apps.

  8. Enter 'Turning Account Registration' in the Name field.
  9. Enter the Consumer Key provided by Turning Technologies.
  10. Enter the Shared Secret provided by Turning Technologies.

    Requests for Keys and Secrets should be sent to support@turningtechnologies.zendesk.com.

  11. Click the Configuration Type drop-down menu and select Manual Entry.
  12. Enter the URL.  The LMS URL below is an example of what would be entered: https://account.turningtechnologies.com/account/lti?lmsUrl=http://school.turningtechnologies.com/  
  13. Click the Privacy drop-down menu and select Public.
  14. Enter lms_type=Canvas in the Custom Fields field.

    Optionally, enter the description of the LTI link in the Description field.

  15. Click Submit.

Adding the LTI to a Course

  1. Log in to Canvas with an administrative account.
  2. Mouse over Courses.
  3. Click the appropriate course.
  4. Click Modules from the left menu.
  5. Select Add a New Module.
  6. Enter 'Turning Account Registration' or desired name in the Name field.
  7. Leave lock module until a given date unchecked.
  8. Leave students must move through requirements in this module in sequential order unchecked.
  9. Click Add Module.
  10. Below Turning Account Registration, select Add item to module.
  11. Click the drop-down menu to the right of Add and select External Tool.
  12. Select Turning Account Registration from the list.
  13. Select Load this tool in a new tab.
  14. Click the Indentation drop-down menu and select Don’t Indent .
  15. Click Add Item.
