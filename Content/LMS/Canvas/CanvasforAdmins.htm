Canvas® Administrator Guide

Logging into Canvas

  1. Enter the middleware server URL followed by /admin. This is the same URL as the BLTI link in
    Canvas. For example:
    http://'yourschool'.turningtechnologies.com/CanvasRegistrationSystem/admin
  2. Enter admin in the Login ID field and admin in the Password field.
  3. Click Login.
  4. Once in Canvas, administrators have the option to do the following:
    • Create a new user
    • View current registered users
    • Search current users
    • Remove inactive devices
    • Change the account password
