About ExamView Test Manager

ExamView® Test Manager provides the features you need to track results for paper and online tests created with the ExamView Test Generator. With the ExamView Test Manager, you can...

  • create class rosters that can include a wide range of student information including demographic information.
  • score paper tests automatically using a scanner.
  • administer paper tests and get results immediately using response pads.
  • administer and automatically score LAN-based and Internet tests.
  • produce a wide variety of useful reports, such as a test item analysis, proficiency reports, and much more.
