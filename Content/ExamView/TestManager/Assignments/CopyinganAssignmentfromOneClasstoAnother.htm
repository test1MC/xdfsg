Copying an Assignment from One Class to Another

If you teach multiple sections of the same class and use the same tests for each section, the Copy Selected Assignment command is a quick way to copy an assignment and all of its settings from one class to another.

  1. Highlight the assignment you wish to copy.
  2. Click Edit from the menu bar and select Copy Selected Assignment or use the keyboard shortcut Ctrl+C (PC) or Cmd+C (Mac).
  3. Open the class to which you want to copy the assignment.
  4. Click Edit from the menu bar and select Paste Assignment or use the keyboard shortcut Ctrl+V (PC) or Cmd+V (Mac).
