Changing the Preferences

Changes made to most ExamView Test Manager preferences will only affect the currently open class (this excludes the General preferences and color settings). If you wish to change the preferences for all new classes you create, you must choose to save the preferences as the default preferences.

  1. Click Edit from the menu bar and select Preferences or click Preferences in the Edit Class Information dialog.
  2. Set the preference options as desired.
  3. Click OK to record the settings.

    If a class is open when you change the settings, the program applies the new settings only to the open class. If no class is open, or if you click Save as Default, the new settings will apply to all new classes you create.
