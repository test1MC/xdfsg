Exporting Assignment Results

  1. Open ExamView Test Manager and open or create a class file with at least one assignment.
  2. Click File from the menu bar, mouse over Export and select the target LMS.
  3. Complete the export dialog options and click OK to export the assignment results.
  4. Select the location where you want to save the exported file, enter a file name, and click Save.

    The results are saved as a CSV file.

  5. Log in to your LMS and upload the exported assignment results.
