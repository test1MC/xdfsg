Known Issues

  • Multiple Response questions do not import into D2L and are skipped.
  • Point values do not import into D2L. All questions import with a 1 pt. value.
  • All questions import with no enumeration (e.g. A, B, C or 1, 2, 3) for answer choices.
  • The Blackboard 7.1+ export workaround does not currently work for Desire2Learn v.8.3
