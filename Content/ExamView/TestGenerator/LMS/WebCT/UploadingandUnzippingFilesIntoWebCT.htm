Uploading and Unzipping Files Into WebCT CE 6/Vista 4

  1. Log in to your WebCT course.
  2. From the Create tab, click on the icon or the Manage Course link below Designer Tools.
  3. Click the icon or Import option.
  4. Navigate to the ZIP file and click Open.
  5. Click Return when the Content Import Progress shows that importing is complete.

    The ZIP file is automatically unzipped in the import process. Importing either a test or a question bank generates a quiz and adds the question items to the question database.

  6. To access the quiz on WebCT CE 6/Vista 4:
    1. Click the Assessments link below Course Tools.
    2. Use the menu icon following the title to open the ActionLinks menu for that particular item.
    3. Click Go to Question Database from the Assessments screen.
