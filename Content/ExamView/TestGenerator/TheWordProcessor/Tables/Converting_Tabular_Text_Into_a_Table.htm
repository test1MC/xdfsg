Converting Tabular Text Into a Table

If you plan to create internet tests with tabular material; you should use tables, not tabs.

  1. Select the text you want to convert to a table.

    For best results, the text should be separated by tabs or commas.

  2. Click Table from the menu bar, mouse over Convert and select Text to Table.
  3. If necessary, change the number of columns and table format.
  4. Select the Separate Text At option.
  5. Click OK to convert the text.
