Adjusting Table Cell Height and Width

How to adjust table cell height and width using the menu command...

  1. Select a column or row if you want to change a specific column or row height/width. If you simply place the cursor inside a table, the default column and row is 1.
  2. Click Table from the menu bar and select Cell Height and Width.
  3. To adjust the row height:
    1. Click the Row tab.
    2. Select the Height of row option from the drop-down menu.
    3. If necessary, enter a row height in points (72 pts = 1 inch).
    4. Select the Alignment option from the drop-down menu.
    5. Click Next Row to change to the next row and then set the cell height.
  4. To adjust the column width:
    1. Click the Column tab.
    2. Enter the width of column 1 (in inches) if you need to adjust this cell.
    3. Click Next Column to change to the next column and then set the cell width.
    4. If you want the program to automatically adjust all cells in a table to fit within the margins, click Autofit.
  5. Click OK.

    With ExamView Test Generator (as with other word processors), it is possible to create a table where some columns go past the right margin. In these instances, you can't use the mouse to adjust the cell width. You can use the Cell Height and Width command to set the cell width for those cells you cannot access with the mouse.

How to adjust table cell height and width using the mouse...

  1. To set the cell width using the mouse:
    1. Select a cell if you want to change the width for that cell only. To set the cell width for all cells in a column, select the column or make sure that no individual cells are selected.
    2. Position the mouse pointer on either the left or right cell border. When the pointer changes to a resize cursor, hold down the mouse button to drag the border to the new location. Alternatively, you can drag the cell column indicators shown on the ruler.

  2. To set the row height using the mouse:
    1. Position the mouse pointer on the bottom edge of a row border.
    2. When the pointer changes to a resize cursor, drag the border to the new location. This has the same effect as changing the row height to “At least” the positioned value.
