Splitting Cells

  1. Select one or more adjacent cells.

    You cannot change the layout, split cells, or make any other changes to the predefined multiple choice and matching tables.

  2. Click Table from the menu bar and select Split Cells to divide one cell into two or more cells.
  3. If you selected two or more cells, click Merge cells before splitting to combine the selected cells before splitting the cell into columns. If you selected cells that span multiple table rows, you can enter the number of rows as well.
  4. Enter the number of columns or number of rows for the new cell.
  5. Click OK.
