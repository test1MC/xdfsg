Merging Cells

The Merge Cells command allows you to combine one or more cells in the same row or column to form one new cell. For example, you might want to join all the cells in the top row.

  1. Select two or more adjacent cells. You can join cells that are in the same row or cells that are in the same column.

    You cannot change the layout, join cells, or make any other changes to the predefined multiple choice and matching tables.

  2. Click Table from the menu bar and select Merge Cells.
