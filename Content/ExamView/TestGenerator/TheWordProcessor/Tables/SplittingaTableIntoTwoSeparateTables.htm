Splitting a Table Into Two Separate Tables

  1. Click in the row where you want to split the table. All rows above the selected row will be placed in one table, and the selected row and all subsequent rows will be placed in a new table.
  2. Click Table and select Split Table.

    If you created a table on the first line of a question, you can insert a blank line before the table. Click in the first row, click Table and select Split Table.
