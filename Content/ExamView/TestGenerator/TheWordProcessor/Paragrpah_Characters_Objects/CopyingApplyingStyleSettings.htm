Copying/Applying Style Settings

Copying a style copies these settings: font, size, and style (bold, underline, color, etc.) If you just want to change some text to bold, it's easier to click the Bold toolbar button. However, it can be faster to copy and apply styles when the text you want to format uses multiple formatting options.

  1. Position the cursor anywhere in a text block from which you want to copy its style.
  2. Click Format from the menu bar and select Copy Style.
  3. Select the text you want to format using the settings you just copied.
  4. Click Format from the menu bar and select Apply Style.
