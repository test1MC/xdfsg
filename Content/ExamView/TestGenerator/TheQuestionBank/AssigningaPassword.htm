Assigning a Password to a Question Bank

The program will prompt you to enter a password when you open a question bank or when you create a test using a password-protected question bank.

Be sure to write your password on a piece of paper and store it in a safe place. You will not be able to open the question bank without the password.

  1. Click Edit from the menu bar and select Question Bank Password.
  2. Enter a password and click OK.
  3. Re-enter the password and click OK to confirm the password.

    To remove an existing password, click Edit and select Question Bank Password. Delete the existing password and click OK.
