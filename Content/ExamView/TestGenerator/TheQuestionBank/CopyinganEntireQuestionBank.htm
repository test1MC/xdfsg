Copying an Entire Question Bank

You can copy an entire question bank using the Copy Entire Bank command. Then, you can paste the question bank into a word processor.

  1. Create your question bank.
  2. Click Edit from the menu bar, mouse over Copy and select Entire Question Bank.

    The program places a complete copy of the question bank on the Clipboard.

  3. Switch to another application (e.g., Microsoft Word or WordPerfect) that supports RTF (Rich Text Format).
  4. Paste the question bank into the document.

    You cannot paste the question bank back into the ExamView Test Generator.
