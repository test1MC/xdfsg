Importing Questions from CPS Lessons (PC Only)

If you have all or most of your questions in a Classroom Performance System (CPS) database, you can easily import your CPS lessons into ExamView Test Generator. The program includes step-by-step instructions that explain how to import your questions.

You can only import a single lesson at a time.

  1. Open ExamView Test Generator in either the Test Builder or Question Bank Editor mode.
  2. Click File from the menu bar, mouse over Import and select CPS Lesson.

    A file selection window appears.

  3. Select the CPS file that contains the lesson to import and click Open.

    The Import CPS Lesson window appears. The window displays the lesson file tree structure from the CPS file.

  4. Select the lesson and click OK to complete the import.
