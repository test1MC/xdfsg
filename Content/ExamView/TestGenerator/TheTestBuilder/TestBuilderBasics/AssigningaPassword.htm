Assigning a Password to a Test

Be sure to write your password on a piece of paper and store it in a safe place. You will not be able to open the test without the password.

  1. Click Edit from the menu bar and select Test Password.
  2. Enter a password and click OK.
  3. Re-enter the password and click OK to confirm the password.

    To remove an existing password, click Edit and select Test Password. Delete the existing password and click OK.
