Saving a Test

You can automatically open test files from previous versions of the ExamView Test Generator program. However, once you save a test using the new version, you cannot open that file with an older version. It is recommended that you use a different file name.

  1. Click from the toolbar or click File and select Save.

    Click File and select Save As to save the test with a different file name and/or to save the test to a new file location.

  2. Enter a name for the test.
  3. Identify the folder where you want to save the test. If you want to save the file on a different drive or on a network, choose the location.
  4. Click Save to save the test.
