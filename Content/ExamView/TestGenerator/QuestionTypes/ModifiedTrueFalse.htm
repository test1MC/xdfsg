Modified True/False

A modified true/false question is similar to a standard true/false question, in that a student must indicate if a sentence or phrase is true or false. For these kinds of questions, a student must also change or edit the question if it is false to make it true.

For internet tests, modified true/false questions can be scored only if there is one word or phrase to enter.

  1. Select a modified true/false question and click Edit, or click New and select Modified True/False to create a new question.

  2. When you enter a modified true/false question, identify the word or phrase that needs to be modified. Also, provide an underscore for the student to write their answers.
