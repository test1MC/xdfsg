USING A PICK LIST TO POPULATE A QUESTION’S STANDARD FIELD IN EXAMVIEW® ASSESSMENT SUITE v9 (Windows)

Your ExamView® Learning Series v11 product contains three pick lists of standards to which the questions are aligned:

  • The national Common Core State Standards for English Language Arts (2010)
  • The national Common Core State Standards for Mathematics (2010)
  • The Next Generation Science Standards (2013)

Follow these instructions to set up ExamView Assessment Suite v9 (your desktop application) and to use the pick lists of standards to populate the State Standard field or the National Standard field in an ExamView question.

Setting Up ExamView Assessment Suite

Figure 1 – ExamView Test Generator Splash Screen

  1. Open ExamView Assessment Suite from the ExamView Test Generator icon on your desktop or from your Start menu, select All Programs | eInstruction | ExamView | ExamView Test Generator. The ExamView Test Generator splash screen opens. See Figure 1.
  2. Select Open an Existing Question Bank .
  3. Navigate to the ExamView | Banks | Learning Series folder. The Learning Series folder includes the complete collection of more than 15,600 unique, high-quality assessment items for English language arts, reading, math, science, and social studies.

    The Learning Series folder has three subfolders:

    • Topic-Aligned, which includes more than 15,600 assessment items for English language arts, reading, math, science, and social studies aligned to core curriculum topics (i.e., no state standards)
    • National Common Core, which includes more than 12,100 English language arts, reading, and math assessment items aligned to the Common Core State Standards for English Language Arts and Mathematics (2010)
    • Next Generation Science, which includes more than 1,900 science assessment items aligned to the Next Generation Science Standards (2013)

    Within each of the subfolders, you will see the files for your Learning Series product.

  4. Open a question bank that you would like to edit.

    If you prefer, you can navigate to your own ExamView® question banks and use these steps to add standards to questions you have authored.

  5. Confirm that the ExamView® preferences are set to the state on which you are working. In the ExamView Question Bank Editor, click Edit from the menu bar and select Preferences. See Figure 2.

    Figure 2 - Edit Menu

  6. On the General tab of the Preferences window, confirm that the State/Region is set to your state. See Figure 3.

    Figure 3 - Set State/Region Preferences

  7. Click Save as Default and then click OK.

Using the Standards Pick Lists

Figure 4 - State Standard Field of the
Question Information Window

  1. Double-click the first question in the question bank to which you would like to add a standard.
  2. Click Info at the bottom of the question window.

    The Question Information window appears.

  3. Place your cursor in the State Standard field or the National Standard field immediately after any listed standard. See Figure 4.

    You can add national Common Core State Standards or Next Generation Science Standards to the National Standard and to the State Standard fields.

  4. Click in the top right corner of the Question Information window.

    If you are in the State Standard field, the Add from List: State Standard window will appear. If you are in the National Standard field, the Add from List: National Standard window will appear.

    Figure 5 - National Standard Pick List Menu

  5. Click in the top right corner of the Add from List window.

    The list of available standards is displayed. Each pick list of standards is labeled with CCSS (for the national Common Core State Standards), or NGSS (for Next Generation Science Standards) followed with a subject area (for example, mathematics). See Figure 5.

  6. Select the desired pick list from the menu.

    You can add national Common Core State Standards or Next Generation Science Standards to the National Standard field and the State Standard field. A list of the standards and descriptions for the selected subject will display.

     

     

     

     

    Figure 6 - National Standards List

  7. Select the standard you want to add to the State Standard field or the National Standard field for the selected question. See Figure 6.
  8. Click Add.

    The selected standard will display in the State Standard or National Standard field for this question.

    If a Standard field was already populated with a standard, a pipe character ( | ) will automatically separate the newly added standard from the existing standard. See Figure 7.

  9. Click OK to close the Question Information window.
  10. Click Record to close the question window.
  11. Save your question bank.

    Figure 7 - National Standard Field Populated

  12. To add a standard to subsequent questions, double-click a question, click Info, and place your cursor in the State Standard or National Standard field within the Question Information window.

    Click in the top right corner of the Question Information window. The standards pick list that you last used is displayed.

    • To add a standard from the displayed standards pick list, follow steps 7 - 11.
    • To add a standard from a different standards pick list follow steps 5 - 11.
  13. Always remember to save and close your question bank.
  14. When finished, close the ExamView Assessment Suite.
