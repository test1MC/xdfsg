Preparing Questions in WordExamView Test Generator

ExamView Test Generator provides a convenient way to author questions without worrying about all of the formatting issues inherent with the other authoring options. If you know how to use Microsoft Word, then you already know how to use the ExamView Test Generator Question Bank Editor since the two programs are very similar.

An overview of the authoring process using ExamView Test Generator is presented below. Several helpful hints are presented after the step-by-step instructions. Be sure to review them prior to beginning the authoring process.

Be sure to read the ExamView Assessment Suite User Guide before you begin entering questions.

Tips for Using ExamView® Test Generator

Refer to the following list of tips for using ExamView Test Generator. If you have any techniques that you find useful, please share them with us. See Contact Us.

  • You can set the default number of columns for multiple choice and matching questions. Use the Preferences option in the Edit menu to set the default options. If you have already entered several questions, press Ctrl+M while viewing the question bank to change the column settings for all of the existing multiple choice questions.
  • For online tests, you can provide feedback for each choice of a multiple choice question. Use the feedback field in the question info window to enter a response for each choice. See the online help for exact formatting details.
  • Do not use hard returns as you enter the text. As you enter questions or answers, do not press Enter at the end of every line. Use the word wrap feature unless you specifically need a hard return. Text should be allowed to wrap on its own unless you want to break the lines at a specific point as you would in a poem. Only use hard returns when required.
  • Detailed help is provided for the equation editor. A built-in equation editor is provided to enter equations ranging from simple to complex. Detailed instructions are provided in the online help that give step-by-step instructions for creating an equation.

Before you begin writing questions, take a few minutes to follow the simple suggestions provided to prepare Microsoft Word for entering ExamView Assessment Suite questions. Then, read the overview of the question entry process. You can refer to the Sample Question Bank if you need to see how a question bank is formatted. When you finish you will be ready to start the writing process.

Preparing to use Microsoft Word

When you install Microsoft Word, the program establishes certain default settings. For best results, it is recommended to turn off some of the automatic features.

AutoFormat Options

Turn off all instances of the following auto correct options. Some of them appear more than once.

automatic numbered lists replace text as you type
fractions (1/2) with fraction character format beginning of list item like the one before

Default font

If necessary, set the default font to Times New Roman, 11pt.

  1. Start Microsoft Wordthe software and create a new documentquestion bank.
  2. Enter a question bank title.

    Enter a descriptive title (and optional subtitle) using a style prescribed by your publisher.

    For example, you could enter the chapter number and title in title case or all uppercase.

    Chapter 1—Understand the Basic Concepts or LESSON 5: LEARNING TO BUILD TABLES

    When an instructor selects a question bank, he/she will see the bank title (not the subtitle) you enter here.

  3. Enter the Question type (Multiple Choice, True/False, Completion, Short Answer, etc…)
  4. Enter the questions and the appropriate question information.

    Type the questions for the question bank, or copy/paste the content from another source. It is very important that you follow the guidelines provided in the previous sections.Do not number or enter instructions for the objective questions (e.g., Multiple Choice, True/False, etc.).

    If you copy and paste multiple choice questions from a word processor, you can use the Smart Paste feature. Copy the question and answer choices for a multiple choice question. Then, choose to create a new question. Press F7 instead of Ctrl+V and ExamView Test Generator will automatically paste the information into the choice entry areas.

    1. Click New at the bottom of the screen to display a window to choose the question type.
    2. Enter the question and answer.

      You can use any of the formatting options (bold, italic, tables, etc.) available in ExamView Test Generator to format the text you enter. If you need to link several questions to a figure, table, or passage, review the information on using narratives provided later in this section. If you need to insert art, see Using Artwork on Tests.

      For each new question type you create, click the Help button and review the instructions that pertain to the question type. For example, the Multiple Choice help topic will explain how and why you need to set the “scramble” option.

    3. Click Info if you need to enter any question information. Enter the question information (Difficulty, Reference, etc.). You can use the Rationale field to enter instructor notes or as a means to provide feedback to students who take the test online. For Multiple Choice questions, you can provide unique feedback for each choice. (See the online help for more details.)
    4. Select a narrative from the drop-down menu if you need to link the question to a narrative you already created. (See Narratives.)
    5. If appropriate, enter a rationale for any of the objective questions (e.g., multiple choice, true/false, etc.).

      You can use the Rationale entry area to provide detailed information that explains why a particular answer is correct. If you have a multiple choice question that requires a complex calculation, for example, you can provide the steps to calculate the answer in the Rationale area. For online practice tests, an instructor can choose whether to allow students to see this information.

  5. Click Record to record the question.
  6. Repeat these steps to enter the remaining questions.

    If you need to edit a question, double-click that question. If you find that you are always setting certain options (e.g., setting the number of columns for multiple choice questions), use the Preferences option in the Edit menu to change the defaults.

  7. Proof the question bank.

  8. Click Save on the toolbar and enter a file name. Use a descriptive name that is 27 or fewer characters in length, and select the RTF format

    It is very important to send a sample chapter to your contact so that he/she can forward it to Turning Technologies for review. To get the best feedback, send a representative chapter that includes all of the question types and features you will be using.
