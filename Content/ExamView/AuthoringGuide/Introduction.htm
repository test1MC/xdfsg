Introduction

The information provided in this manual explains how to create question banks for the test generator program ExamView® Assessment Suite—an application that enables instructors to quickly and easily create paper and online tests.

The guidelines included within this manual offer tips and suggestions for entering questions for use with the ExamView Assessment Suite software. The process to create a computerized question bank is very similar to the steps you already follow to create traditional test supplements.

Things You will Need

A list of the items you will need to write questions using Microsoft® WordExamView Test Generator is provided below.

  • ExamView Test Generator Application
  • ExamView Assessment Suite User Guide
  • Microsoft Word
  • ExamView Assessment Suite Software
  • ExamView Assessment Suite User Guide

Please note that all items authored in Rich Text Format (RTF) must be grouped by question type for import into the ExamView Assessment Suite BNK format. EVImport is not able to import RTF files with mixed question types.

The most up to date version of ExamView Assessment Suite software can be downloaded from the Turning Technologies website (http://www.turningtechnologies.com). While you do not need ExamView Assessment Suite to author questions in Microsoft Word, you can try the ExamView Assessment Suite software to see how it works. Using the program will help you understand the guidelines provided in this manual.

Data Entry Mode Instructions: The data entry mode is recommended to streamline the authoring efforts. Opening ExamView Test Generator from this shortcut enables several data entry features not included in the standard software.

These features include:

  • The ability to add question information to the entire bank in one field vs. one question at a time.
  • The ability to see the locked options for multiple choice questions.
  • The ability to see the narrative tags for each question.
