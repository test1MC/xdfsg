Getting Started

Before you begin the writing process, you need to learn how to use the ExamView Assessment Suite software. While every effort was made to make ExamView Assessment Suite easy-to-use, your success in authoring questions directly in ExamView Test Generator depends on how well you know the program. The information provided in this manual will help you with that process.

To learn how to use ExamView Assessment Suite, you can:

  • Obtain a copy of ExamView Assessment Suite – Contact the publisher or a Turning Technologies project manager to obtain the latest copy of the ExamView Assessment Suite software. You may author questions using either the PC or Mac software.

    Each time you start a new project, verify you have the most recent version.

  • Download and review the software user guide – Download a copy of the ExamView Assessment Suite User Guide from the website (http://www.turningtechnologies.com/responsesystemsupport/downloads). It is highly recommended that you read the entire user guide to gain an understanding of how you will use the software to author questions and how instructors will use it to create tests.
  • Install the software – Refer to the user guide for instructions to install the software on your computer.
  • Use the Question Bank Editor – Practice using the Question Bank Editor to enter sample questions. Enter a representative sample of questions. Refer to the user guide and online help for assistance using the software.
