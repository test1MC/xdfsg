Importing a Participant List from a Learning Management System (LMS)

Flow supports the following Learning Management Systems (LMS): Blackboard, Moodle and any LMS with the Flow plug-in properly enabled. Confirm with your IT department that the integration is properly configured before attempting to use the integration.

 

The eInstruction Portal Service (EPS) module must be implemented.

  1. Open Flow.
  2. Click Manage participants.
  3. Click and select Load roster from LMS.
  4. Enter your LMS User Name and Password.
  5. Select the server from the Server Type drop-down menu.
  6. Enter the Server URL.
  7. Click Connect.
  8. Select your participant list from the drop-down menu.

    The participant list is imported into Flow.
