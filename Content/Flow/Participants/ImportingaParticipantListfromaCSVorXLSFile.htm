Importing a Participant List from a CSV or XLSX File

A participant list can be imported from a CSV or XLSX file.

A CSV or XLSX file where the columns equal:

Column A = Student ID

Column B = First Name

Column C = Last Name

Column D = Device ID

*Headers and Columns C and D are optional

The file should contain the necessary information for your class.

  1. Open Flow.
  2. Click Manage participants.
  3. Click and select Load roster from file.
  4. Browse to the saved file location and click Open.

    The file is imported into Flow.

  5. To use this participant list in the current session, select a registration method and refer to the chart below.
