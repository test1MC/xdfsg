Printing a Lesson

 

A lesson with at least one question. See Adding a Question in the Lesson Builder on page 1 to create a lesson in the Lesson Builder.

  1. Open the Lesson Builder .
  2. Do one of the following:
    • Create a new lesson
    • Load a lesson
    • Import a lesson
    • Import a PowerPoint presentation
  3. From the Lesson Builder ribbon, click Print.

    The Print Assistant opens.

  4. Optionally, select the page break from the Force page break drop-down menu.
  5. Optionally, change the image size from the Printed picture size drop-down menu.
  6. Select Print default question order or Print selected lesson models.

    Select Print lesson models to print several versions of a lesson.

  7. Click Preview to preview the lesson.
  8. Click Print.
