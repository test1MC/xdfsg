Technical Requirements

vPad Desktop

  • Windows® XP SP 3, Vista, 7 or 8 (32- and 64-bit)
    • Recommended for use with Windows computers and Windows Surface tablet
  • Mac OS X v10.7, 10.8, 10.9 or 10.10
    • Recommended for use with Mac computers

vPad Java

  • Linux Ubuntu® 12.04 or 14.04
    • Recommended for use with offline Linux computers

vPad Web

PC

  • Windows XP SP 3, Vista, 7, PRO 8.x, RT 8.1
  • IE 9, IE 10, IE 11, Firefox or Chrome
  • Adobe® Flash® Player®

Mac

  • OS X v10.7, 10.8, 10.9
  • Safari, Firefox or Chrome
  • Adobe Flash Player

Linux

  • Ubuntu 12.04 LTS
  • Chrome
  • Adobe Flash Player

vPad Smartphone

  • Android 2.1 or higher
  • iOS 5.1 or higher

vPad Tablet

  • Android 2.3 or higher
    • Recommended for use with Android tablets that have a screen size of 7 inches or larger
  • iOS 5.1 or higher
    • Recommended for use with iPad
