Connecting vPad Web

vPad Web refers to the use of the vPad application on any internet browser, on any operating system, that embeds Flash Player.

  • vPad Web can switch to local mode but cannot display the session on the local network.
  • Internet connection is required to start the application.
  • Homework cannot be stored locally.
  • Closing the web browser will not prompt the User leaving notification inside Flow.
  1. Go to http://www.vpad-web.com.
  2. Click Launch vPad Web.
  3. Enter your Student ID and vPad Session ID.
  4. Tap/click .
