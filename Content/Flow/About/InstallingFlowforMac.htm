Installing Flow for Mac

 

Download the software from http://www.turningtechnologies.eu/downloads. See DownloadingFlow on page 1 for more information.

  1. Double-click the ZIP file.
  2. Double-click the PKG file.

    The Flow Installer opens.

  3. Click Continue.
  4. If you accept the terms in the End User License Agreement (EULA), click Continue.
  5. Click Install.
  6. Click Continue Installation.
  7. Click Restart to reboot the computer.

A message will appear inside of the Flow Notification Center if a software update is available.
