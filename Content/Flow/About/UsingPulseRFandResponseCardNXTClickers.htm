Using Pulse RF and ResponseCard NXT Clickers

When using Pulse RF and ResponseCard NXT clickers, it is recommended to select the ResponseCard NXT, Pulse option from the Flow Settings so that all question types except Constructed Response are supported during a polling session.

 

Flow must be open on the presenter computer.

  1. Open Flow.
  2. Click and select Settings.
  3. Below the Flow settings tab, select ResponseCard NXT, Pulse from the Limit features to this device’s capabilities drop-down menu.
  4. Click Save.
