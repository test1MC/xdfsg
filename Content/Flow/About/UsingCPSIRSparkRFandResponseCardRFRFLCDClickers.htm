Using CPS IR, Spark RF and ResponseCard RF/RF LCD Clickers

When using CPS IR and ResponseCard RF/RF LCD clickers, it is recommended to select the CPS IR, ResponseCard RF option from the Flow Settings so that only True/False and Multiple Choice questions are supported during a polling session.

 

Flow must be open on the presenter computer.

  1. Open Flow.
  2. Click and select Settings.
  3. Below the Flow settings tab, select CPS IR, ResponseCard RF from the Limit features to this device’s capabilities drop-down menu.
  4. Click Save.
