Hardware

The Turning Technologies Device Manager acts as a liaison between the hardware and the Flow software. Device Manager installs with Flow and the program runs in the background.

Connecting the Receiver

If using multiple receivers, each receiver must be set to a different channel. Clickers should be set to the same channel number as the corresponding receiver. For instructions on how to change the receiver channel, see Changing the Receiver Channel.

It is best practice to plug in the receiver BEFORE opening the software.

  1. Plug the receiver into an available USB port.
  2. Open Flow.
  3. Verify that the receiver is displayed in the Flow Main Window.

  4. If the receiver is not displayed, close Flow, verify the receiver is plugged into a working USB port and re-open Flow.
