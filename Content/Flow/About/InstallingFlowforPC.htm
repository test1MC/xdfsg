Installing Flow for PC

 

Download the software from http://www.turningtechnologies.eu/downloads. See DownloadingFlow on page 1 for more information.

  1. Open the executable setup file.
  2. Click Next when prompted by the InstallShield Wizard.
  3. If you accept the terms in the End User License Agreement (EULA), select the I accept the terms in the license agreement button and click Next.
  4. Optionally, change the software install location.
  5. Click Next.
  6. Click Install.
  7. Click Restart.

A message will appear inside of the Flow Notification Center if a software update is available.
