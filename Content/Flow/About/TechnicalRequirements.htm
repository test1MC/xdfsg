Technical Requirements

Flow is designed to function on most computers; however, there are some basic hardware and software requirements.

PC

  • Microsoft Windows® 7 or 8
  • Intel® Core 2 Duo 1 GHz processor (or equivalent)
  • 4 GB RAM
  • 600 MB hard disk space (32-bit OS); 1.5 GB hard disk space (64-bit OS)
  • 1024 x 768 resolution at 24-bit color or higher
  • Standard 2.0 USB port (for USB-based hardware devices)
  • Microsoft Office® 2003, 2007, 2010 or 2013 (if using the Flow PowerPoint add-in)
  • Internet connection for Homework mode

Mac

  • OS X v10.7, 10.8, 10.9 or 10.10
  • Intel® 2Ghz processor or higher
  • 4 GB RAM
  • 2 GB hard disk space
  • 1024 x 768 resolution at 24-bit color or higher
  • Standard 2.0 USB port (for USB-based hardware devices)
  • Internet connection for Homework mode

Linux

  • Linux Ubuntu® 12.04 LTS or 14.04
  • Intel® 2 GHz processor or higher
  • 4 GB RAM
  • 2 GB hard disk space
  • 1024 x 768 resolution at 24-bit color or higher
  • Standard 2.0 USB port (for USB-based hardware devices)
  • Internet connection for dependencies and Homework mode
  • "gksu" utility is required to run the installer on Ubuntu 14.04
