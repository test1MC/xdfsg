Using vPad with Flow

vPad is available for:

  • PC, Mac and Linux platforms
  • Android and iOS devices
  • Any internet browser that embeds Adobe® Flash® Player

Go to http://www.vpad-web.com or the appropriate app store to download the application.

For optimal performance, a stable Wi-Fi connection is recommended. If Wi-Fi is unavailable, a direct LAN connection can be used.

  1. Open vPad.
  2. Enter the Student ID in the Student ID field. The Student ID must be entered to join the session with vPad.
  3. Enter the vPad session number in the vPad session field.
  4. Optionally, select the local session that appears below the vPad session field.
