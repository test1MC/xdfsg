Downloading and Installing the Software

  1. Go to http://www.einstruction.eu/flowsoftware.
  2. Expand Flow below the Polling Software category.
  3. Select the appropriate Stand-alone Offline Installer (Mac, Ubuntu or Windows).
  4. Complete the Software Downloads Registration and click Send.
  5. After the download has completed, open the file and follow the on-screen installation instructions.

    The "gksu" utility is required for Ubuntu 14.04. Please refer to the Flow User Guide for full instructions.
