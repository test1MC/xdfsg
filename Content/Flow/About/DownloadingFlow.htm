Downloading Flow

The Flow software is accessible via http://www.turningtechnologies.eu/downloads.

 

Please verify that your computer meets the Technical Requirements on page 1 before downloading Flow.

  1. Expand Flow below Polling Software.
  2. Select the download option that corresponds to your operating system (OS).

    The Software Downloads Registration page is displayed.

  3. Enter the appropriate information in each field and click Send.

    A second Software Downloads Registration page is displayed.

  4. Click the link to download the software.

 

After the software is downloaded, it must be installed on the computer. See Installing Flow for PC on page 1, Installing Flow for Mac on page 1 or Installing Flow for Linux on page 1 for more information.
