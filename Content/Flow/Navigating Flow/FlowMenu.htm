Flow Menu

Click from the Flow Main Window to access the Flow Menu.

Option

Function
Flow question editor

Use the Flow question editor to edit or create questions and paste them into a variety of formats including Microsoft Office®, OpenOffice™, Adobe® Flash®, SCORM®, HTML, SMART Notebook™, ActivInspire®, a WorkSpace blank page and a Windows file folder.

Load lesson

Click Load lesson to load and display a previously saved lesson in self-paced mode or presentation mode.

Download homework

Retrieve completed homework assignments from the Download homework option.

Load session

Click Load session to load a saved session and either edit the session or begin polling.

Settings

Click Flow settings to:

  • Display results when polling closes
  • Disable Microsoft Office security warnings (PC only)
  • Minimize Flow when polling is opened
  • Display Help at startup
  • Display general and vPad notifications
  • Enable vPad license notification at startup
  • Select device capabilities
  • Enable Auto-save (a copy of the session is kept, refreshing after each question)
  • Select a language

Click vPad settings to:

  • Disable the announcement of the session on the local network
  • Customize the announcement by entering a session name in the Session name box. Optionally, a password can be setup by selecting Require session password.
  • Select a default session icon
  • Question options determine the size of the image to send to the vPad. A small image size is suitable for small smart phones and/or large lecture halls. A large image size is best suited for large tablets and/or regular-sized classrooms.

License information

License information displays the current license type.

Detect devices

Detect devices allows Flow to refresh and search for devices and/or a Wi-Fi connection.

Help and support

The Help and support option opens the Flow User Guide.

About Flow

View the software version and company website link from the About Flow option.
