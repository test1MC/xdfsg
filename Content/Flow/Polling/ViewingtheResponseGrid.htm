Viewing the Response Grid

A response grid indicates which participants have responded to the question. The response grid is available when polling is open.

Participants are identified by Student ID or first name or last name.

  1. Open Flow.
  2. Register participants. See Participants on page 1 for more information.
  3. Poll a question. See for more information.
  4. While polling is opened, click Response grid.

    The response grid is displayed.

    Participants who have responded to the question are highlighted in green, while participants who have not responded are highlighted in red. If Allow answers to be changed is selected, participants who changed answers are highlighted in blue.

  5. Optionally, click and select Student first name to change the display from ID to name.
