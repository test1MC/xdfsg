Viewing Results by Participant

The results by participant view displays each participant’s answer choice, response time and score.

  1. After the session, click Session results.

  2. Select the Participants tab.
  3. From within the Session results window, you can:
    • Adjust grading options
    • Select a participant and view data

      The Edit icon is made available when a participant is selected. Click to edit the score, view the participant's comment or add a comment.

    • Save the session data
    • Export the results to HTML, Excel or to a Flow lesson
    • Send the results to vPads
    • Clear session data
    • Drag and drop the session data into another application
  4. Click Close.
