Chart Options

There are two ways to view session results:

  • Question statistics - The Question statistics window is available when you want to view results in real-time. During polling, click Question statistics to view participant results by pie, bar, participant, word cloud or response over time chart.
  • Session results - View all questions asked during the session from the Session results window. When you are finished polling, click Session results on the Flow Main Window. The Session results window displays the collected answers in a pie, bar, participant, word cloud, or response over time chart.

For reporting and viewing purposes, charts can be dragged and dropped into any external application.

If dragging and dropping into PowerPoint, drop the chart outside of a text zone while in edit mode.

For Multiple Answer questions, a bar chart displays the response given while a histogram chart displays the distribution.

Icon Chart Type

Pie chart (Answer distribution) - A pie chart displays the results of participant responses. From the Question statistics window, a correct answer can be set by clicking the chart, clicking the chart legend or selecting .

A correct answer can also be set from the Session results window.

Bar chart (Answer distribution) - A bar chart displays the results of participant responses. From the Question statistics window, a correct answer can be set by clicking the chart, clicking the chart legend or selecting .

A correct answer can also be set from the Session results window.

Participant responses - The Participant responses chart displays the Student ID, participant name, Device ID, time, response and associated score. From the Question Statistics window, a correct answer can be set by clicking the chart, clicking the chart legend or selecting .

A correct answer can also be set from the Session results window.

Word cloud - The Word cloud chart is a graph that displays a large amount of text in a meaningful way. Responses are displayed in blue and correct answers are displayed in green. Hover your cursor over a word to display the number of responses for that particular entry. You can also click a word to edit its answer.

A correct answer can be set by clicking on a specific word.

Response over time - The Response over time chart tracks the time (in seconds) it takes for each participant to respond to the question. From the Question Statistics window, a correct answer can be set by selecting .

A correct answer can also be set from the Session results window.
Content - Click the Content icon in the Session results window to view a screenshot of the desktop when a particular question was polled.
