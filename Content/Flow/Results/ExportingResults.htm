Exporting Session Results

Session results can be quickly exported to HTML, XLSX or a Flow lesson. Use exported results to view results data, screenshots and the session summary. Export to a Flow lesson (FLL file) to utilize previous session content in a new session.

  1. After the session, or when the session is reloaded, click Session results from the Flow Main Window.

    .

  2. Click .
  3. Select one of the following options:
    • HTML - The export includes question titles, screenshots, detailed grades and answers, and a summary of the session.
    • XLSX - An editable spreadsheet that includes question titles, screenshots, detailed grades and answers, and a summary of the session.

For Excel 2002 users, please refer to the following Microsoft KnowledgeBase article concerning compatibility packs http://support.microsoft.com/kb/924074/en.

  • Flow lesson - Saves as a particular file format (FLL) specific to Flow that allows you to use the file in:
    • Lesson Builder for editing
    • Lesson Builder to publish as Homework
    • Flow to display as self-paced mode or presentation mode
