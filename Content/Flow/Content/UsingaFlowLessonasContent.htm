Using a Flow Lesson as Content

  1. Open Flow.
  2. Click and select Load lesson.
  3. Browse to the saved lesson and click Open.

    The lesson opens.

  4. To include all questions in the lesson, select the box labeled Select all.
  5. To rearrange the order of questions for vPad users, select the box labeled Shuffle questions for vPad.
  6. To register participants or assign participants to a participant list, click Manage participants. See Participants on page 1 for more information.
  7. To begin the lesson in self-paced mode, click Engage lesson.
  8. To manually advance the questions, select Presentation mode and then click Engage lesson.
  9. While polling is open the following options are available:
    • Allow answers to be changed - This option allows participants to change their answers until polling ends.
    • Auto advance when questions end - Select this option to allow the software to automatically advance polling questions.
    • Display the Response grid - The response grid displays the participants that have responded to the question. For more information on the response grid, see Response Grid on page 1.
    • Show statistics - This option allows you to view incoming responses in real-time. For more information on question statistics, see Question Statistics on page 1.
  10. Click End polling to close polling.
  11. Click Session results to view the results data. See Results on page 1 for more information.
