True/False

A true/false question requires participants to respond with 1/A for true or 2/B for false.

  1. Open Flow.
  2. Register participants. See Participants on page 1 for more information.
  3. Click Engage question.
  4. Select True/False.
  5. Adjust the question parameters as necessary.
Question parameters
Title

Value

Max Time

Penalty

Answer

Select True or False as the correct answer. A value of None can also be selected if there is no correct answer.
Use letters for answers

If checked, letters will be used as answer bullets. If unchecked, numbers will be displayed as answer bullets.
