Multiple Choice

A multiple choice question may have up to 10 answer options.

  1. Open Flow.
  2. Register participants. See Participants on page 1 for more information.
  3. Click Engage question.
  4. Select Multiple Choice.
  5. Adjust the question parameters as necessary.
Question parameters
Title

Value

Choices Select up to 10 answer choices from the drop-down menu.
Max Time

Penalty

Answer

Select the correct answer from the drop-down menu.
Use letters for answers

If checked, letters will be used as answer bullets. If unchecked, numbers will be displayed as answer bullets.
Advanced mode If checked, the percentage of the total value is awarded for each answer choice (+100/-1000).
