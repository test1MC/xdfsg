Using the Flow Question Editor

In the Flow question editor, you can edit current questions, create new questions and set question parameters.

  1. Click and select Flow question editor.

    The Flow question editor opens.

  2. Select a question type from the Type drop-down menu.
  3. Enter a question title.
  4. Optionally, enter an ID, a Max. Time, Point Value, Penalty Value, Answer Option(s) and Correct Answer(s).
  5. Optionally, click Content Screenshot to set up question content.

    The Edit question content window opens.

  6. Select the type of content to add:
    • Screenshot - Flow takes a screenshot of the desktop. The screenshot is displayed when the question is polled or when results are viewed.
    • Custom text - Enter the text in box provided.
    • Internet content - Select the content Content source drop-down menu and enter the URL.
  7. Click Save.
  8. Optionally, customize the question by changing the placeholder from the Show question as drop-down menu, including text with the question, including the correct answer in the tooltip, or displaying a custom tooltip.
  9. Click Generate question to create a Flow question.

    The question is copied to the clipboard and can be pasted into a number of formats. The compatible formats include: Microsoft Office®, OpenOffice™, Adobe Flash®, SCORM®, HTML, SMART Notebook™, ActivInspire®, a WorkSpace blank page and in a folder.

  10. Paste or drag and drop the question tag into an application, such as Word.

    Alternatively, place the question tag into the desired format by selecting the icon next to Customize your question, dragging the icon and dropping it into the document.

    A tag is placed next to the Flow question.

    For OpenOffice Users:

    If using Open Impress, select Include Text.

    To insert a question in OpenOffice Writer, click Generate Question and select Edition > Special Paste > HTML.
