Publishing Homework via the Lesson Builder

This feature is available for:

  • Premium licenses users
  • Trial users and users with Demonstration Premium licenses (one (1) complimentary Homework license)
  1. Open the Lesson Builder .
  2. Manually create a lesson or l Load a lesson file.

  3. After the lesson is created or loaded, click the Publish homework icon.

    The Homework window opens.

  4. Enter your name in the Instructor name field.
  5. Enter your class name in the Class name field.
  6. Enter your e-mail address in the Instructor e-mail field.
  7. Optionally, enter a homework description in the Description field.
  8. Enter the Homework ID prefix; i.e. HIS (for history).
  9. Enter a password with a minimum of six (6) alpha-numeric characters in the Password field.
  10. Enter the completion date in the Date limit field.
  11. Click Start Upload.

    You will receive an e-mail confirmation containing the Homework ID and password.
